# 13.1.2 代理角色与职责划分

在多代理协作系统中，合理的角色划分和职责分配是系统高效运行的关键。LangGraph v1.2.22通过灵活的状态图机制，支持多种代理角色模式，从传统的任务分解者、执行者到更复杂的协调者、监督者等角色，为构建专业化的多代理系统提供了强大支持。

## 概念讲解

### 代理角色的重要性
在多代理系统中，明确的角色划分带来以下优势：

1. **专业化分工**：每个代理专注于特定领域，提高处理效率
2. **责任清晰**：明确的职责边界，便于问题定位和调试
3. **系统可维护性**：角色分离降低系统复杂性，便于扩展和修改
4. **资源优化**：根据角色特点合理分配计算资源和数据访问权限

### 基础代理角色类型
根据代理在系统中的功能和职责，可以划分为以下几类基础角色：

1. **任务分解者（Task Decomposer）**
   - 职责：将复杂任务分解为可执行的子任务
   - 技能：任务分析、依赖识别、工作流设计
   - 输出：结构化任务树或任务网络

2. **任务分配者（Task Assigner）**
   - 职责：将子任务分配给合适的执行代理
   - 技能：代理能力评估、负载均衡、优先级调度
   - 输出：任务-代理分配映射

3. **任务执行者（Task Executor）**
   - 职责：执行具体的子任务
   - 技能：领域专业知识、工具使用、结果生成
   - 输出：任务执行结果

4. **结果聚合者（Result Aggregator）**
   - 职责：整合多个执行者的结果
   - 技能：数据融合、冲突解决、质量评估
   - 输出：统一的最终结果

### 高级代理角色
在复杂系统中，还需要更高级的角色：

1. **协调者（Coordinator）**
   - 职责：协调多个代理的协作，处理冲突
   - 技能：通信协议、协商机制、冲突解决

2. **监督者（Supervisor）**
   - 职责：监控代理状态和任务执行情况
   - 技能：状态监控、性能评估、故障检测

3. **学习者（Learner）**
   - 职责：从历史经验中学习，优化系统性能
   - 技能：机器学习、模式识别、策略优化

4. **适配者（Adapter）**
   - 职责：根据环境变化调整系统配置
   - 技能：环境感知、动态调整、配置管理

### LangGraph的角色支持机制
LangGraph v1.2.22通过以下机制支持角色划分：

1. **节点专业化**：每个节点可以代表特定角色的代理
2. **状态分区**：状态数据按角色需求进行分区管理
3. **消息路由**：基于角色的消息路由和过滤机制
4. **权限控制**：角色级别的访问控制和资源限制

## 核心要点

### 1. 角色-能力映射模型
建立代理角色与所需能力的映射关系：

```python
class RoleCapabilityMapping:
    """角色-能力映射模型"""
    
    ROLES = {
        "decomposer": ["task_analysis", "workflow_design", "dependency_detection"],
        "assigner": ["agent_evaluation", "load_balancing", "priority_scheduling"],
        "executor": ["domain_expertise", "tool_usage", "execution_quality"],
        "aggregator": ["data_fusion", "conflict_resolution", "quality_assessment"],
        "coordinator": ["communication_protocol", "negotiation", "conflict_resolution"],
        "supervisor": ["state_monitoring", "performance_evaluation", "fault_detection"]
    }
    
    @classmethod
    def get_required_capabilities(cls, role: str) -> list:
        """获取角色所需能力"""
        return cls.ROLES.get(role, [])
    
    @classmethod
    def get_suitable_roles(cls, capabilities: list) -> list:
        """根据能力获取适合的角色"""
        suitable_roles = []
        for role, required_caps in cls.ROLES.items():
            if all(cap in capabilities for cap in required_caps):
                suitable_roles.append(role)
        return suitable_roles
```

### 2. 职责边界定义原则
定义清晰的职责边界：

1. **单一职责**：每个代理只负责一个主要职责
2. **最小权限**：代理只拥有完成职责所需的最小权限
3. **接口明确**：代理间通过明确的接口进行交互
4. **依赖可控**：控制代理间的依赖关系，避免循环依赖

### 3. 动态角色切换机制
支持代理根据任务需求动态切换角色：

1. **角色上下文**：根据当前任务上下文选择合适角色
2. **能力匹配**：基于代理能力动态分配角色
3. **负载均衡**：根据系统负载调整角色分配
4. **故障转移**：角色故障时自动切换到备用代理

### 4. 角色间通信协议
设计角色间的标准通信协议：

1. **消息格式**：统一的角色间消息格式
2. **通信模式**：请求-响应、发布-订阅、广播等模式
3. **错误处理**：角色间通信错误的标准化处理
4. **安全机制**：角色间通信的认证和加密

## 简单示例

### 示例1：多角色代理系统实现
以下示例展示了如何在LangGraph中实现多角色代理系统：

```python
from typing import TypedDict, Annotated, List, Dict, Any, Optional
from typing_extensions import TypedDict
from langgraph.graph import StateGraph, START, END
from dataclasses import dataclass
from enum import Enum
import operator
import time

class TaskStatus(Enum):
    """任务状态枚举"""
    PENDING = "pending"
    ANALYZING = "analyzing"
    DECOMPOSED = "decomposed"
    ASSIGNED = "assigned"
    EXECUTING = "executing"
    COMPLETED = "completed"
    FAILED = "failed"

@dataclass
class Task:
    """任务定义"""
    id: str
    description: str
    status: TaskStatus
    complexity: int  # 1-5，复杂度等级
    subtasks: List[Dict[str, Any]] = None
    assigned_agents: Dict[str, str] = None  # 子任务ID -> 代理ID
    results: Dict[str, Any] = None
    final_result: Any = None

@dataclass
class Agent:
    """代理定义"""
    id: str
    name: str
    roles: List[str]  # 可担任的角色
    capabilities: Dict[str, float]  # 能力名称 -> 能力等级(0-1)
    current_role: Optional[str] = None
    current_tasks: List[str] = None
    performance_history: List[float] = None
    
    def __post_init__(self):
        if self.current_tasks is None:
            self.current_tasks = []
        if self.performance_history is None:
            self.performance_history = []
    
    def can_perform_role(self, role: str) -> bool:
        """检查代理是否能担任某个角色"""
        if role not in self.roles:
            return False
        
        # 检查角色所需能力
        role_requirements = {
            "decomposer": ["task_analysis", "workflow_design"],
            "assigner": ["agent_evaluation", "load_balancing"],
            "executor": ["domain_expertise", "execution_quality"],
            "aggregator": ["data_fusion", "quality_assessment"]
        }
        
        required_capabilities = role_requirements.get(role, [])
        for capability in required_capabilities:
            if capability not in self.capabilities or self.capabilities[capability] < 0.5:
                return False
        
        return True
    
    def assign_role(self, role: str):
        """分配角色给代理"""
        if self.can_perform_role(role):
            self.current_role = role
            print(f"代理 {self.name} 被分配角色: {role}")
            return True
        else:
            print(f"代理 {self.name} 无法担任角色: {role} (能力不足)")
            return False

class MultiRoleSystemState(TypedDict):
    """多角色系统状态"""
    tasks: Dict[str, Task]
    agents: Dict[str, Agent]
    task_queue: Annotated[List[str], operator.add]
    role_assignments: Dict[str, str]  # 代理ID -> 角色
    execution_log: Annotated[List[Dict[str, Any]], operator.add]
    system_performance: Dict[str, float]

def create_agents() -> Dict[str, Agent]:
    """创建示例代理"""
    agents = {}
    
    # 任务分解专家
    agents["agent_decomposer"] = Agent(
        id="agent_decomposer",
        name="任务分解专家",
        roles=["decomposer", "analyzer"],
        capabilities={
            "task_analysis": 0.9,
            "workflow_design": 0.85,
            "dependency_detection": 0.8,
            "complexity_assessment": 0.75
        }
    )
    
    # 任务分配专家
    agents["agent_assigner"] = Agent(
        id="agent_assigner",
        name="任务分配专家",
        roles=["assigner", "scheduler"],
        capabilities={
            "agent_evaluation": 0.88,
            "load_balancing": 0.82,
            "priority_scheduling": 0.79,
            "resource_management": 0.85
        }
    )
    
    # 数据处理专家
    agents["agent_data_executor"] = Agent(
        id="agent_data_executor",
        name="数据处理专家",
        roles=["executor"],
        capabilities={
            "data_cleaning": 0.92,
            "data_analysis": 0.87,
            "data_visualization": 0.8,
            "execution_quality": 0.85
        }
    )
    
    # 文本分析专家
    agents["agent_text_executor"] = Agent(
        id="agent_text_executor",
        name="文本分析专家",
        roles=["executor"],
        capabilities={
            "text_processing": 0.9,
            "sentiment_analysis": 0.88,
            "topic_extraction": 0.83,
            "execution_quality": 0.82
        }
    )
    
    # 结果聚合专家
    agents["agent_aggregator"] = Agent(
        id="agent_aggregator",
        name="结果聚合专家",
        roles=["aggregator", "quality_checker"],
        capabilities={
            "data_fusion": 0.85,
            "conflict_resolution": 0.8,
            "quality_assessment": 0.87,
            "report_generation": 0.83
        }
    )
    
    return agents

def role_assignment_node(state: MultiRoleSystemState) -> dict:
    """角色分配节点"""
    print("=== 角色分配阶段 ===")
    
    # 初始角色分配
    role_assignments = {}
    
    # 为每个代理分配最适合的角色
    for agent_id, agent in state["agents"].items():
        # 根据代理能力选择最合适的角色
        if "decomposer" in agent.roles and agent.capabilities.get("task_analysis", 0) > 0.8:
            role_assignments[agent_id] = "decomposer"
        elif "assigner" in agent.roles and agent.capabilities.get("agent_evaluation", 0) > 0.8:
            role_assignments[agent_id] = "assigner"
        elif "aggregator" in agent.roles and agent.capabilities.get("data_fusion", 0) > 0.8:
            role_assignments[agent_id] = "aggregator"
        elif "executor" in agent.roles:
            # 根据专业领域进一步细分
            if agent.capabilities.get("data_analysis", 0) > 0.8:
                role_assignments[agent_id] = "data_executor"
            elif agent.capabilities.get("text_processing", 0) > 0.8:
                role_assignments[agent_id] = "text_executor"
            else:
                role_assignments[agent_id] = "general_executor"
        
        # 更新代理的当前角色
        if agent_id in role_assignments:
            agent.assign_role(role_assignments[agent_id])
    
    print(f"角色分配完成: {role_assignments}")
    
    # 记录执行日志
    log_entry = {
        "timestamp": time.time(),
        "stage": "role_assignment",
        "assignments": role_assignments.copy(),
        "message": "角色分配完成"
    }
    
    return {
        "role_assignments": role_assignments,
        "execution_log": [log_entry]
    }

def task_decomposition_node(state: MultiRoleSystemState) -> dict:
    """任务分解节点"""
    print("\n=== 任务分解阶段 ===")
    
    decomposer_id = None
    # 找到任务分解代理
    for agent_id, role in state["role_assignments"].items():
        if role == "decomposer":
            decomposer_id = agent_id
            break
    
    if not decomposer_id:
        print("错误: 没有找到任务分解代理")
        return {}
    
    decomposer = state["agents"][decomposer_id]
    print(f"任务分解代理: {decomposer.name}")
    
    # 处理队列中的任务
    updated_tasks = state["tasks"].copy()
    
    for task_id in state["task_queue"]:
        task = updated_tasks[task_id]
        
        if task.status == TaskStatus.PENDING:
            print(f"分解任务: {task.description}")
            
            # 根据任务复杂度生成子任务
            subtasks = []
            if task.complexity >= 4:
                # 高复杂度任务：分解为多个专业子任务
                subtasks = [
                    {"id": f"{task_id}_sub1", "type": "data_processing", "description": "数据处理"},
                    {"id": f"{task_id}_sub2", "type": "text_analysis", "description": "文本分析"},
                    {"id": f"{task_id}_sub3", "type": "result_synthesis", "description": "结果综合"}
                ]
            elif task.complexity >= 2:
                # 中等复杂度任务
                subtasks = [
                    {"id": f"{task_id}_sub1", "type": "data_analysis", "description": "数据分析"},
                    {"id": f"{task_id}_sub2", "type": "report_generation", "description": "报告生成"}
                ]
            else:
                # 低复杂度任务
                subtasks = [
                    {"id": f"{task_id}_sub1", "type": "basic_processing", "description": "基础处理"}
                ]
            
            task.subtasks = subtasks
            task.status = TaskStatus.DECOMPOSED
            updated_tasks[task_id] = task
            
            print(f"  生成 {len(subtasks)} 个子任务")
    
    # 记录执行日志
    log_entry = {
        "timestamp": time.time(),
        "stage": "task_decomposition",
        "decomposer": decomposer_id,
        "tasks_decomposed": len([t for t in updated_tasks.values() if t.status == TaskStatus.DECOMPOSED]),
        "message": "任务分解完成"
    }
    
    return {
        "tasks": updated_tasks,
        "execution_log": [log_entry]
    }

def task_assignment_node(state: MultiRoleSystemState) -> dict:
    """任务分配节点"""
    print("\n=== 任务分配阶段 ===")
    
    assigner_id = None
    # 找到任务分配代理
    for agent_id, role in state["role_assignments"].items():
        if role == "assigner":
            assigner_id = agent_id
            break
    
    if not assigner_id:
        print("错误: 没有找到任务分配代理")
        return {}
    
    assigner = state["agents"][assigner_id]
    print(f"任务分配代理: {assigner.name}")
    
    updated_tasks = state["tasks"].copy()
    assignment_log = []
    
    for task_id, task in updated_tasks.items():
        if task.status == TaskStatus.DECOMPOSED and task.subtasks:
            print(f"分配任务 {task_id} 的子任务")
            
            assigned_agents = {}
            
            for subtask in task.subtasks:
                subtask_id = subtask["id"]
                subtask_type = subtask["type"]
                
                # 根据子任务类型选择最合适的执行代理
                suitable_agent = None
                for agent_id, agent in state["agents"].items():
                    role = state["role_assignments"].get(agent_id)
                    
                    if role == "data_executor" and subtask_type in ["data_processing", "data_analysis"]:
                        suitable_agent = agent_id
                        break
                    elif role == "text_executor" and subtask_type in ["text_analysis"]:
                        suitable_agent = agent_id
                        break
                    elif role == "general_executor" and suitable_agent is None:
                        suitable_agent = agent_id
                
                if suitable_agent:
                    assigned_agents[subtask_id] = suitable_agent
                    print(f"  子任务 {subtask_id} 分配给 {suitable_agent}")
                else:
                    print(f"  警告: 没有找到适合子任务 {subtask_id} 的代理")
            
            task.assigned_agents = assigned_agents
            task.status = TaskStatus.ASSIGNED
            assignment_log.append({
                "task_id": task_id,
                "assignments": assigned_agents.copy()
            })
    
    # 记录执行日志
    log_entry = {
        "timestamp": time.time(),
        "stage": "task_assignment",
        "assigner": assigner_id,
        "assignments": assignment_log,
        "message": "任务分配完成"
    }
    
    return {
        "tasks": updated_tasks,
        "execution_log": [log_entry]
    }

def task_execution_node(state: MultiRoleSystemState) -> dict:
    """任务执行节点"""
    print("\n=== 任务执行阶段 ===")
    
    updated_tasks = state["tasks"].copy()
    execution_results = []
    
    for task_id, task in updated_tasks.items():
        if task.status == TaskStatus.ASSIGNED and task.assigned_agents:
            print(f"执行任务 {task_id}")
            
            task_results = {}
            
            for subtask_id, agent_id in task.assigned_agents.items():
                agent = state["agents"][agent_id]
                subtask = next((st for st in task.subtasks if st["id"] == subtask_id), None)
                
                if subtask:
                    print(f"  代理 {agent.name} 执行子任务 {subtask_id}: {subtask['description']}")
                    
                    # 模拟执行过程
                    import random
                    success_rate = agent.capabilities.get("execution_quality", 0.5)
                    success = random.random() < success_rate
                    
                    if success:
                        result = {
                            "subtask_id": subtask_id,
                            "agent_id": agent_id,
                            "status": "completed",
                            "result": f"{subtask['description']} 执行成功",
                            "quality_score": random.uniform(0.7, 1.0),
                            "execution_time": random.uniform(0.5, 2.0)
                        }
                        task_results[subtask_id] = result
                        execution_results.append(result)
                        print(f"    执行成功")
                    else:
                        result = {
                            "subtask_id": subtask_id,
                            "agent_id": agent_id,
                            "status": "failed",
                            "error": "执行失败",
                            "retry_count": 1
                        }
                        task_results[subtask_id] = result
                        print(f"    执行失败")
            
            task.results = task_results
            task.status = TaskStatus.EXECUTING
            updated_tasks[task_id] = task
    
    # 记录执行日志
    log_entry = {
        "timestamp": time.time(),
        "stage": "task_execution",
        "completed_subtasks": len([r for r in execution_results if r["status"] == "completed"]),
        "failed_subtasks": len([r for r in execution_results if r["status"] == "failed"]),
        "message": "任务执行完成"
    }
    
    return {
        "tasks": updated_tasks,
        "execution_log": [log_entry]
    }

def result_aggregation_node(state: MultiRoleSystemState) -> dict:
    """结果聚合节点"""
    print("\n=== 结果聚合阶段 ===")
    
    aggregator_id = None
    # 找到结果聚合代理
    for agent_id, role in state["role_assignments"].items():
        if role == "aggregator":
            aggregator_id = agent_id
            break
    
    if not aggregator_id:
        print("错误: 没有找到结果聚合代理")
        return {}
    
    aggregator = state["agents"][aggregator_id]
    print(f"结果聚合代理: {aggregator.name}")
    
    updated_tasks = state["tasks"].copy()
    
    for task_id, task in updated_tasks.items():
        if task.status == TaskStatus.EXECUTING and task.results:
            print(f"聚合任务 {task_id} 的结果")
            
            # 收集所有成功的子任务结果
            successful_results = [
                r for r in task.results.values() 
                if r.get("status") == "completed"
            ]
            
            if successful_results:
                # 计算平均质量分数
                avg_quality = sum(r.get("quality_score", 0) for r in successful_results) / len(successful_results)
                
                # 生成聚合报告
                final_result = {
                    "task_id": task_id,
                    "total_subtasks": len(task.subtasks),
                    "successful_subtasks": len(successful_results),
                    "average_quality": round(avg_quality, 3),
                    "aggregation_time": time.time(),
                    "aggregated_by": aggregator_id,
                    "detailed_results": successful_results
                }
                
                task.final_result = final_result
                task.status = TaskStatus.COMPLETED
                updated_tasks[task_id] = task
                
                print(f"  聚合完成: {len(successful_results)}/{len(task.subtasks)} 个子任务成功")
                print(f"  平均质量: {avg_quality:.3f}")
            else:
                task.status = TaskStatus.FAILED
                print(f"  聚合失败: 所有子任务都失败了")
    
    # 计算系统性能
    total_tasks = len(updated_tasks)
    completed_tasks = len([t for t in updated_tasks.values() if t.status == TaskStatus.COMPLETED])
    success_rate = completed_tasks / total_tasks if total_tasks > 0 else 0
    
    system_performance = {
        "total_tasks": total_tasks,
        "completed_tasks": completed_tasks,
        "success_rate": success_rate,
        "timestamp": time.time()
    }
    
    # 记录执行日志
    log_entry = {
        "timestamp": time.time(),
        "stage": "result_aggregation",
        "aggregator": aggregator_id,
        "completed_tasks": completed_tasks,
        "system_performance": system_performance.copy(),
        "message": "结果聚合完成"
    }
    
    return {
        "tasks": updated_tasks,
        "execution_log": [log_entry],
        "system_performance": system_performance
    }

def build_multi_role_system():
    """构建多角色代理系统"""
    # 创建系统状态图
    builder = StateGraph(MultiRoleSystemState)
    
    # 添加节点（每个节点对应一个角色阶段）
    builder.add_node("assign_roles", role_assignment_node)
    builder.add_node("decompose_tasks", task_decomposition_node)
    builder.add_node("assign_tasks", task_assignment_node)
    builder.add_node("execute_tasks", task_execution_node)
    builder.add_node("aggregate_results", result_aggregation_node)
    
    # 构建工作流
    builder.add_edge(START, "assign_roles")
```

```python
def dynamic_role_switching():
    """动态角色切换机制"""
    
    class DynamicRoleAgent:
        """动态角色代理"""
        
        def __init__(self, agent_id: str, base_capabilities: Dict[str, float]):
            self.agent_id = agent_id
            self.base_capabilities = base_capabilities
            self.current_role = None
            self.role_history = []
            self.performance_by_role = {}
            self.learning_rate = 0.1  # 学习速率
            
        def evaluate_role_fitness(self, role_requirements: Dict[str, float]) -> float:
            """评估代理对某个角色的适合度"""
            fitness = 0.0
            total_weight = 0.0
            
            for requirement, weight in role_requirements.items():
                if requirement in self.base_capabilities:
                    capability_score = self.base_capabilities[requirement]
                    fitness += capability_score * weight
                    total_weight += weight
            
            return fitness / total_weight if total_weight > 0 else 0.0
        
        def switch_role(self, new_role: str, role_requirements: Dict[str, float]) -> bool:
            """切换角色"""
            fitness = self.evaluate_role_fitness(role_requirements)
            
            if fitness > 0.6:  # 适合度阈值
                # 记录角色切换
                if self.current_role:
                    self.role_history.append({
                        "from_role": self.current_role,
                        "to_role": new_role,
                        "timestamp": time.time(),
                        "fitness": fitness
                    })
                
                self.current_role = new_role
                print(f"代理 {self.agent_id} 切换到角色 {new_role} (适合度: {fitness:.2f})")
                
                # 学习：根据表现调整能力值
                self.learn_from_role_switch(new_role, fitness)
                
                return True
            else:
                print(f"代理 {self.agent_id} 不适合角色 {new_role} (适合度: {fitness:.2f})")
                return False
        
        def learn_from_role_switch(self, role: str, performance: float):
            """从角色切换中学习"""
            if role not in self.performance_by_role:
                self.performance_by_role[role] = []
            
            self.performance_by_role[role].append(performance)
            
            # 如果表现好，增强相关能力
            if performance > 0.8:
                # 获取该角色所需的能力
                role_capabilities = {
                    "decomposer": ["task_analysis", "workflow_design"],
                    "executor": ["domain_expertise", "execution_quality"],
                    "coordinator": ["communication_protocol", "conflict_resolution"]
                }
                
                required_caps = role_capabilities.get(role, [])
                for cap in required_caps:
                    if cap in self.base_capabilities:
                        # 增强能力
                        self.base_capabilities[cap] = min(1.0, 
                            self.base_capabilities[cap] + self.learning_rate)
        
        def adapt_capabilities(self, target_role: str):
            """根据目标角色调整能力"""
            # 识别需要提升的能力
            role_requirements = {
                "decomposer": {"task_analysis": 0.6, "workflow_design": 0.4},
                "executor": {"domain_expertise": 0.5, "execution_quality": 0.5},
                "coordinator": {"communication_protocol": 0.4, "conflict_resolution": 0.6}
            }
            
            requirements = role_requirements.get(target_role, {})
            
            for requirement, _ in requirements.items():
                if requirement in self.base_capabilities:
                    # 如果需要的能力不足，尝试提升
                    if self.base_capabilities[requirement] < 0.7:
                        # 模拟学习过程
                        improvement = self.learning_rate * 0.5
                        self.base_capabilities[requirement] = min(1.0,
                            self.base_capabilities[requirement] + improvement)
                        print(f"  提升能力 {requirement}: {self.base_capabilities[requirement]:.2f}")
    
    class DynamicRoleSystem:
        """动态角色系统"""
        
        def __init__(self):
            self.agents = {}
            self.tasks = []
            self.role_requirements = {
                "decomposer": {"task_analysis": 0.6, "workflow_design": 0.4},
                "executor": {"domain_expertise": 0.5, "execution_quality": 0.5},
                "coordinator": {"communication_protocol": 0.4, "conflict_resolution": 0.6},
                "supervisor": {"state_monitoring": 0.5, "fault_detection": 0.5}
            }
            
        def add_agent(self, agent_id: str, capabilities: Dict[str, float]):
            """添加代理"""
            self.agents[agent_id] = DynamicRoleAgent(agent_id, capabilities)
        
        def assign_role_by_task(self, task_type: str):
            """根据任务类型分配角色"""
            print(f"\n根据任务类型 '{task_type}' 分配角色:")
            
            # 确定任务需要的角色
            required_roles = []
            if task_type == "complex_analysis":
                required_roles = ["decomposer", "executor", "coordinator"]
            elif task_type == "simple_processing":
                required_roles = ["executor"]
            elif task_type == "system_monitoring":
                required_roles = ["supervisor", "coordinator"]
            
            # 为每个需要的角色分配最合适的代理
            role_assignments = {}
            
            for role in required_roles:
                best_agent = None
                best_fitness = 0.0
                
                for agent_id, agent in self.agents.items():
                    # 跳过已经有角色的代理
                    if agent_id in role_assignments.values():
                        continue
                    
                    fitness = agent.evaluate_role_fitness(self.role_requirements[role])
                    
                    if fitness > best_fitness:
                        best_fitness = fitness
                        best_agent = agent_id
                
                if best_agent and best_fitness > 0.6:
                    # 分配角色
                    agent = self.agents[best_agent]
                    if agent.switch_role(role, self.role_requirements[role]):
                        role_assignments[role] = best_agent
            
            return role_assignments
        
        def optimize_role_distribution(self):
            """优化角色分配"""
            print("\n优化角色分配...")
            
            # 收集每个代理在不同角色上的表现
            agent_performance = {}
            
            for agent_id, agent in self.agents.items():
                agent_performance[agent_id] = {}
                for role in self.role_requirements.keys():
                    fitness = agent.evaluate_role_fitness(self.role_requirements[role])
                    agent_performance[agent_id][role] = fitness
            
            # 简单的优化：让每个代理担任最适合的角色
            optimized_assignments = {}
            assigned_agents = set()
            
            # 为每个角色选择最合适的代理
            for role in self.role_requirements.keys():
                best_agent = None
                best_score = 0.0
                
                for agent_id, performances in agent_performance.items():
                    if agent_id in assigned_agents:
                        continue
                    
                    score = performances[role]
                    if score > best_score:
                        best_score = score
                        best_agent = agent_id
                
                if best_agent and best_score > 0.6:
                    optimized_assignments[role] = best_agent
                    assigned_agents.add(best_agent)
            
            # 应用优化后的分配
            for role, agent_id in optimized_assignments.items():
                agent = self.agents[agent_id]
                agent.switch_role(role, self.role_requirements[role])
            
            return optimized_assignments
```

```python
def reinforcement_learning_role_optimization():
    """基于强化学习的角色优化"""
    import numpy as np
    
    class RLRoleOptimizer:
        """强化学习角色优化器"""
        
        def __init__(self, state_size: int, action_size: int):
            self.state_size = state_size
            self.action_size = action_size
            self.q_table = np.zeros((state_size, action_size))
            self.learning_rate = 0.1
            self.discount_factor = 0.95
            self.epsilon = 0.1  # 探索率
            
        def get_state_index(self, system_state: dict) -> int:
            """将系统状态转换为状态索引"""
            # 简化：基于任务数量和代理数量计算状态
            task_count = len(system_state.get("tasks", []))
            agent_count = len(system_state.get("agents", []))
            
            # 离散化状态
            if task_count == 0:
                task_state = 0
            elif task_count <= 3:
                task_state = 1
            elif task_count <= 6:
                task_state = 2
            else:
                task_state = 3
            
            if agent_count <= 2:
                agent_state = 0
            elif agent_count <= 4:
                agent_state = 1
            else:
                agent_state = 2
            
            # 组合状态
            return task_state * 3 + agent_state
        
        def get_action(self, state_index: int) -> int:
            """选择动作"""
            # ε-greedy策略
            if np.random.rand() < self.epsilon:
                # 探索：随机选择动作
                return np.random.randint(self.action_size)
            else:
                # 利用：选择Q值最高的动作
                return np.argmax(self.q_table[state_index])
        
        def update_q_table(self, state: int, action: int, reward: float, next_state: int):
            """更新Q表"""
            # Q-learning更新公式
            current_q = self.q_table[state, action]
            max_future_q = np.max(self.q_table[next_state])
            
            new_q = current_q + self.learning_rate * (
                reward + self.discount_factor * max_future_q - current_q
            )
            
            self.q_table[state, action] = new_q
        
        def get_role_strategy(self, action_index: int) -> dict:
            """根据动作索引获取角色分配策略"""
            strategies = [
                {"decomposer": 1, "executor": 2, "coordinator": 1},  # 策略1
                {"decomposer": 1, "executor": 3, "coordinator": 0},  # 策略2
                {"decomposer": 0, "executor": 4, "coordinator": 0},  # 策略3
                {"decomposer": 2, "executor": 2, "coordinator": 1},  # 策略4
            ]
            
            return strategies[action_index % len(strategies)]
```

## 常见问题

### Q1: 如何确定合适的角色划分粒度？
**A**: 角色划分粒度确定方法：
1. **业务复杂度**：业务越复杂，角色划分越细
2. **代理数量**：代理数量多时，需要更细的角色划分
3. **任务类型**：多样化任务需要专业化的角色
4. **性能要求**：高性能要求可能需要更细的角色分工
5. **维护成本**：过细的角色划分会增加系统维护成本

### Q2: 角色冲突如何处理？
**A**: 角色冲突处理策略：
1. **角色优先级**：为角色设置优先级，高优先级优先
2. **冲突检测**：实时检测角色间的资源冲突
3. **协商机制**：冲突角色通过协商达成一致
4. **仲裁代理**：引入第三方代理进行冲突仲裁
5. **动态调整**：根据冲突情况动态调整角色职责

### Q3: 如何评估角色分配的效果？
**A**: 角色分配效果评估指标：
1. **任务完成率**：分配后任务的完成比例
2. **资源利用率**：代理能力和资源的利用效率
3. **响应时间**：从任务分配到开始执行的时间
4. **质量指标**：任务执行的质量评分
5. **代理满意度**：代理对角色分配的满意程度

### Q4: 角色固化与灵活性的平衡？
**A**: 平衡角色固化与灵活性的方法：
1. **核心角色固定**：关键角色保持固定，确保稳定性
2. **辅助角色灵活**：辅助角色可根据需求动态调整
3. **角色能力矩阵**：建立角色-能力矩阵，支持灵活匹配
4. **渐进式固化**：经过验证的有效角色逐渐固化
5. **定期评估**：定期评估角色划分效果，适时调整
