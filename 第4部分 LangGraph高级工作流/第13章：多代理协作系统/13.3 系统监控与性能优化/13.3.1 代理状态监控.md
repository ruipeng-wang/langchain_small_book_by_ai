# 13.3.1 代理状态监控

## 概念讲解

在多代理协作系统中，代理状态监控是确保系统稳定运行、快速发现和解决问题的关键技术。它通过持续收集、分析和可视化代理的运行状态信息，为系统管理员提供实时洞察和预警能力。

**监控的核心价值**：
1. **可观测性**：理解系统内部状态，而非仅关注外部表现
2. **故障诊断**：快速定位问题根源，减少MTTR（平均修复时间）
3. **性能优化**：发现性能瓶颈，指导系统优化
4. **容量规划**：基于历史数据预测未来资源需求
5. **合规审计**：满足监管要求和安全审计

**LangGraph中的监控挑战**：
- **状态分散**：代理状态分布在多个节点和边中
- **异步执行**：监控需要处理并发和异步操作
- **状态持久化**：需要可靠的存储机制保存历史状态
- **实时性要求**：监控数据需要低延迟更新

**监控体系层次**：
1. **基础设施层**：CPU、内存、网络、磁盘等硬件资源
2. **运行时层**：Python进程、线程、内存使用情况
3. **应用层**：代理执行逻辑、任务状态、消息队列
4. **业务层**：任务成功率、响应时间、服务质量

## 核心要点

### 1. 监控数据类型

**性能指标**
- **响应时间**：代理处理请求的时间
- **吞吐量**：单位时间内处理的请求数量
- **并发数**：同时处理的请求数量
- **错误率**：失败请求占总请求的比例

**资源指标**
- **CPU使用率**：代理进程的CPU占用
- **内存使用**：代理进程的内存占用
- **网络I/O**：网络流量和连接数
- **磁盘I/O**：文件读写操作

**业务指标**
- **任务状态**：任务执行进度和结果
- **代理活跃度**：代理是否正常运行
- **消息队列**：待处理消息数量
- **资源竞争**：资源等待时间和竞争情况

**健康指标**
- **心跳检测**：定期检查代理是否存活
- **依赖健康**：外部服务（数据库、API）可用性
- **配置状态**：配置文件的完整性和正确性
- **安全状态**：安全策略和权限检查

### 2. LangGraph监控实现模式

**内置状态追踪**
- 利用`StateGraph`的状态管理能力
- 自动记录状态变化历史
- 支持状态回放和调试

**自定义监控节点**
- 在图中插入专门的监控节点
- 定期收集和上报监控数据
- 支持条件触发和警报

**外部监控集成**
- 集成Prometheus、Grafana等监控系统
- 支持OpenTelemetry标准
- 提供REST API查询接口

**实时流式监控**
- 利用LangGraph的流式输出能力
- 实时推送监控事件
- 支持WebSocket连接

### 3. 监控数据采集策略

**拉取模式（Pull）**
- 监控系统定期从代理拉取数据
- 适合批量数据收集
- 实现简单，但实时性差

**推送模式（Push）**
- 代理主动向监控系统推送数据
- 实时性好，但增加代理负担
- 适合关键指标监控

**混合模式**
- 关键指标使用推送模式
- 非关键指标使用拉取模式
- 平衡实时性和系统负载

**采样策略**
- **全量采集**：收集所有数据，准确性高但开销大
- **随机采样**：随机选择部分数据，降低开销
- **分层采样**：不同重要性的数据使用不同采样率
- **自适应采样**：根据系统负载动态调整采样率

### 示例1：基于LangGraph的代理状态监控系统

```python
from langgraph.graph import StateGraph, START, END
from typing import TypedDict, List, Dict, Optional
from datetime import datetime, timedelta
import asyncio
import psutil
import time
from collections import deque
import statistics

# 定义监控状态
class MonitoringState(TypedDict):
    agents: List[Dict]  # 代理列表
    metrics: Dict[str, Dict]  # 监控指标
    alerts: List[Dict]  # 警报列表
    history: Dict[str, List]  # 历史数据
    last_check_time: float  # 上次检查时间

# 资源监控节点
def monitor_resources(state: MonitoringState):
    """监控系统资源使用情况"""
    current_time = time.time()
    
    # 收集系统级指标
    system_metrics = {
        "timestamp": datetime.now().isoformat(),
        "cpu_percent": psutil.cpu_percent(interval=0.1),
        "memory_percent": psutil.virtual_memory().percent,
        "disk_usage": psutil.disk_usage("/").percent,
        "network_io": {
            "bytes_sent": psutil.net_io_counters().bytes_sent,
            "bytes_recv": psutil.net_io_counters().bytes_recv
        }
    }
    
    # 收集进程级指标（模拟代理进程）
    process_metrics = {}
    for agent in state["agents"]:
        if agent["status"] == "active":
            # 模拟进程指标
            process_metrics[agent["id"]] = {
                "cpu_percent": agent.get("cpu_usage", 0.5) * 100,  # 模拟
                "memory_mb": agent.get("memory_usage", 50),  # 模拟MB
                "thread_count": agent.get("threads", 1),
                "uptime_seconds": (current_time - agent.get("start_time", current_time))
            }
    
    # 更新指标历史
    metrics_history = state.get("history", {}).get("system", [])
    metrics_history.append(system_metrics)
    
    # 保留最近1000条记录
    if len(metrics_history) > 1000:
        metrics_history = metrics_history[-1000:]
    
    return {
        "metrics": {
            "system": system_metrics,
            "processes": process_metrics
        },
        "history": {
            "system": metrics_history,
            "last_check_time": current_time
        }
    }

# 代理健康检查节点
def check_agent_health(state: MonitoringState):
    """检查代理健康状态"""
    alerts = []
    
    for agent in state["agents"]:
        agent_id = agent["id"]
        current_time = time.time()
        
        # 检查心跳
        last_heartbeat = agent.get("last_heartbeat", 0)
        heartbeat_timeout = agent.get("heartbeat_timeout", 30)
        
        if current_time - last_heartbeat > heartbeat_timeout:
            alert = {
                "type": "heartbeat_timeout",
                "agent_id": agent_id,
                "severity": "critical",
                "message": f"代理 {agent_id} 心跳超时",
                "timestamp": datetime.now().isoformat(),
                "details": {
                    "last_heartbeat": datetime.fromtimestamp(last_heartbeat).isoformat(),
                    "timeout_seconds": heartbeat_timeout
                }
            }
            alerts.append(alert)
            agent["status"] = "unhealthy"
        
        # 检查资源使用
        process_metrics = state["metrics"].get("processes", {}).get(agent_id, {})
        cpu_usage = process_metrics.get("cpu_percent", 0)
        memory_usage = process_metrics.get("memory_mb", 0)
        
        if cpu_usage > 90:  # CPU使用率超过90%
            alert = {
                "type": "high_cpu_usage",
                "agent_id": agent_id,
                "severity": "warning",
                "message": f"代理 {agent_id} CPU使用率过高: {cpu_usage:.1f}%",
                "timestamp": datetime.now().isoformat()
            }
            alerts.append(alert)
        
        if memory_usage > 1024:  # 内存使用超过1GB
            alert = {
                "type": "high_memory_usage",
                "agent_id": agent_id,
                "severity": "warning",
                "message": f"代理 {agent_id} 内存使用过高: {memory_usage}MB",
                "timestamp": datetime.now().isoformat()
            }
            alerts.append(alert)
        
        # 检查任务执行状态
        if agent.get("consecutive_failures", 0) > 5:
            alert = {
                "type": "consecutive_failures",
                "agent_id": agent_id,
                "severity": "error",
                "message": f"代理 {agent_id} 连续失败次数过多: {agent['consecutive_failures']}",
                "timestamp": datetime.now().isoformat()
            }
            alerts.append(alert)
    
    # 更新警报历史
    all_alerts = state.get("alerts", []) + alerts
    
    return {
        "alerts": all_alerts,
        "agents": state["agents"]  # 更新后的代理状态
    }

# 性能分析节点
def analyze_performance(state: MonitoringState):
    """分析系统性能趋势"""
    system_history = state.get("history", {}).get("system", [])
    
    if len(system_history) < 10:
        return {"performance_analysis": {}}  # 数据不足
    
    # 提取最近数据
    recent_data = system_history[-100:]  # 最近100个数据点
    
    # 计算基本统计信息
    cpu_values = [d["cpu_percent"] for d in recent_data]
    memory_values = [d["memory_percent"] for d in recent_data]
    
    analysis = {
        "timestamp": datetime.now().isoformat(),
        "cpu_stats": {
            "mean": statistics.mean(cpu_values),
            "median": statistics.median(cpu_values),
            "stddev": statistics.stdev(cpu_values) if len(cpu_values) > 1 else 0,
            "min": min(cpu_values),
            "max": max(cpu_values),
            "p95": statistics.quantiles(cpu_values, n=20)[18] if len(cpu_values) >= 20 else max(cpu_values)
        },
        "memory_stats": {
            "mean": statistics.mean(memory_values),
            "median": statistics.median(memory_values),
            "stddev": statistics.stdev(memory_values) if len(memory_values) > 1 else 0,
            "min": min(memory_values),
            "max": max(memory_values)
        },
        "trends": {}
    }
    
    # 检测趋势（简单实现）
    if len(system_history) >= 20:
        last_10_avg = statistics.mean([d["cpu_percent"] for d in system_history[-10:]])
        prev_10_avg = statistics.mean([d["cpu_percent"] for d in system_history[-20:-10]])
        
        if last_10_avg > prev_10_avg * 1.2:
            analysis["trends"]["cpu_trend"] = "increasing"
        elif last_10_avg < prev_10_avg * 0.8:
            analysis["trends"]["cpu_trend"] = "decreasing"
        else:
            analysis["trends"]["cpu_trend"] = "stable"
    
    # 检测异常值
    cpu_mean = analysis["cpu_stats"]["mean"]
    cpu_stddev = analysis["cpu_stats"]["stddev"]
    
    if cpu_stddev > 0:
        # 识别超过3个标准差的异常值
        outliers = [v for v in cpu_values if abs(v - cpu_mean) > 3 * cpu_stddev]
        if outliers:
            analysis["anomalies"] = {
                "cpu_outliers": len(outliers),
                "outlier_values": outliers[:5]  # 最多返回5个异常值
            }
    
    return {"performance_analysis": analysis}

# 警报处理节点
def handle_alerts(state: MonitoringState):
    """处理监控警报"""
    critical_alerts = [a for a in state["alerts"] if a["severity"] == "critical"]
    warning_alerts = [a for a in state["alerts"] if a["severity"] == "warning"]
    
    actions_taken = []
    
    # 处理严重警报
    for alert in critical_alerts:
        if alert["type"] == "heartbeat_timeout":
            # 尝试重启代理
            agent_id = alert["agent_id"]
            actions_taken.append({
                "action": "restart_agent",
                "agent_id": agent_id,
                "reason": "heartbeat_timeout",
                "timestamp": datetime.now().isoformat()
            })
            
            # 更新代理状态
            for agent in state["agents"]:
                if agent["id"] == agent_id:
                    agent["status"] = "restarting"
                    agent["restart_count"] = agent.get("restart_count", 0) + 1
                    agent["last_restart"] = time.time()
                    break
    
    # 处理警告警报
    for alert in warning_alerts:
        if alert["type"] == "high_cpu_usage":
            # 降低优先级或限制资源
            agent_id = alert["agent_id"]
            actions_taken.append({
                "action": "throttle_agent",
                "agent_id": agent_id,
                "reason": "high_cpu_usage",
                "timestamp": datetime.now().isoformat()
            })
    
    # 发送通知（模拟）
    if critical_alerts or warning_alerts:
        notification = {
            "timestamp": datetime.now().isoformat(),
            "critical_count": len(critical_alerts),
            "warning_count": len(warning_alerts),
            "actions_taken": actions_taken
        }
        
        # 这里可以集成邮件、Slack、Webhook等通知方式
        print(f"监控通知: {notification}")
    
    return {
        "alert_actions": actions_taken,
        "processed_alerts": state["alerts"][:]  # 记录已处理的警报
    }

# 构建监控图
def build_monitoring_graph():
    workflow = StateGraph(MonitoringState)
    
    workflow.add_node("monitor_resources", monitor_resources)
    workflow.add_node("check_health", check_agent_health)
    workflow.add_node("analyze_perf", analyze_performance)
    workflow.add_node("handle_alerts", handle_alerts)
    
    workflow.add_edge(START, "monitor_resources")
    workflow.add_edge("monitor_resources", "check_health")
    workflow.add_edge("check_health", "analyze_perf")
    workflow.add_edge("analyze_perf", "handle_alerts")
    workflow.add_edge("handle_alerts", END)
    
    return workflow.compile()

# 周期性监控调度器
class MonitoringScheduler:
    """周期性监控调度器"""
    
    def __init__(self, interval_seconds: int = 60):
        self.interval = interval_seconds
        self.monitoring_graph = build_monitoring_graph()
        self.is_running = False
    
    async def start_monitoring(self, initial_state: MonitoringState):
        """启动周期性监控"""
        self.is_running = True
        state = initial_state
        
        while self.is_running:
            try:
                # 执行监控图
                result = self.monitoring_graph.invoke(state)
                
                # 更新状态
                state.update(result)
                
                # 记录监控日志
                self._log_monitoring_result(result)
                
                # 等待下一个周期
                await asyncio.sleep(self.interval)
                
            except Exception as e:
                print(f"监控执行失败: {e}")
                await asyncio.sleep(5)  # 错误后等待5秒重试
    
    def stop_monitoring(self):
        """停止监控"""
        self.is_running = False
    
    def _log_monitoring_result(self, result: Dict):
        """记录监控结果"""
        timestamp = datetime.now().isoformat()
        
        log_entry = {
            "timestamp": timestamp,
            "metrics_collected": len(result.get("metrics", {})),
            "alerts_generated": len(result.get("alerts", [])),
            "performance_analysis": result.get("performance_analysis", {}).get("timestamp")
        }
        
        # 这里可以写入文件、数据库或发送到日志系统
        print(f"监控周期完成: {log_entry}")
```

### 示例2：实时监控数据流

```python
import asyncio
from typing import AsyncGenerator
from datetime import datetime

class RealTimeMonitor:
    """实时监控数据流"""
    
    def __init__(self, update_interval: float = 1.0):
        self.update_interval = update_interval
        self.subscribers = []
        self.metrics_buffer = deque(maxlen=1000)
    
    def subscribe(self, callback):
        """订阅监控数据更新"""
        self.subscribers.append(callback)
    
    def unsubscribe(self, callback):
        """取消订阅"""
        if callback in self.subscribers:
            self.subscribers.remove(callback)
    
    async def start_streaming(self):
        """启动实时数据流"""
        while True:
            # 收集实时数据
            metrics = self._collect_real_time_metrics()
            self.metrics_buffer.append(metrics)
            
            # 通知所有订阅者
            for callback in self.subscribers:
                try:
                    if asyncio.iscoroutinefunction(callback):
                        await callback(metrics)
                    else:
                        callback(metrics)
                except Exception as e:
                    print(f"通知订阅者失败: {e}")
            
            await asyncio.sleep(self.update_interval)
    
    def _collect_real_time_metrics(self) -> Dict:
        """收集实时监控指标"""
        return {
            "timestamp": datetime.now().isoformat(),
            "agents": self._get_agent_metrics(),
            "system": self._get_system_metrics(),
            "performance": self._get_performance_metrics()
        }
    
    def _get_agent_metrics(self) -> List[Dict]:
        """获取代理实时指标"""
        # 模拟数据
        return [
            {
                "id": "agent1",
                "status": "active",
                "active_tasks": 5,
                "queue_size": 2,
                "response_time_ms": 120,
                "error_rate": 0.01
            },
            {
                "id": "agent2",
                "status": "active",
                "active_tasks": 3,
                "queue_size": 0,
                "response_time_ms": 85,
                "error_rate": 0.005
            }
        ]
    
    def _get_system_metrics(self) -> Dict:
        """获取系统实时指标"""
        import psutil
        
        return {
            "cpu_percent": psutil.cpu_percent(interval=None),
            "memory_percent": psutil.virtual_memory().percent,
            "disk_io": {
                "read_bytes": psutil.disk_io_counters().read_bytes,
                "write_bytes": psutil.disk_io_counters().write_bytes
            },
            "network_io": {
                "bytes_sent": psutil.net_io_counters().bytes_sent,
                "bytes_recv": psutil.net_io_counters().bytes_recv
            }
        }
    
    def _get_performance_metrics(self) -> Dict:
        """获取性能指标"""
        # 计算性能统计数据
        if len(self.metrics_buffer) > 0:
            recent_metrics = list(self.metrics_buffer)[-10:]  # 最近10个数据点
            
            response_times = []
            for metric in recent_metrics:
                for agent in metric.get("agents", []):
                    response_times.append(agent.get("response_time_ms", 0))
            
            if response_times:
                avg_response_time = sum(response_times) / len(response_times)
                max_response_time = max(response_times)
            else:
                avg_response_time = max_response_time = 0
            
            return {
                "avg_response_time_ms": avg_response_time,
                "max_response_time_ms": max_response_time,
                "throughput_tps": len(recent_metrics) / 10.0  # 每秒事务数
            }
        
        return {}

# WebSocket监控服务器示例
async def websocket_monitor_handler(websocket):
    """WebSocket监控处理器"""
    monitor = RealTimeMonitor(update_interval=0.5)
    
    # 定义回调函数
    async def send_metrics(metrics):
        """通过WebSocket发送监控数据"""
        try:
            await websocket.send_json(metrics)
        except Exception as e:
            print(f"WebSocket发送失败: {e}")
    
    # 订阅监控数据
    monitor.subscribe(send_metrics)
    
    try:
        # 启动监控流
        monitor_task = asyncio.create_task(monitor.start_streaming())
        
        # 等待WebSocket连接关闭
        await websocket.wait_closed()
        
    finally:
        # 清理资源
        monitor_task.cancel()
        monitor.unsubscribe(send_metrics)
```

### 1. 基于机器学习的异常检测

```python
import numpy as np
from sklearn.ensemble import IsolationForest
from sklearn.preprocessing import StandardScaler
from collections import deque

class AnomalyDetector:
    """基于机器学习的异常检测器"""
    
    def __init__(self, contamination=0.1, window_size=100):
        self.contamination = contamination
        self.window_size = window_size
        self.model = IsolationForest(contamination=contamination, random_state=42)
        self.scaler = StandardScaler()
        self.data_window = deque(maxlen=window_size)
        self.is_fitted = False
        
        # 要监控的指标
        self.metrics_to_monitor = [
            'cpu_percent',
            'memory_percent', 
            'response_time_ms',
            'error_rate',
            'queue_size'
        ]
    
    def add_metrics(self, metrics: Dict):
        """添加监控指标数据"""
        # 提取特征向量
        features = []
        for metric_name in self.metrics_to_monitor:
            if metric_name in metrics:
                features.append(metrics[metric_name])
            else:
                features.append(0.0)  # 缺失值用0填充
        
        self.data_window.append(features)
        
        # 当有足够数据时训练模型
        if len(self.data_window) >= self.window_size and not self.is_fitted:
            self._train_model()
    
    def _train_model(self):
        """训练异常检测模型"""
        data_array = np.array(list(self.data_window))
        
        # 标准化数据
        scaled_data = self.scaler.fit_transform(data_array)
        
        # 训练模型
        self.model.fit(scaled_data)
        self.is_fitted = True
    
    def detect_anomalies(self, current_metrics: Dict) -> Dict:
        """检测当前指标是否异常"""
        if not self.is_fitted:
            return {"anomaly": False, "confidence": 0.0, "reason": "model_not_trained"}
        
        # 提取当前特征
        current_features = []
        for metric_name in self.metrics_to_monitor:
            if metric_name in current_metrics:
                current_features.append(current_metrics[metric_name])
            else:
                current_features.append(0.0)
        
        # 标准化
        current_scaled = self.scaler.transform([current_features])
        
        # 预测
        prediction = self.model.predict(current_scaled)
        decision_score = self.model.decision_function(current_scaled)
        
        # prediction为-1表示异常，1表示正常
        is_anomaly = prediction[0] == -1
        
        # 计算异常置信度（决策函数值越小越异常）
        confidence = abs(decision_score[0])
        
        # 分析异常原因
        reason = None
        if is_anomaly:
            # 找出最异常的指标
            feature_contributions = []
            for i, metric_name in enumerate(self.metrics_to_monitor):
                # 计算该特征与平均值的偏差
                mean_value = np.mean([d[i] for d in self.data_window])
                std_value = np.std([d[i] for d in self.data_window])
                
                if std_value > 0:
                    z_score = abs((current_features[i] - mean_value) / std_value)
                    feature_contributions.append((metric_name, z_score))
            
            # 按Z-score排序，找出最异常的指标
            feature_contributions.sort(key=lambda x: x[1], reverse=True)
            top_anomalies = feature_contributions[:3]
            
            reason = f"异常指标: {', '.join([f'{name}(z={z:.2f})' for name, z in top_anomalies])}"
        
        return {
            "anomaly": is_anomaly,
            "confidence": float(confidence),
            "reason": reason,
            "timestamp": datetime.now().isoformat()
        }
    
    def get_metrics_distribution(self) -> Dict:
        """获取指标分布统计"""
        if not self.data_window:
            return {}
        
        data_array = np.array(list(self.data_window))
        
        distribution = {}
        for i, metric_name in enumerate(self.metrics_to_monitor):
            values = data_array[:, i]
            
            distribution[metric_name] = {
                "mean": float(np.mean(values)),
                "std": float(np.std(values)),
                "min": float(np.min(values)),
                "max": float(np.max(values)),
                "percentiles": {
                    "25": float(np.percentile(values, 25)),
                    "50": float(np.percentile(values, 50)),
                    "75": float(np.percentile(values, 75)),
                    "95": float(np.percentile(values, 95))
                }
            }
        
        return distribution
```

### 2. 分布式监控数据聚合

```python
from typing import Dict, List
import time
from concurrent.futures import ThreadPoolExecutor
import requests

class DistributedMonitorAggregator:
    """分布式监控数据聚合器"""
    
    def __init__(self, agent_endpoints: Dict[str, str]):
        self.agent_endpoints = agent_endpoints
        self.executor = ThreadPoolExecutor(max_workers=10)
        self.aggregation_window = 60  # 聚合窗口（秒）
        self.aggregated_data = {}
    
    async def collect_distributed_metrics(self) -> Dict:
        """收集分布式代理的监控数据"""
        collection_tasks = []
        
        for agent_id, endpoint in self.agent_endpoints.items():
            task = self._fetch_agent_metrics(agent_id, endpoint)
            collection_tasks.append(task)
        
        # 并行收集所有代理的数据
        results = await asyncio.gather(*collection_tasks, return_exceptions=True)
        
        # 处理结果
        collected_metrics = {}
        for result in results:
            if isinstance(result, Exception):
                print(f"收集监控数据失败: {result}")
                continue
            
            agent_id, metrics = result
            collected_metrics[agent_id] = metrics
        
        return collected_metrics
    
    async def _fetch_agent_metrics(self, agent_id: str, endpoint: str) -> tuple:
        """获取单个代理的监控数据"""
        try:
            # 这里可以使用HTTP请求、gRPC或其他通信协议
            # 示例使用HTTP
            async with aiohttp.ClientSession() as session:
                async with session.get(f"{endpoint}/metrics", timeout=5) as response:
                    if response.status == 200:
                        metrics = await response.json()
                        return (agent_id, metrics)
                    else:
                        raise Exception(f"HTTP {response.status}: {await response.text()}")
        except Exception as e:
            # 返回默认的故障指标
            default_metrics = {
                "status": "unreachable",
                "last_seen": time.time(),
                "error": str(e)
            }
            return (agent_id, default_metrics)
    
    def aggregate_metrics(self, collected_metrics: Dict) -> Dict:
        """聚合所有代理的监控数据"""
        aggregation_time = time.time()
        
        # 初始化聚合结果
        aggregated = {
            "timestamp": datetime.fromtimestamp(aggregation_time).isoformat(),
            "total_agents": len(self.agent_endpoints),
            "active_agents": 0,
            "unhealthy_agents": 0,
            "aggregated_metrics": {},
            "agent_details": {}
        }
        
        # 处理每个代理的数据
        for agent_id, metrics in collected_metrics.items():
            aggregated["agent_details"][agent_id] = metrics
            
            if metrics.get("status") == "active":
                aggregated["active_agents"] += 1
                
                # 聚合性能指标
                for metric_name, metric_value in metrics.get("performance", {}).items():
                    if metric_name not in aggregated["aggregated_metrics"]:
                        aggregated["aggregated_metrics"][metric_name] = []
                    
                    aggregated["aggregated_metrics"][metric_name].append(metric_value)
            else:
                aggregated["unhealthy_agents"] += 1
        
        # 计算聚合统计
        for metric_name, values in aggregated["aggregated_metrics"].items():
            if values:
                aggregated["aggregated_metrics"][metric_name] = {
                    "count": len(values),
                    "mean": sum(values) / len(values),
                    "min": min(values),
                    "max": max(values),
                    "sum": sum(values)
                }
        
        # 计算系统级健康度
        if aggregated["total_agents"] > 0:
            aggregated["health_score"] = (aggregated["active_agents"] / aggregated["total_agents"]) * 100
        else:
            aggregated["health_score"] = 0
        
        # 保存聚合结果
        self.aggregated_data[aggregation_time] = aggregated
        
        # 清理旧数据
        self._cleanup_old_data(aggregation_time)
        
        return aggregated
    
    def _cleanup_old_data(self, current_time: float):
        """清理旧的聚合数据"""
        cutoff_time = current_time - (self.aggregation_window * 10)  # 保留10个窗口
        
        keys_to_remove = [k for k in self.aggregated_data.keys() if k < cutoff_time]
        for key in keys_to_remove:
            del self.aggregated_data[key]
    
    def get_historical_trends(self, metric_name: str, window_minutes: int = 60) -> List[Dict]:
        """获取历史趋势数据"""
        current_time = time.time()
        cutoff_time = current_time - (window_minutes * 60)
        
        trend_data = []
        
        for timestamp, aggregated in sorted(self.aggregated_data.items()):
            if timestamp < cutoff_time:
                continue
            
            if metric_name in aggregated["aggregated_metrics"]:
                metric_value = aggregated["aggregated_metrics"][metric_name]["mean"]
                trend_data.append({
                    "timestamp": datetime.fromtimestamp(timestamp).isoformat(),
                    "value": metric_value,
                    "active_agents": aggregated["active_agents"]
                })
        
        return trend_data
```

## 常见问题

### Q1: 如何设计高效的监控系统？

**回答**：设计高效监控系统的关键原则：

1. **明确监控目标**：
   - 故障检测：快速发现问题
   - 性能优化：识别瓶颈
   - 容量规划：预测资源需求
   - 合规审计：满足监管要求

2. **选择合适的监控粒度**：
   - **粗粒度**：系统级监控，开销小
   - **中粒度**：应用级监控，平衡开销和信息量
   - **细粒度**：方法级监控，开销大但信息详细

3. **优化数据采集**：
   - 使用异步采集避免阻塞业务逻辑
   - 实现数据采样减少数据量
   - 采用增量更新减少重复数据

4. **设计可扩展架构**：
   - 支持水平扩展应对增长
   - 使用消息队列解耦采集和处理
   - 实现插件架构支持新监控类型

### Q2: 监控系统可能引入哪些性能问题？

**回答**：监控系统可能引入的性能问题：

1. **资源竞争**：
   - 监控数据采集占用CPU和内存
   - 大量监控数据占用网络带宽
   - 监控存储占用磁盘空间

2. **延迟影响**：
   - 同步监控调用增加请求延迟
   - 监控数据处理占用业务处理时间
   - 监控数据同步影响系统响应

3. **稳定性风险**：
   - 监控系统故障影响业务系统
   - 监控数据异常触发误报警
   - 监控配置错误导致性能下降

**解决方案**：
- 使用异步和非阻塞监控调用
- 实现监控降级和熔断机制
- 设置合理的监控采样率和频率
- 分离监控网络和业务网络

### Q3: 如何在LangGraph中实现无侵入监控？

**回答**：在LangGraph中实现无侵入监控的策略：

1. **装饰器模式**：
   - 使用Python装饰器包装节点函数
   - 自动收集执行时间和错误信息
   - 不修改原有业务逻辑

2. **中间件机制**：
   - 实现LangGraph中间件拦截请求
   - 在请求前后添加监控逻辑
   - 支持链式中间件组合

3. **AOP（面向切面编程）**：
   - 定义监控切面拦截特定方法
   - 统一处理横切关注点
   - 通过配置启用/禁用监控

4. **代理模式**：
   - 创建代理对象包装原始节点
   - 代理对象添加监控逻辑
   - 对调用者透明

**示例实现**：
```python
def monitor_node_execution(func):
    """监控节点执行的装饰器"""
    @wraps(func)
    def wrapper(state, config=None):
        start_time = time.time()
        
        try:
            result = func(state, config)
            execution_time = time.time() - start_time
            
            # 记录监控数据
            monitor_data = {
                "node": func.__name__,
                "execution_time": execution_time,
                "status": "success",
                "timestamp": datetime.now().isoformat()
            }
            
            # 发送到监控系统
            send_to_monitoring(monitor_data)
            
            return result
            
        except Exception as e:
            execution_time = time.time() - start_time
            
            error_data = {
                "node": func.__name__,
                "execution_time": execution_time,
                "status": "error",
                "error": str(e),
                "timestamp": datetime.now().isoformat()
            }
            
            send_to_monitoring(error_data)
            raise
    
    return wrapper
```

3. **智能化**：结合机器学习实现智能异常检测和预测
4. **可操作性**：监控数据必须能够指导具体的运维操作
