# 13.3.2 性能指标收集

## 概念讲解

性能指标收集是多代理协作系统中监控与优化的基础环节，它系统性地采集、记录和分析系统运行时的各项性能数据，为性能评估、瓶颈诊断和容量规划提供数据支持。有效的性能指标收集不仅能够反映系统当前的健康状态，还能揭示潜在的性能趋势和优化机会。

**性能指标收集的核心价值**：
1. **量化评估**：将系统性能转化为可度量、可比较的数字
2. **趋势分析**：通过历史数据识别性能变化趋势
3. **瓶颈定位**：精准定位系统性能瓶颈所在
4. **容量预测**：基于历史趋势预测未来资源需求
5. **优化验证**：验证性能优化措施的实际效果

**LangGraph中的性能指标特点**：
- **图结构感知**：指标与图节点、边、状态流转相关
- **异步复杂性**：需要处理并发执行和异步操作的指标
- **状态相关性**：指标与系统状态紧密关联
- **实时性要求**：部分指标需要实时或近实时采集

**性能指标收集层次**：
1. **节点级指标**：单个节点的执行时间、成功率、资源使用
2. **边级指标**：状态转移频率、条件分支比例
3. **图级指标**：整体执行时间、并发度、吞吐量
4. **业务级指标**：任务完成时间、服务质量、用户满意度

## 核心要点

### 1. 性能指标分类体系

**响应时间指标**
- **端到端延迟**：从请求开始到响应完成的总时间
- **处理时间**：实际处理逻辑的执行时间
- **排队时间**：在队列中等待处理的时间
- **网络延迟**：网络传输消耗的时间

**吞吐量指标**
- **请求率**：单位时间内的请求数量
- **处理率**：单位时间内成功处理的请求数量
- **并发数**：同时处理的请求数量
- **队列长度**：等待处理的请求数量

**资源使用指标**
- **CPU使用率**：处理器资源占用比例
- **内存使用**：内存占用量和碎片情况
- **I/O操作**：磁盘和网络读写频率
- **连接数**：数据库、API等外部连接数量

**错误与可靠性指标**
- **错误率**：失败请求占总请求的比例
- **重试率**：需要重试的操作比例
- **超时率**：操作超时的比例
- **可用性**：系统可正常服务的时间比例

**业务相关指标**
- **任务完成率**：成功完成的任务比例
- **服务质量**：响应时间满足SLA的比例
- **用户满意度**：用户反馈的满意程度
- **成本效益**：性能与资源消耗的比值

### 2. 指标收集策略

**定时采集**
- 固定时间间隔采集指标
- 实现简单，数据均匀
- 可能错过瞬时峰值

**事件驱动采集**
- 特定事件触发指标采集
- 捕捉关键事件时的状态
- 数据稀疏，可能遗漏常规状态

**自适应采集**
- 根据系统负载动态调整采集频率
- 负载高时降低频率，负载低时提高频率
- 平衡数据质量和系统开销

**分层采集**
- 不同重要性的指标使用不同采集策略
- 关键指标高频采集，次要指标低频采集
- 优化资源使用效率

### 3. LangGraph中的指标收集实现

**装饰器模式**
- 使用Python装饰器包装节点函数
- 自动收集执行时间和错误信息
- 代码侵入性低，易于维护

**中间件机制**
- 在LangGraph执行管道中插入监控中间件
- 统一处理所有节点的指标收集
- 支持灵活的扩展和配置

**自定义监控节点**
- 创建专门的指标收集节点
- 与其他业务节点协同工作
- 支持复杂的指标计算逻辑

**外部集成**
- 集成Prometheus、StatsD等监控系统
- 使用OpenTelemetry标准收集指标
- 支持多种存储后端和分析工具

### 示例1：基于装饰器的节点性能指标收集

```python
from langgraph.graph import StateGraph, START, END
from typing import TypedDict, List, Dict, Any, Callable
from datetime import datetime
import time
import functools
from collections import defaultdict
import statistics

# 定义指标状态
class MetricsState(TypedDict):
    execution_metrics: Dict[str, List[Dict]]  # 执行指标
    resource_metrics: Dict[str, Dict]  # 资源指标
    aggregated_stats: Dict[str, Dict]  # 聚合统计

# 性能指标收集装饰器
def collect_performance_metrics(func):
    """收集函数执行性能指标的装饰器"""
    @functools.wraps(func)
    def wrapper(state: Any, config=None):
        node_name = func.__name__
        start_time = time.time()
        
        try:
            # 记录开始指标
            metrics_entry = {
                "node": node_name,
                "start_time": datetime.now().isoformat(),
                "input_state_keys": list(state.keys()) if isinstance(state, dict) else [],
                "state_size": len(str(state)) if hasattr(state, '__str__') else 0
            }
            
            # 执行节点函数
            result = func(state, config)
            
            # 记录结束指标
            execution_time = time.time() - start_time
            metrics_entry.update({
                "end_time": datetime.now().isoformat(),
                "execution_time": execution_time,
                "status": "success",
                "output_keys": list(result.keys()) if isinstance(result, dict) else [],
                "result_size": len(str(result)) if hasattr(result, '__str__') else 0
            })
            
            # 添加到指标状态
            if isinstance(state, dict) and "execution_metrics" in state:
                if node_name not in state["execution_metrics"]:
                    state["execution_metrics"][node_name] = []
                state["execution_metrics"][node_name].append(metrics_entry)
            
            return result
            
        except Exception as e:
            # 记录错误指标
            execution_time = time.time() - start_time
            error_metrics = {
                "node": node_name,
                "start_time": datetime.fromtimestamp(start_time).isoformat(),
                "end_time": datetime.now().isoformat(),
                "execution_time": execution_time,
                "status": "error",
                "error_type": type(e).__name__,
                "error_message": str(e)
            }
            
            if isinstance(state, dict) and "execution_metrics" in state:
                if node_name not in state["execution_metrics"]:
                    state["execution_metrics"][node_name] = []
                state["execution_metrics"][node_name].append(error_metrics)
            
            raise
    
    return wrapper

# 资源监控装饰器
def monitor_resource_usage(func):
    """监控资源使用情况的装饰器"""
    @functools.wraps(func)
    def wrapper(state: Any, config=None):
        import psutil
        import os
        
        process = psutil.Process(os.getpid())
        
        # 记录开始时的资源使用
        start_cpu = process.cpu_percent(interval=None)
        start_memory = process.memory_info().rss / 1024 / 1024  # MB
        
        result = func(state, config)
        
        # 记录结束时的资源使用
        end_cpu = process.cpu_percent(interval=None)
        end_memory = process.memory_info().rss / 1024 / 1024  # MB
        
        resource_metrics = {
            "node": func.__name__,
            "timestamp": datetime.now().isoformat(),
            "cpu_usage_percent": end_cpu - start_cpu,
            "memory_usage_mb": end_memory - start_memory,
            "memory_peak_mb": process.memory_info().vms / 1024 / 1024
        }
        
        if isinstance(state, dict) and "resource_metrics" in state:
            state["resource_metrics"][func.__name__] = resource_metrics
        
        return result
    
    return wrapper

# 应用装饰器的节点函数示例
@collect_performance_metrics
@monitor_resource_usage
def data_processing_node(state: MetricsState):
    """数据处理节点（示例）"""
    # 模拟数据处理逻辑
    time.sleep(0.1)  # 模拟处理时间
    
    processed_data = {
        "original": state.get("input_data", ""),
        "processed": state.get("input_data", "").upper() if isinstance(state.get("input_data"), str) else None,
        "timestamp": datetime.now().isoformat()
    }
    
    return {"processed_data": processed_data}

@collect_performance_metrics
@monitor_resource_usage  
def validation_node(state: MetricsState):
    """数据验证节点（示例）"""
    # 模拟验证逻辑
    time.sleep(0.05)  # 模拟验证时间
    
    data = state.get("processed_data", {})
    is_valid = bool(data.get("processed"))
    
    return {"validation_result": is_valid, "validated_data": data}

# 指标聚合节点
def aggregate_metrics(state: MetricsState):
    """聚合性能指标"""
    aggregated_stats = {}
    
    # 聚合执行指标
    for node_name, metrics_list in state.get("execution_metrics", {}).items():
        if metrics_list:
            execution_times = [m.get("execution_time", 0) for m in metrics_list if "execution_time" in m]
            success_count = sum(1 for m in metrics_list if m.get("status") == "success")
            error_count = sum(1 for m in metrics_list if m.get("status") == "error")
            
            if execution_times:
                aggregated_stats[node_name] = {
                    "execution_count": len(metrics_list),
                    "success_rate": success_count / len(metrics_list) if len(metrics_list) > 0 else 0,
                    "error_rate": error_count / len(metrics_list) if len(metrics_list) > 0 else 0,
                    "avg_execution_time": statistics.mean(execution_times),
                    "min_execution_time": min(execution_times) if execution_times else 0,
                    "max_execution_time": max(execution_times) if execution_times else 0,
                    "p95_execution_time": statistics.quantiles(execution_times, n=20)[18] if len(execution_times) >= 20 else max(execution_times) if execution_times else 0
                }
    
    # 聚合资源指标
    resource_stats = {}
    for node_name, metrics in state.get("resource_metrics", {}).items():
        resource_stats[node_name] = {
            "cpu_usage": metrics.get("cpu_usage_percent", 0),
            "memory_usage_mb": metrics.get("memory_usage_mb", 0),
            "memory_peak_mb": metrics.get("memory_peak_mb", 0)
        }
    
    aggregated_stats["resource_usage"] = resource_stats
    
    return {"aggregated_stats": aggregated_stats}

# 构建带性能监控的图
def build_monitored_graph():
    workflow = StateGraph(MetricsState)
    
    workflow.add_node("process", data_processing_node)
    workflow.add_node("validate", validation_node)
    workflow.add_node("aggregate", aggregate_metrics)
    
    workflow.add_edge(START, "process")
    workflow.add_edge("process", "validate")
    workflow.add_edge("validate", "aggregate")
    workflow.add_edge("aggregate", END)
    
    return workflow.compile()

# 使用示例
if __name__ == "__main__":
    # 初始化状态
    initial_state = {
        "input_data": "test performance metrics collection",
        "execution_metrics": {},
        "resource_metrics": {},
        "aggregated_stats": {}
    }
    
    # 编译并执行图
    monitored_graph = build_monitored_graph()
    
    # 执行多次以收集足够的指标数据
    for i in range(5):
        result = monitored_graph.invoke(initial_state)
        print(f"执行 {i+1} 完成")
        
        # 更新状态，保留历史指标
        initial_state.update(result)
    
    # 展示聚合指标
    print("\n性能指标汇总:")
    for node_name, stats in result.get("aggregated_stats", {}).items():
        if node_name != "resource_usage":
            print(f"\n节点 {node_name}:")
            print(f"  执行次数: {stats.get('execution_count', 0)}")
            print(f"  成功率: {stats.get('success_rate', 0):.2%}")
            print(f"  平均执行时间: {stats.get('avg_execution_time', 0):.3f}秒")
            print(f"  P95执行时间: {stats.get('p95_execution_time', 0):.3f}秒")
    
    if "resource_usage" in result.get("aggregated_stats", {}):
        print("\n资源使用情况:")
        for node_name, usage in result["aggregated_stats"]["resource_usage"].items():
            print(f"  节点 {node_name}: CPU={usage.get('cpu_usage', 0):.1f}%, 内存={usage.get('memory_usage_mb', 0):.1f}MB")
```

### 示例2：实时性能指标流处理

```python
import asyncio
from typing import Dict, List, Any
from datetime import datetime, timedelta
from collections import deque
import json

class RealTimeMetricsProcessor:
    """实时性能指标处理器"""
    
    def __init__(self, window_size: int = 1000, aggregation_interval: float = 1.0):
        self.window_size = window_size
        self.aggregation_interval = aggregation_interval
        self.metrics_stream = deque(maxlen=window_size)
        self.aggregated_metrics = {}
        self.subscribers = []
        self.processing = False
    
    async def start_processing(self):
        """启动指标处理"""
        self.processing = True
        
        while self.processing:
            try:
                # 处理新到达的指标
                self._process_new_metrics()
                
                # 定期聚合指标
                self._aggregate_metrics()
                
                # 通知订阅者
                self._notify_subscribers()
                
                await asyncio.sleep(self.aggregation_interval)
                
            except Exception as e:
                print(f"指标处理错误: {e}")
                await asyncio.sleep(5)
    
    def stop_processing(self):
        """停止处理"""
        self.processing = False
    
    def add_metrics(self, metrics: Dict):
        """添加性能指标"""
        self.metrics_stream.append({
            **metrics,
            "received_time": datetime.now().isoformat()
        })
    
    def _process_new_metrics(self):
        """处理新指标"""
        if not self.metrics_stream:
            return
        
        # 按节点类型分组
        node_metrics = {}
        
        for metric in list(self.metrics_stream):
            node_type = metric.get("node_type", "unknown")
            if node_type not in node_metrics:
                node_metrics[node_type] = []
            node_metrics[node_type].append(metric)
        
        # 计算基本统计
        for node_type, metrics_list in node_metrics.items():
            if metrics_list:
                execution_times = [m.get("execution_time", 0) for m in metrics_list]
                error_count = sum(1 for m in metrics_list if m.get("status") == "error")
                
                self.aggregated_metrics[node_type] = {
                    "timestamp": datetime.now().isoformat(),
                    "total_requests": len(metrics_list),
                    "error_rate": error_count / len(metrics_list) if len(metrics_list) > 0 else 0,
                    "avg_response_time": sum(execution_times) / len(execution_times) if execution_times else 0,
                    "min_response_time": min(execution_times) if execution_times else 0,
                    "max_response_time": max(execution_times) if execution_times else 0,
                    "throughput": len(metrics_list) / 60.0  # 每分钟请求数
                }
    
    def _aggregate_metrics(self):
        """聚合指标数据"""
        # 计算系统级指标
        total_requests = sum(stats.get("total_requests", 0) for stats in self.aggregated_metrics.values())
        total_errors = sum(stats.get("error_rate", 0) * stats.get("total_requests", 0) for stats in self.aggregated_metrics.values())
        
        if total_requests > 0:
            system_metrics = {
                "timestamp": datetime.now().isoformat(),
                "total_requests": total_requests,
                "system_error_rate": total_errors / total_requests,
                "avg_system_response_time": sum(
                    stats.get("avg_response_time", 0) * stats.get("total_requests", 0) 
                    for stats in self.aggregated_metrics.values()
                ) / total_requests if total_requests > 0 else 0,
                "node_count": len(self.aggregated_metrics),
                "throughput": total_requests / 60.0  # 系统总吞吐量
            }
            
            self.aggregated_metrics["system"] = system_metrics
    
    def _notify_subscribers(self):
        """通知指标订阅者"""
        for callback in self.subscribers:
            try:
                if asyncio.iscoroutinefunction(callback):
                    asyncio.create_task(callback(self.aggregated_metrics))
                else:
                    callback(self.aggregated_metrics)
            except Exception as e:
                print(f"通知订阅者失败: {e}")
    
    def subscribe(self, callback):
        """订阅指标更新"""
        self.subscribers.append(callback)
    
    def unsubscribe(self, callback):
        """取消订阅"""
        if callback in self.subscribers:
            self.subscribers.remove(callback)
    
    def get_recent_metrics(self, node_type: str = None, limit: int = 100) -> List[Dict]:
        """获取最近的指标"""
        if node_type:
            return [
                m for m in list(self.metrics_stream)
                if m.get("node_type") == node_type
            ][-limit:]
        else:
            return list(self.metrics_stream)[-limit:]
    
    def get_aggregated_stats(self, node_type: str = None) -> Dict:
        """获取聚合统计"""
        if node_type:
            return self.aggregated_metrics.get(node_type, {})
        else:
            return self.aggregated_metrics

# 指标收集器集成到LangGraph节点
class MetricsCollectorNode:
    """指标收集器节点"""
    
    def __init__(self, processor: RealTimeMetricsProcessor):
        self.processor = processor
    
    def __call__(self, state: Dict) -> Dict:
        """收集当前节点的性能指标"""
        node_name = self.__class__.__name__
        
        # 收集基本指标
        metrics = {
            "node_type": node_name,
            "timestamp": datetime.now().isoformat(),
            "state_size": len(json.dumps(state)) if isinstance(state, dict) else 0,
            "status": "processing"
        }
        
        # 添加到处理器
        self.processor.add_metrics(metrics)
        
        # 返回状态（不修改）
        return state

# WebSocket指标推送示例
async def metrics_websocket_handler(websocket):
    """WebSocket指标推送处理器"""
    processor = RealTimeMetricsProcessor()
    
    # 定义指标推送回调
    async def push_metrics(metrics):
        """推送指标到WebSocket"""
        try:
            await websocket.send_json({
                "type": "metrics_update",
                "timestamp": datetime.now().isoformat(),
                "data": metrics
            })
        except Exception as e:
            print(f"WebSocket推送失败: {e}")
    
    # 订阅指标更新
    processor.subscribe(push_metrics)
    
    try:
        # 启动指标处理
        processing_task = asyncio.create_task(processor.start_processing())
        
        # 等待连接关闭
        await websocket.wait_closed()
        
    finally:
        # 清理资源
        processing_task.cancel()
        processor.unsubscribe(push_metrics)
```

### 1. 分布式性能指标聚合

```python
from typing import Dict, List, Any
import asyncio
import aiohttp
from datetime import datetime, timedelta
import hashlib
import json

class DistributedMetricsAggregator:
    """分布式性能指标聚合器"""
    
    def __init__(self, node_endpoints: Dict[str, str], aggregation_interval: int = 30):
        self.node_endpoints = node_endpoints  # 节点ID -> 指标端点URL
        self.aggregation_interval = aggregation_interval
        self.aggregated_data = {}
        self.last_aggregation = {}
        self.session = None
    
    async def initialize(self):
        """初始化HTTP会话"""
        self.session = aiohttp.ClientSession()
    
    async def close(self):
        """关闭资源"""
        if self.session:
            await self.session.close()
    
    async def collect_distributed_metrics(self) -> Dict[str, Any]:
        """收集所有节点的分布式指标"""
        collection_tasks = []
        
        for node_id, endpoint in self.node_endpoints.items():
            task = self._fetch_node_metrics(node_id, endpoint)
            collection_tasks.append(task)
        
        # 并行收集所有节点指标
        results = await asyncio.gather(*collection_tasks, return_exceptions=True)
        
        # 处理结果
        collected_metrics = {}
        for i, (node_id, endpoint) in enumerate(self.node_endpoints.items()):
            result = results[i]
            
            if isinstance(result, Exception):
                print(f"收集节点 {node_id} 指标失败: {result}")
                collected_metrics[node_id] = {
                    "status": "error",
                    "error": str(result),
                    "timestamp": datetime.now().isoformat()
                }
            else:
                collected_metrics[node_id] = result
        
        return collected_metrics
    
    async def _fetch_node_metrics(self, node_id: str, endpoint: str) -> Dict:
        """获取单个节点的指标"""
        try:
            async with self.session.get(f"{endpoint}/metrics", timeout=10) as response:
                if response.status == 200:
                    metrics = await response.json()
                    return {
                        "status": "success",
                        "node_id": node_id,
                        "metrics": metrics,
                        "timestamp": datetime.now().isoformat(),
                        "response_time": response.elapsed.total_seconds()
                    }
                else:
                    raise Exception(f"HTTP {response.status}: {await response.text()}")
        except asyncio.TimeoutError:
            raise Exception(f"请求超时: {endpoint}")
        except Exception as e:
            raise Exception(f"请求失败: {str(e)}")
    
    def aggregate_metrics(self, collected_metrics: Dict) -> Dict:
        """聚合所有节点的指标"""
        aggregation_time = datetime.now()
        
        # 初始化聚合结果
        aggregated = {
            "aggregation_time": aggregation_time.isoformat(),
            "total_nodes": len(self.node_endpoints),
            "healthy_nodes": 0,
            "unhealthy_nodes": 0,
            "node_metrics": {},
            "system_metrics": {}
        }
        
        # 处理每个节点的指标
        for node_id, node_data in collected_metrics.items():
            aggregated["node_metrics"][node_id] = node_data
            
            if node_data.get("status") == "success":
                aggregated["healthy_nodes"] += 1
                
                # 提取性能指标
                metrics = node_data.get("metrics", {})
                if "performance" in metrics:
                    aggregated["node_metrics"][node_id]["performance"] = metrics["performance"]
            else:
                aggregated["unhealthy_nodes"] += 1
        
        # 计算系统级指标
        if aggregated["total_nodes"] > 0:
            aggregated["system_metrics"] = {
                "availability": aggregated["healthy_nodes"] / aggregated["total_nodes"],
                "avg_response_time": self._calculate_avg_response_time(collected_metrics),
                "throughput": self._calculate_system_throughput(collected_metrics),
                "error_rate": aggregated["unhealthy_nodes"] / aggregated["total_nodes"]
            }
        
        # 检测性能异常
        aggregated["anomalies"] = self._detect_performance_anomalies(collected_metrics)
        
        # 保存聚合结果
        self.aggregated_data[aggregation_time] = aggregated
        self.last_aggregation = aggregated
        
        return aggregated
    
    def _calculate_avg_response_time(self, metrics: Dict) -> float:
        """计算平均响应时间"""
        response_times = []
        
        for node_data in metrics.values():
            if node_data.get("status") == "success":
                perf_metrics = node_data.get("metrics", {}).get("performance", {})
                if "response_time" in perf_metrics:
                    response_times.append(perf_metrics["response_time"])
                elif "response_time" in node_data:
                    response_times.append(node_data["response_time"])
        
        return sum(response_times) / len(response_times) if response_times else 0
    
    def _calculate_system_throughput(self, metrics: Dict) -> float:
        """计算系统吞吐量"""
        total_throughput = 0
        
        for node_data in metrics.values():
            if node_data.get("status") == "success":
                perf_metrics = node_data.get("metrics", {}).get("performance", {})
                if "throughput" in perf_metrics:
                    total_throughput += perf_metrics["throughput"]
        
        return total_throughput
    
    def _detect_performance_anomalies(self, metrics: Dict) -> List[Dict]:
        """检测性能异常"""
        anomalies = []
        
        for node_id, node_data in metrics.items():
            if node_data.get("status") == "success":
                perf_metrics = node_data.get("metrics", {}).get("performance", {})
                
                # 检测响应时间异常
                if "response_time" in perf_metrics:
                    response_time = perf_metrics["response_time"]
                    
                    # 简单阈值检测（实际应用中可以使用更复杂的算法）
                    if response_time > 5.0:  # 响应时间超过5秒
                        anomalies.append({
                            "node_id": node_id,
                            "type": "high_response_time",
                            "severity": "warning",
                            "value": response_time,
                            "threshold": 5.0,
                            "timestamp": datetime.now().isoformat()
                        })
                
                # 检测错误率异常
                if "error_rate" in perf_metrics:
                    error_rate = perf_metrics["error_rate"]
                    
                    if error_rate > 0.1:  # 错误率超过10%
                        anomalies.append({
                            "node_id": node_id,
                            "type": "high_error_rate",
                            "severity": "critical",
                            "value": error_rate,
                            "threshold": 0.1,
                            "timestamp": datetime.now().isoformat()
                        })
        
        return anomalies
    
    async def start_periodic_aggregation(self):
        """启动周期性聚合"""
        await self.initialize()
        
        try:
            while True:
                # 收集分布式指标
                collected_metrics = await self.collect_distributed_metrics()
                
                # 聚合指标
                aggregated = self.aggregate_metrics(collected_metrics)
                
                print(f"聚合完成: {aggregated['aggregation_time']}")
                print(f"健康节点: {aggregated['healthy_nodes']}/{aggregated['total_nodes']}")
                print(f"系统可用性: {aggregated['system_metrics'].get('availability', 0):.2%}")
                
                # 等待下一个聚合周期
                await asyncio.sleep(self.aggregation_interval)
                
        except Exception as e:
            print(f"周期性聚合失败: {e}")
        finally:
            await self.close()
```

```python
import pandas as pd
import numpy as np
from datetime import datetime, timedelta
from typing import Dict, List, Tuple
import matplotlib.pyplot as plt

class MetricsCorrelationAnalyzer:
    """性能指标关联分析器"""
    
    def __init__(self, metrics_history: List[Dict]):
        self.metrics_history = metrics_history
        self.df = self._create_dataframe()
        self.correlation_matrix = None
    
    def _create_dataframe(self) -> pd.DataFrame:
        """创建分析数据框"""
        data = []
        
        for metrics in self.metrics_history:
            row = {
                "timestamp": pd.to_datetime(metrics.get("timestamp")),
                "response_time": metrics.get("response_time", 0),
                "throughput": metrics.get("throughput", 0),
                "error_rate": metrics.get("error_rate", 0),
                "cpu_usage": metrics.get("cpu_usage", 0),
                "memory_usage": metrics.get("memory_usage", 0),
                "concurrent_requests": metrics.get("concurrent_requests", 0),
                "queue_length": metrics.get("queue_length", 0)
            }
            data.append(row)
        
        return pd.DataFrame(data)
    
    def calculate_correlations(self) -> pd.DataFrame:
        """计算指标相关性"""
        numeric_columns = self.df.select_dtypes(include=[np.number]).columns
        self.correlation_matrix = self.df[numeric_columns].corr()
        
        return self.correlation_matrix
    
    def identify_key_correlations(self, threshold: float = 0.7) -> List[Dict]:
        """识别关键相关性"""
        if self.correlation_matrix is None:
            self.calculate_correlations()
        
        key_correlations = []
        corr_matrix = self.correlation_matrix
        
        # 获取所有相关性对
        for i in range(len(corr_matrix.columns)):
            for j in range(i+1, len(corr_matrix.columns)):
                col1 = corr_matrix.columns[i]
                col2 = corr_matrix.columns[j]
                correlation = corr_matrix.iloc[i, j]
                
                if abs(correlation) >= threshold:
                    key_correlations.append({
                        "metric1": col1,
                        "metric2": col2,
                        "correlation": correlation,
                        "strength": "strong" if abs(correlation) >= 0.8 else "moderate",
                        "direction": "positive" if correlation > 0 else "negative"
                    })
        
        # 按相关性强度排序
        key_correlations.sort(key=lambda x: abs(x["correlation"]), reverse=True)
        
        return key_correlations
    
    def analyze_performance_bottlenecks(self) -> Dict:
        """分析性能瓶颈"""
        analysis = {
            "timestamp": datetime.now().isoformat(),
            "bottlenecks": [],
            "recommendations": []
        }
        
        # 分析响应时间相关因素
        if "response_time" in self.correlation_matrix.columns:
            response_correlations = self.correlation_matrix["response_time"].sort_values(ascending=False)
            
            for metric, correlation in response_correlations.items():
                if metric != "response_time" and abs(correlation) > 0.5:
                    if correlation > 0:  # 正相关
                        analysis["bottlenecks"].append({
                            "type": "response_time_bottleneck",
                            "metric": metric,
                            "correlation": correlation,
                            "description": f"响应时间与{metric}高度正相关，{metric}增加可能导致响应时间增加"
                        })
                        
                        if metric == "concurrent_requests":
                            analysis["recommendations"].append({
                                "type": "concurrency_optimization",
                                "description": "考虑优化并发处理能力或实现限流机制",
                                "priority": "high" if correlation > 0.8 else "medium"
                            })
                    
                    elif correlation < -0.5:  # 强负相关
                        analysis["bottlenecks"].append({
                            "type": "inverse_relationship",
                            "metric": metric,
                            "correlation": correlation,
                            "description": f"响应时间与{metric}高度负相关，{metric}增加可能改善响应时间"
                        })
        
        # 分析错误率相关因素
        if "error_rate" in self.correlation_matrix.columns:
            error_correlations = self.correlation_matrix["error_rate"].sort_values(ascending=False)
            
            for metric, correlation in error_correlations.items():
                if metric != "error_rate" and abs(correlation) > 0.6:
                    analysis["bottlenecks"].append({
                        "type": "error_rate_bottleneck",
                        "metric": metric,
                        "correlation": correlation,
                        "description": f"错误率与{metric}高度相关，{metric}变化可能影响系统稳定性"
                    })
        
        return analysis
    
    def plot_correlation_heatmap(self, save_path: str = None):
        """绘制相关性热力图"""
        if self.correlation_matrix is None:
            self.calculate_correlations()
        
        fig, ax = plt.subplots(figsize=(10, 8))
        
        # 创建热力图
        cax = ax.matshow(self.correlation_matrix, cmap='coolwarm', vmin=-1, vmax=1)
        fig.colorbar(cax)
        
        # 设置刻度标签
        ax.set_xticks(range(len(self.correlation_matrix.columns)))
        ax.set_yticks(range(len(self.correlation_matrix.columns)))
        ax.set_xticklabels(self.correlation_matrix.columns, rotation=45, ha='left')
        ax.set_yticklabels(self.correlation_matrix.columns)
        
        # 添加相关性数值
        for i in range(len(self.correlation_matrix.columns)):
            for j in range(len(self.correlation_matrix.columns)):
                value = self.correlation_matrix.iloc[i, j]
                ax.text(j, i, f'{value:.2f}', ha='center', va='center', 
                       color='white' if abs(value) > 0.5 else 'black')
        
        ax.set_title('性能指标相关性热力图')
        plt.tight_layout()
        
        if save_path:
            plt.savefig(save_path, dpi=300, bbox_inches='tight')
        
        plt.show()
    
    def generate_performance_report(self) -> Dict:
        """生成性能分析报告"""
        report = {
            "generated_at": datetime.now().isoformat(),
            "data_points": len(self.df),
            "time_range": {
                "start": self.df["timestamp"].min().isoformat(),
                "end": self.df["timestamp"].max().isoformat(),
                "duration_days": (self.df["timestamp"].max() - self.df["timestamp"].min()).days
            },
            "summary_statistics": self._calculate_summary_stats(),
            "key_correlations": self.identify_key_correlations(),
            "bottleneck_analysis": self.analyze_performance_bottlenecks(),
            "recommendations": []
        }
        
        # 基于分析结果生成建议
        bottlenecks = report["bottleneck_analysis"]["bottlenecks"]
        
        for bottleneck in bottlenecks:
            if bottleneck["type"] == "response_time_bottleneck":
                if bottleneck["metric"] == "cpu_usage":
                    report["recommendations"].append({
                        "category": "resource_optimization",
                        "title": "优化CPU使用",
                        "description": "响应时间与CPU使用率高度相关，考虑优化算法或增加计算资源",
                        "priority": "high" if bottleneck["correlation"] > 0.8 else "medium"
                    })
                elif bottleneck["metric"] == "queue_length":
                    report["recommendations"].append({
                        "category": "queue_management",
                        "title": "优化队列管理",
                        "description": "响应时间与队列长度高度相关，考虑实现更智能的队列调度或增加处理能力",
                        "priority": "high"
                    })
        
        return report
    
    def _calculate_summary_stats(self) -> Dict:
        """计算汇总统计"""
        stats = {}
        
        for column in self.df.select_dtypes(include=[np.number]).columns:
            stats[column] = {
                "mean": float(self.df[column].mean()),
                "median": float(self.df[column].median()),
                "std": float(self.df[column].std()),
                "min": float(self.df[column].min()),
                "max": float(self.df[column].max()),
                "p95": float(self.df[column].quantile(0.95))
            }
        
        return stats
```

## 常见问题

### Q1: 如何选择关键性能指标？

**回答**：选择关键性能指标的原则：

1. **业务相关性**：指标应直接反映业务价值
   - 用户体验：响应时间、成功率
   - 业务目标：转化率、交易量
   - 成本效益：资源使用效率

2. **可操作性**：指标应能够指导具体行动
   - 阈值明确：能够设置合理的警报阈值
   - 根因可查：问题发生时能够定位原因
   - 改进可测：优化措施的效果可以验证

3. **平衡性**：覆盖不同维度的指标
   - 效率指标：响应时间、吞吐量
   - 质量指标：错误率、数据一致性
   - 资源指标：CPU、内存、网络使用
   - 容量指标：并发数、队列长度

4. **成本效益**：收集成本与价值平衡
   - 高频采集：关键核心指标
   - 低频采集：次要辅助指标
   - 按需采集：调试和诊断指标

**建议**：
- 从黄金信号开始：延迟、流量、错误、饱和度
- 根据业务特点定制关键指标
- 定期审查和调整指标集合
- 建立指标层次结构（业务→应用→系统）

### Q2: 性能指标数据一致性如何保证？

**回答**：保证性能指标数据一致性的策略：

1. **时间同步**
   - 使用NTP同步所有节点时间
   - 记录协调世界时（UTC）时间戳
   - 实现逻辑时钟处理分布式事件排序

2. **数据完整性**
   - 使用校验和验证数据传输完整性
   - 实现数据版本控制和冲突解决
   - 定期进行数据一致性检查

3. **采样一致性**
   - 确保采样时间窗口对齐
   - 处理边界条件和数据丢失
   - 实现数据插值和补全机制

4. **聚合一致性**
   - 使用一致性哈希确保数据路由一致
   - 实现最终一致性或强一致性聚合
   - 处理网络分区和节点故障

**技术方案**：
- 使用向量时钟跟踪分布式事件顺序
- 实现CRDT（无冲突复制数据类型）
- 采用两阶段提交确保事务一致性
- 使用流处理框架保证事件顺序

### Q3: 性能指标收集对系统性能的影响如何最小化？

**回答**：最小化性能影响的技术：

1. **异步收集**
   - 使用异步I/O避免阻塞主线程
   - 实现非阻塞的数据收集接口
   - 使用消息队列解耦收集和处理

2. **采样优化**
   - 根据系统负载动态调整采样率
   - 实现分层采样（重要指标高频，次要指标低频）
   - 使用随机采样减少数据量

3. **内存优化**
   - 使用环形缓冲区避免内存无限增长
   - 实现数据压缩和聚合
   - 定期清理历史数据

4. **计算优化**
   - 使用增量计算避免重复计算
   - 实现近似算法降低计算复杂度
   - 使用缓存减少重复数据收集

**最佳实践**：
- 监控监控系统本身的性能
- 实现降级和熔断机制
- 分离监控网络和业务网络
- 定期进行性能影响评估
