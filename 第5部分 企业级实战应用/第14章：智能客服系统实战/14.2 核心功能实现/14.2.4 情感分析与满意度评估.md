# 14.2.4 情感分析与满意度评估

## 概念讲解

情感分析与满意度评估是智能客服系统的重要组成部分，它通过分析用户对话中的情感倾向和满意度水平，为企业提供用户反馈的量化指标和洞察。在LangChain v1.2.22框架中，情感分析通过结合传统NLP技术和LLM的语义理解能力，实现了精准、多维度的情感识别和满意度评估。

### 情感分析技术的发展

**情感分析技术演进**：
```
词典方法 → 机器学习 → 深度学习 → 大语言模型 → 多模态分析
├── 情感词典匹配    ├── 监督学习分类    ├── 神经网络模型    ├── 上下文理解    ├── 文本+语音+图像
├── 简单快速        ├── 需要标注数据    ├── 自动特征学习    ├── 零样本学习    ├── 综合情感理解
├── 准确率有限      ├── 特征工程复杂    ├── 计算资源要求高    ├── 语义理解强    ├── 全面情感捕捉
└── 难以处理复杂句  └── 泛化能力中等    └── 可解释性差      └── 成本相对较高    └── 技术集成复杂
```

**LangChain情感分析优势**：
1. **多方法融合**：支持词典、机器学习、深度学习、LLM多种分析方法
2. **细粒度分析**：支持情感极性、强度、维度、目标的多层次分析
3. **实时处理**：支持流式情感分析和实时满意度跟踪
4. **可解释性**：提供情感分析的可解释性和置信度评估

### 智能客服情感分析体系

**情感分析多层次架构**：
```
原始对话 → 预处理 → 情感检测 → 情感分类 → 强度分析 → 目标识别 → 满意度评估
```

## 核心要点

### 1. LangChain v1.2.22情感分析组件

**LangChain情感分析工具集**：

| 组件类型 | 具体实现 | 核心功能 | 适用场景 |
|---------|----------|----------|----------|
| **词典分析法** | `SentimentLexiconAnalyzer`, `VADER` | 基于情感词典的分析 | 快速简单情感分析 |
| **机器学习法** | `SentimentClassifier`, `TextClassifier` | 训练分类模型分析 | 定制化情感分析 |
| **深度学法** | `BERTSentiment`, `TransformersAnalyzer` | 深度学习模型分析 | 高精度情感分析 |
| **LLM分析法** | `LLMSentimentAnalyzer`, `ChatModel` | LLM语义分析 | 复杂语境情感分析 |
| **混合分析法** | 多种方法组合 | 综合情感分析 | 高要求场景 |

**情感分析维度设计**：
1. **情感极性**：正面(Positive)、负面(Negative)、中性(Neutral)
2. **情感强度**：弱(Weak)、中等(Medium)、强(Strong)、非常强(Very Strong)
3. **情感维度**：喜悦(Joy)、悲伤(Sadness)、愤怒(Anger)、恐惧(Fear)、惊讶(Surprise)
4. **情感目标**：产品(Product)、服务(Service)、价格(Price)、交付(Delivery)、客服(Support)

### 2. 满意度评估指标体系

**智能客服满意度评估模型**：

```python
用户满意度 = 基础满意度 + 情感调节因子 + 问题解决因子 + 体验优化因子
├── 基础满意度        ├── 情感极性权重    ├── 问题解决速度    ├── 交互体验评分
├── 响应速度评分      ├── 情感强度影响    ├── 解决方案质量    ├── 界面友好度
├── 回答准确率        ├── 情感目标关联    ├── 问题复杂程度    ├── 流程顺畅度
├── 服务态度评价      ├── 情感演变趋势    ├── 解决彻底性      ├── 个性化程度
└── 专业能力评估      └── 情感一致性      └── 用户确认度      └── 预期超越度
```

**关键满意度指标**：
1. **NPS(净推荐值)**：用户推荐意愿的量化指标
2. **CES(客户费力度)**：解决问题所需努力的感知程度
3. **CSAT(客户满意度)**：单次交互的满意度评分
4. **情感趋势**：用户情感随时间的变化趋势
5. **问题闭环率**：问题彻底解决的比例

### 3. 情感驱动的客服优化策略

**情感智能响应机制**：
- **情感感知**：实时识别用户情感状态
- **情感适配**：根据用户情感调整响应策略
- **情感安抚**：对负面情感的针对性安抚
- **情感增强**：对正面情感的积极强化
- **情感转介**：极端情感时转人工客服

**满意度预警与干预**：
- **满意度阈值**：设置满意度预警阈值
- **实时监控**：实时监控满意度变化
- **预警触发**：满意度下降时触发预警
- **干预措施**：自动或人工干预措施
- **效果跟踪**：干预效果的跟踪评估

```python
from typing import List, Dict, Optional, Any, Tuple
from dataclasses import dataclass
from enum import Enum
import time
from datetime import datetime

class SentimentPolarity(Enum):
    """情感极性枚举"""
    POSITIVE = "positive"
    NEGATIVE = "negative"
    NEUTRAL = "neutral"
    MIXED = "mixed"

class SentimentIntensity(Enum):
    """情感强度枚举"""
    VERY_WEAK = "very_weak"
    WEAK = "weak"
    MEDIUM = "medium"
    STRONG = "strong"
    VERY_STRONG = "very_strong"

class SentimentDimension(Enum):
    """情感维度枚举"""
    JOY = "joy"
    SADNESS = "sadness"
    ANGER = "anger"
    FEAR = "fear"
    SURPRISE = "surprise"
    DISGUST = "disgust"
    ANTICIPATION = "anticipation"
    TRUST = "trust"

@dataclass
class SentimentAnalysisResult:
    """情感分析结果"""
    text: str
    polarity: SentimentPolarity
    polarity_score: float
    intensity: SentimentIntensity
    intensity_score: float
    dimensions: Dict[SentimentDimension, float]
    entities: List[Dict[str, Any]]
    analysis_method: str
    confidence: float
    timestamp: datetime

@dataclass  
class SatisfactionAssessment:
    """满意度评估结果"""
    conversation_id: str
    user_id: str
    csat_score: float  # 客户满意度分数 0-100
    nps_category: str  # 推荐者/被动者/贬低者

    key_issues: List[str]

    overall_rating: str  # 优秀/良好/一般/差

class SentimentAnalyzer:
    """情感分析器基类"""
    
    def analyze(self, text: str) -> SentimentAnalysisResult:
        """分析文本情感"""
        raise NotImplementedError
    
    def batch_analyze(self, texts: List[str]) -> List[SentimentAnalysisResult]:
        """批量分析文本情感"""
        return [self.analyze(text) for text in texts]

class LexiconBasedAnalyzer(SentimentAnalyzer):
    """基于词典的情感分析器"""
    
    def __init__(self):
        # 初始化情感词典（简化版）
        self.positive_words = {
            "好", "优秀", "满意", "感谢", "谢谢", "棒", "不错", "愉快", "顺利",
            "高效", "专业", "友好", "快速", "准确", "解决", "帮助", "支持"
        }
        
        self.negative_words = {
            "差", "不好", "不满意", "糟糕", "问题", "错误", "投诉", "失望",
            "慢", "困难", "复杂", "麻烦", "等待", "失败", "错误", "故障"
        }
        
        self.intensity_modifiers = {
            "非常": 1.5, "很": 1.3, "特别": 1.4, "极其": 1.6, "超级": 1.5,
            "有点": 0.7, "稍微": 0.8, "略微": 0.8, "基本": 0.9
        }
    
    def analyze(self, text: str) -> SentimentAnalysisResult:
        """基于词典分析情感"""
        
        # 初始化分数
        positive_score = 0
        negative_score = 0
        
        # 分词（简化版）
        words = text.split()
        
        # 计算情感分数
        for i, word in enumerate(words):
            word_lower = word.lower()
            
            # 检查情感词
            if word_lower in self.positive_words:
                score = 1.0
                # 检查强度修饰词
                if i > 0 and words[i-1] in self.intensity_modifiers:
                    score *= self.intensity_modifiers[words[i-1]]
                positive_score += score
                
            elif word_lower in self.negative_words:
                score = 1.0
                if i > 0 and words[i-1] in self.intensity_modifiers:
                    score *= self.intensity_modifiers[words[i-1]]
                negative_score += score
        
        # 确定情感极性
        total_score = positive_score + negative_score
        if total_score == 0:
            polarity = SentimentPolarity.NEUTRAL
            polarity_score = 0.5
        elif positive_score > negative_score:
            polarity = SentimentPolarity.POSITIVE
            polarity_score = positive_score / total_score
        elif negative_score > positive_score:
            polarity = SentimentPolarity.NEGATIVE
            polarity_score = negative_score / total_score
        else:
            polarity = SentimentPolarity.MIXED
            polarity_score = 0.5
        
        # 确定情感强度
        max_score = max(positive_score, negative_score)
        if max_score == 0:
            intensity = SentimentIntensity.VERY_WEAK
            intensity_score = 0.1
        elif max_score <= 1:
            intensity = SentimentIntensity.WEAK
            intensity_score = 0.3
        elif max_score <= 2:
            intensity = SentimentIntensity.MEDIUM
            intensity_score = 0.5
        elif max_score <= 3:
            intensity = SentimentIntensity.STRONG
            intensity_score = 0.7
        else:
            intensity = SentimentIntensity.VERY_STRONG
            intensity_score = 0.9
        
        # 简化的维度分析
        dimensions = {
            SentimentDimension.JOY: positive_score / (positive_score + 1),
            SentimentDimension.ANGER: negative_score / (negative_score + 1) * 0.8,
            SentimentDimension.SADNESS: negative_score / (negative_score + 1) * 0.6,
            SentimentDimension.TRUST: positive_score / (positive_score + 1) * 0.7
        }
        
        # 实体提取（简化版）
        entities = self._extract_entities(text)
        
        return SentimentAnalysisResult(
            text=text,
            polarity=polarity,
            polarity_score=polarity_score,
            intensity=intensity,
            intensity_score=intensity_score,
            dimensions=dimensions,
            entities=entities,
            analysis_method="lexicon_based",
            confidence=0.7,
            timestamp=datetime.now()
        )
    
    def _extract_entities(self, text: str) -> List[Dict[str, Any]]:
        """提取实体（简化版）"""
        entities = []
        
        # 识别常见实体类型
        import re
        
        patterns = {
            "product": r"产品[：:]\s*([\u4e00-\u9fa5]+)",
            "order": r"订单[：:]\s*([A-Z0-9]+)",
            "service": r"服务|客服|支持",
            "price": r"价格|费用|价钱|成本"
        }
        
        for entity_type, pattern in patterns.items():
            if re.search(pattern, text):
                entities.append({
                    "type": entity_type,
                    "value": "detected",
                    "source": "regex"
                })
        
        return entities

class LLMBasedAnalyzer(SentimentAnalyzer):
    """基于LLM的情感分析器"""
    
    def __init__(self, model_name: str = "gpt-3.5-turbo"):
        from langchain_openai import ChatOpenAI
        from langchain_core.prompts import ChatPromptTemplate
        
        self.llm = ChatOpenAI(model=model_name, temperature=0)
        
        # 定义情感分析提示模板
        self.analysis_prompt = ChatPromptTemplate.from_messages([
            ("system", """你是一个专业的情感分析助手。请分析以下文本的情感信息，并按照指定格式返回分析结果。

请分析以下情感维度：
1. 情感极性：正面(positive)、负面(negative)、中性(neutral)、混合(mixed)
2. 情感强度：非常弱(very_weak)、弱(weak)、中等(medium)、强(strong)、非常强(very_strong)
3. 情感维度：喜悦(joy)、悲伤(sadness)、愤怒(anger)、恐惧(fear)、惊讶(surprise)、厌恶(disgust)、期待(anticipation)、信任(trust)

请严格按照以下JSON格式返回结果：
{{
    "polarity": "情感极性",
    "polarity_score": 0.0到1.0之间的分数,
    "intensity": "情感强度", 
    "intensity_score": 0.0到1.0之间的分数,
    "dimensions": {{
        "joy": 0.0到1.0之间的分数,
        "sadness": 0.0到1.0之间的分数,
        "anger": 0.0到1.0之间的分数,
        "fear": 0.0到1.0之间的分数,
        "surprise": 0.0到1.0之间的分数,
        "disgust": 0.0到1.0之间的分数,
        "anticipation": 0.0到1.0之间的分数,
        "trust": 0.0到1.0之间的分数
    }},
    "entities": [
        {{"type": "实体类型", "value": "实体值", "sentiment": "实体相关情感"}}
    ],
    "confidence": 0.0到1.0之间的置信度
}}

请只返回JSON，不要有其他内容。"""),
            ("human", "分析文本：{text}")
        ])
    
    def analyze(self, text: str) -> SentimentAnalysisResult:
        """使用LLM分析情感"""
        
        try:
            # 调用LLM进行分析
            chain = self.analysis_prompt | self.llm
            response = chain.invoke({"text": text})
            
            # 解析响应
            import json
            result_json = json.loads(response.content)
            
            # 映射枚举值
            polarity = SentimentPolarity(result_json["polarity"])
            intensity = SentimentIntensity(result_json["intensity"])
            
            # 转换维度分数
            dimensions = {}
            for dim_name, score in result_json["dimensions"].items():
                try:
                    dimension = SentimentDimension(dim_name)
                    dimensions[dimension] = score
                except ValueError:
                    continue
            
            return SentimentAnalysisResult(
                text=text,
                polarity=polarity,
                polarity_score=result_json["polarity_score"],
                intensity=intensity,
                intensity_score=result_json["intensity_score"],
                dimensions=dimensions,
                entities=result_json["entities"],
                analysis_method="llm_based",
                confidence=result_json["confidence"],
                timestamp=datetime.now()
            )
            
        except Exception as e:
            # 出错时返回默认结果
            print(f"LLM情感分析失败: {e}")
            return self._fallback_analysis(text)
    
    def _fallback_analysis(self, text: str) -> SentimentAnalysisResult:
        """后备分析方法"""
        return SentimentAnalysisResult(
            text=text,
            polarity=SentimentPolarity.NEUTRAL,
            polarity_score=0.5,
            intensity=SentimentIntensity.MEDIUM,
            intensity_score=0.5,
            dimensions={},
            entities=[],
            analysis_method="fallback",
            confidence=0.3,
            timestamp=datetime.now()
        )

class SatisfactionEvaluator:
    """满意度评估器"""
    
    def __init__(self):
        # 满意度评估权重
        self.weights = {
            "response_speed": 0.15,
            "answer_accuracy": 0.25,
            "service_attitude": 0.20,
            "problem_solving": 0.30,
            "overall_experience": 0.10
        }
        
        # 情感影响因子
        self.sentiment_factors = {
            SentimentPolarity.POSITIVE: 1.2,
            SentimentPolarity.NEGATIVE: 0.6,
            SentimentPolarity.NEUTRAL: 1.0,
            SentimentPolarity.MIXED: 0.9
        }
        
        # 强度影响因子
        self.intensity_factors = {
            SentimentIntensity.VERY_WEAK: 0.8,
            SentimentIntensity.WEAK: 0.9,
            SentimentIntensity.MEDIUM: 1.0,
            SentimentIntensity.STRONG: 1.1,
            SentimentIntensity.VERY_STRONG: 1.2
        }
    
    def evaluate_conversation(self, 
                             conversation_data: Dict[str, Any],
                             sentiment_results: List[SentimentAnalysisResult]) -> SatisfactionAssessment:
        """评估对话满意度"""
        
        # 计算基础满意度分数
        base_scores = self._calculate_base_scores(conversation_data)
        
        # 计算情感调整因子
        sentiment_adjustment = self._calculate_sentiment_adjustment(sentiment_results)
        
        # 计算问题解决因子
        problem_solving_factor = self._calculate_problem_solving_factor(conversation_data)
        
        # 计算体验优化因子
        experience_factor = self._calculate_experience_factor(conversation_data)
        
        # 计算综合满意度分数
        csat_score = self._calculate_csat_score(
            base_scores, 
            sentiment_adjustment, 
            problem_solving_factor, 
            experience_factor
        )
        
        # 确定NPS分类
        nps_category = self._determine_nps_category(csat_score)
        
        # 计算CES分数
        ces_score = self._calculate_ces_score(conversation_data)
        
        # 提取关键问题
        key_issues = self._extract_key_issues(conversation_data, sentiment_results)
        
        # 生成改进建议
        improvement_suggestions = self._generate_improvement_suggestions(
            base_scores, key_issues, csat_score
        )
        
        # 确定总体评级
        overall_rating = self._determine_overall_rating(csat_score)
        
        # 计算情感趋势
        sentiment_trend = [result.polarity_score for result in sentiment_results]
        
        return SatisfactionAssessment(
            conversation_id=conversation_data.get("conversation_id", "unknown"),
            user_id=conversation_data.get("user_id", "anonymous"),
            csat_score=csat_score,
            nps_category=nps_category,
            ces_score=ces_score,
            sentiment_trend=sentiment_trend,
            key_issues=key_issues,
            improvement_suggestions=improvement_suggestions,
            overall_rating=overall_rating,
            assessment_time=datetime.now()
        )
    
    def _calculate_base_scores(self, conversation_data: Dict[str, Any]) -> Dict[str, float]:
        """计算基础满意度分数"""
        # 这里简化为模拟计算
        # 在实际应用中，这里会有更复杂的计算逻辑
        
        return {
            "response_speed": 85.0,  # 响应速度评分
            "answer_accuracy": 90.0,  # 回答准确率
            "service_attitude": 88.0,  # 服务态度评价
            "problem_solving": 92.0,  # 问题解决能力
            "overall_experience": 87.0  # 整体体验
        }
    
    def _calculate_sentiment_adjustment(self, sentiment_results: List[SentimentAnalysisResult]) -> float:
        """计算情感调整因子"""
        if not sentiment_results:
            return 1.0
        
        # 计算平均情感分数
        avg_polarity_score = sum(r.polarity_score for r in sentiment_results) / len(sentiment_results)
        
        # 根据平均分数计算调整因子
        if avg_polarity_score >= 0.7:
            return 1.15  # 正面情感增强
        elif avg_polarity_score >= 0.5:
            return 1.05  # 略正面
        elif avg_polarity_score >= 0.3:
            return 0.95  # 略负面
        else:
            return 0.85  # 负面情感减弱
    
    def _calculate_problem_solving_factor(self, conversation_data: Dict[str, Any]) -> float:
        """计算问题解决因子"""
        # 简化的计算逻辑
        problem_solved = conversation_data.get("problem_solved", True)
        resolution_time = conversation_data.get("resolution_time_minutes", 10)
        
        if problem_solved and resolution_time <= 5:
            return 1.2
        elif problem_solved and resolution_time <= 15:
            return 1.1
        elif problem_solved:
            return 1.0
        else:
            return 0.7
    
    def _calculate_experience_factor(self, conversation_data: Dict[str, Any]) -> float:
        """计算体验优化因子"""
        # 简化的计算逻辑
        factors = []
        
        # 交互流畅度
        turn_count = conversation_data.get("turn_count", 5)
        if turn_count <= 5:
            factors.append(1.1)  # 交互简洁
        elif turn_count <= 10:
            factors.append(1.0)  # 正常交互
        else:
            factors.append(0.9)  # 交互冗长
        
        # 个性化程度
        personalized = conversation_data.get("personalized", False)
        factors.append(1.05 if personalized else 1.0)
        
        # 计算平均因子
        return sum(factors) / len(factors)
    
    def _calculate_csat_score(self, 
                             base_scores: Dict[str, float],
                             sentiment_adjustment: float,
                             problem_solving_factor: float,
                             experience_factor: float) -> float:
        """计算CSAT分数"""
        
        # 计算加权平均分
        weighted_sum = 0
        total_weight = 0
        
        for metric, weight in self.weights.items():
            if metric in base_scores:
                weighted_sum += base_scores[metric] * weight
                total_weight += weight
        
        if total_weight == 0:
            base_score = 80.0  # 默认分数
        else:
            base_score = weighted_sum / total_weight
        
        # 应用调整因子
        adjusted_score = base_score * sentiment_adjustment * problem_solving_factor * experience_factor
        
        # 确保分数在0-100之间
        return max(0, min(100, adjusted_score))
    
    def _determine_nps_category(self, csat_score: float) -> str:
        """确定NPS分类"""
        if csat_score >= 90:
            return "promoter"  # 推荐者
        elif csat_score >= 70:
            return "passive"   # 被动者
        else:
            return "detractor" # 贬低者
    
    def _calculate_ces_score(self, conversation_data: Dict[str, Any]) -> float:
        """计算CES分数（1-7分）"""
        # 简化的计算逻辑
        effort_level = conversation_data.get("user_effort_level", 3)  # 1-5分
        
        # 转换为1-7分制
        ces_score = 1 + (effort_level - 1) * (6 / 4)
        return min(7, max(1, ces_score))
    
    def _extract_key_issues(self, 
                           conversation_data: Dict[str, Any],
                           sentiment_results: List[SentimentAnalysisResult]) -> List[str]:
        """提取关键问题"""
        issues = []
        
        # 从对话数据中提取问题
        if "user_issues" in conversation_data:
            issues.extend(conversation_data["user_issues"])
        
        # 从情感分析结果中提取问题
        for result in sentiment_results:
            if result.polarity == SentimentPolarity.NEGATIVE and result.intensity_score >= 0.7:
                # 提取负面情感的实体作为问题
                for entity in result.entities:
                    if entity.get("sentiment") == "negative":
                        issues.append(f"{entity.get('type')}问题: {entity.get('value')}")
        
        # 去重
        return list(set(issues))[:5]  # 最多返回5个问题
    
    def _generate_improvement_suggestions(self,
                                        base_scores: Dict[str, float],
                                        key_issues: List[str],
                                        csat_score: float) -> List[str]:
        """生成改进建议"""
        suggestions = []
        
        # 基于低分项生成建议
        for metric, score in base_scores.items():
            if score < 80:
                if metric == "response_speed":
                    suggestions.append("优化响应速度，减少用户等待时间")
                elif metric == "answer_accuracy":
                    suggestions.append("提高回答准确性，加强知识库建设")
                elif metric == "service_attitude":
                    suggestions.append("改善服务态度，加强客服培训")
                elif metric == "problem_solving":
                    suggestions.append("提升问题解决能力，完善解决方案库")
                elif metric == "overall_experience":
                    suggestions.append("优化整体用户体验，简化操作流程")
        
        # 基于关键问题生成建议
        for issue in key_issues[:2]:  # 最多基于2个关键问题
            if "产品" in issue:
                suggestions.append("加强产品质量监控和问题反馈机制")
            elif "服务" in issue:
                suggestions.append("提升服务水平，建立服务标准")
            elif "价格" in issue:
                suggestions.append("优化价格策略，提高价格透明度")
            elif "交付" in issue:
                suggestions.append("改善物流配送服务，提供实时跟踪")
        
        # 基于总体分数生成建议
        if csat_score < 70:
            suggestions.append("全面优化客服系统，提升整体服务质量")
        
        # 去重并限制数量
        return list(set(suggestions))[:3]
    
    def _determine_overall_rating(self, csat_score: float) -> str:
        """确定总体评级"""
        if csat_score >= 90:
            return "excellent"
        elif csat_score >= 80:
            return "good"
        elif csat_score >= 70:
            return "average"
        else:
            return "poor"
```

```python
from typing import List, Dict, Any
from dataclasses import dataclass
from datetime import datetime, timedelta
import time
from collections import deque

@dataclass
class RealTimeSentimentMetrics:
    """实时情感指标"""
    timestamp: datetime
    polarity_distribution: Dict[str, float]  # 情感极性分布
    intensity_average: float  # 平均情感强度
    sentiment_trend: List[float]  # 情感趋势
    alert_level: str  # 警报级别
    active_issues: List[str]  # 活跃问题

class RealTimeSentimentMonitor:
    """实时情感监控器"""
    
    def __init__(self, 
                 time_window_minutes: int = 30,
                 alert_threshold: float = 0.3):
        
        self.time_window = timedelta(minutes=time_window_minutes)
        self.alert_threshold = alert_threshold
        
        # 情感数据存储
        self.sentiment_data = deque(maxlen=1000)  # 存储最近1000条情感数据
        
        # 实时指标
        self.real_time_metrics = None
        
        # 警报状态
        self.alerts = []
        self.alert_history = []
    
    def add_sentiment_data(self, sentiment_result: SentimentAnalysisResult):
        """添加情感数据"""
        self.sentiment_data.append({
            "timestamp": sentiment_result.timestamp,
            "polarity": sentiment_result.polarity.value,
            "polarity_score": sentiment_result.polarity_score,
            "intensity": sentiment_result.intensity.value,
            "intensity_score": sentiment_result.intensity_score,
            "text": sentiment_result.text,
            "entities": sentiment_result.entities
        })
        
        # 更新实时指标
        self._update_real_time_metrics()
        
        # 检查警报
        self._check_alerts()
    
    def _update_real_time_metrics(self):
        """更新实时指标"""
        if not self.sentiment_data:
            return
        
        # 获取时间窗口内的数据
        cutoff_time = datetime.now() - self.time_window
        recent_data = [
            data for data in self.sentiment_data 
            if data["timestamp"] > cutoff_time
        ]
        
        if not recent_data:
            return
        
        # 计算情感极性分布
        polarity_counts = {}
        total_count = len(recent_data)
        
        for data in recent_data:
            polarity = data["polarity"]
            polarity_counts[polarity] = polarity_counts.get(polarity, 0) + 1
        
        polarity_distribution = {
            polarity: count / total_count 
            for polarity, count in polarity_counts.items()
        }
        
        # 计算平均情感强度
        avg_intensity = sum(data["intensity_score"] for data in recent_data) / total_count
        
        # 计算情感趋势（最近10个时间点的情感分数）
        sentiment_scores = [data["polarity_score"] for data in recent_data[-10:]]
        
        # 确定警报级别
        alert_level = self._determine_alert_level(polarity_distribution, avg_intensity)
        
        # 提取活跃问题
        active_issues = self._extract_active_issues(recent_data)
        
        self.real_time_metrics = RealTimeSentimentMetrics(
            timestamp=datetime.now(),
            polarity_distribution=polarity_distribution,
            intensity_average=avg_intensity,
            sentiment_trend=sentiment_scores,
            alert_level=alert_level,
            active_issues=active_issues
        )
    
    def _determine_alert_level(self, 
                              polarity_distribution: Dict[str, float],
                              avg_intensity: float) -> str:
        """确定警报级别"""
        
        # 检查负面情感比例
        negative_ratio = polarity_distribution.get("negative", 0)
        
        if negative_ratio > 0.5 and avg_intensity > 0.7:
            return "critical"  # 严重警报
        elif negative_ratio > 0.3 and avg_intensity > 0.6:
            return "high"      # 高警报
        elif negative_ratio > 0.2:
            return "medium"    # 中警报
        elif negative_ratio > 0.1:
            return "low"       # 低警报
        else:
            return "normal"    # 正常
    
    def _extract_active_issues(self, recent_data: List[Dict[str, Any]]) -> List[str]:
        """提取活跃问题"""
        issues = []
        entity_counts = {}
        
        # 统计实体出现次数
        for data in recent_data:
            for entity in data.get("entities", []):
                entity_type = entity.get("type", "unknown")
                entity_value = entity.get("value", "unknown")
                
                key = f"{entity_type}:{entity_value}"
                entity_counts[key] = entity_counts.get(key, 0) + 1
        
        # 选择出现频率最高的实体作为活跃问题
        if entity_counts:
            top_entities = sorted(entity_counts.items(), key=lambda x: x[1], reverse=True)[:3]
            issues = [entity[0] for entity in top_entities]
        
        return issues
    
    def _check_alerts(self):
        """检查警报"""
        if not self.real_time_metrics:
            return
        
        alert_level = self.real_time_metrics.alert_level
        
        if alert_level in ["critical", "high"]:
            # 生成警报
            alert = {
                "timestamp": datetime.now(),
                "level": alert_level,
                "metrics": self.real_time_metrics,
                "message": self._generate_alert_message(alert_level)
            }
            
            self.alerts.append(alert)
            self.alert_history.append(alert)
    
    def _generate_alert_message(self, alert_level: str) -> str:
        """生成警报消息"""
        metrics = self.real_time_metrics
        
        if alert_level == "critical":
            return f"严重警报：负面情感比例{metrics.polarity_distribution.get('negative', 0)*100:.1f}%，平均强度{metrics.intensity_average:.2f}"
        elif alert_level == "high":
            return f"高警报：负面情感比例{metrics.polarity_distribution.get('negative', 0)*100:.1f}%，需要关注"
        else:
            return f"{alert_level}警报：情感状态异常"
    
    def get_dashboard_data(self) -> Dict[str, Any]:
        """获取仪表板数据"""
        if not self.real_time_metrics:
            return {}
        
        return {
            "current_metrics": {
                "polarity_distribution": self.real_time_metrics.polarity_distribution,
                "intensity_average": self.real_time_metrics.intensity_average,
                "sentiment_trend": self.real_time_metrics.sentiment_trend,
                "alert_level": self.real_time_metrics.alert_level,
                "active_issues": self.real_time_metrics.active_issues
            },
            "alerts": self.alerts,
            "total_data_points": len(self.sentiment_data),
            "time_window_minutes": self.time_window.total_seconds() / 60
        }
    
    def clear_alerts(self):
        """清除当前警报"""
        self.alerts.clear()
```
