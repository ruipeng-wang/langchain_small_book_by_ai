# 5.1.3 多模态嵌入支持

## 概念讲解

多模态嵌入支持是现代AI系统的关键能力，它使机器能够理解和处理文本、图像、音频、视频等多种类型的数据，并将它们映射到统一的语义空间中。在LangChain v1.2.22框架中，多模态嵌入不仅仅是多种数据类型的简单支持，而是一个完整的跨模态语义理解体系。

### 多模态智能的演进历程

多模态智能的发展经历了从独立处理到统一理解的演进过程：

1. **单模态独立处理阶段**：文本、图像、音频各自独立的处理管道，缺乏跨模态理解
2. **早期多模态融合阶段**：通过特征拼接、注意力机制等方式进行后期融合
3. **统一表示学习阶段**：CLIP、ALIGN等模型实现图像-文本对的对比学习
4. **大规模多模态预训练阶段**：Flamingo、BLIP、CoCa等模型实现真正的跨模态理解
5. **生成式多模态阶段**：DALL-E、Stable Diffusion、Sora等实现跨模态生成能力

### LangChain多模态嵌入架构

LangChain设计了层次化的多模态嵌入支持架构：

- **统一接口层**：`MultimodalEmbeddings`基类定义跨模态的统一接口
- **模态适配层**：为不同数据类型提供专门的预处理和特征提取
- **对齐学习层**：实现不同模态数据在向量空间中的语义对齐
- **融合推理层**：支持多模态信息的深度融合和联合推理
- **扩展集成层**：便于集成新的模态类型和嵌入模型

### 跨模态语义对齐理论

多模态嵌入的核心是建立不同模态数据间的语义对应关系：

1. **对比学习对齐**：通过正负样本对训练，使相关内容的跨模态向量距离更近
2. **注意力机制对齐**：利用交叉注意力实现模态间的细粒度对齐
3. **层次化对齐**：从低级特征到高级语义的多层次对齐策略
4. **领域自适应对齐**：针对特定应用领域优化跨模态对齐效果
5. **动态可扩展对齐**：支持新模态类型的快速对齐能力

## 核心要点

### 多模态嵌入的关键能力

1. **跨模态检索能力**：
   - 以文搜图：通过文本描述检索相关图像
   - 以图搜文：通过图像内容检索相关文本描述
   - 跨模态关联：建立图像、文本、音频间的复杂语义关联

2. **统一表示学习能力**：
   - 共享向量空间：不同模态数据映射到同一向量空间
   - 语义一致性：相关内容的跨模态向量具有高相似度
   - 可扩展性：支持新模态类型的快速集成

3. **多模态融合能力**：
   - 早期融合：在特征提取阶段进行融合
   - 中期融合：在中间表示层进行融合
   - 晚期融合：在决策层进行融合
   - 自适应融合：根据任务需求动态调整融合策略

4. **模态互补利用能力**：
   - 信息互补：利用不同模态的互补信息提升理解效果
   - 冗余验证：通过多模态信息交叉验证提升可靠性
   - 容错处理：某一模态缺失时仍能保持基本功能

5. **实时处理能力**：
   - 流式处理：支持实时多模态数据流处理
   - 增量学习：支持在线学习和模型更新
   - 资源优化：在计算资源有限时进行智能调度

### 多模态嵌入技术挑战

1. **模态差异挑战**：
   - 特征空间差异：不同模态数据的特征表示天然不同
   - 信息密度差异：图像信息密集，文本信息稀疏
   - 时序特性差异：音频、视频具有时序特性，文本、图像没有

2. **对齐质量挑战**：
   - 标注数据稀缺：高质量的跨模态对齐标注数据有限
   - 语义鸿沟：低级特征与高级语义间的差距
   - 文化差异：不同文化背景下的多模态理解差异

3. **计算效率挑战**：
   - 模型复杂度：多模态模型通常参数量大，计算需求高
   - 存储需求：多模态向量通常维度高，存储需求大
   - 实时性要求：某些应用场景对实时性要求极高

4. **部署实施挑战**：
   - 硬件异构：不同模态可能需要不同的硬件加速
   - 系统集成：与现有系统的集成复杂度
   - 维护成本：多模态系统的维护和更新成本

### 多模态嵌入应用场景

1. **智能内容管理**：
   - 跨媒体内容检索和推荐
   - 自动图像标注和描述生成
   - 多媒体内容理解和分类

2. **增强人机交互**：
   - 多模态对话系统
   - 手势和语音控制
   - 情感识别和响应

3. **专业领域应用**：
   - 医学影像分析
   - 工业视觉检测
   - 教育内容理解

4. **创意内容生成**：
   - 图文互生成
   - 音乐和视频生成
   - 跨模态创意辅助

```python
from typing import Union, List, Dict, Any
import numpy as np
from PIL import Image
import base64
from io import BytesIO

class MultimodalEmbeddingExample:
    """多模态嵌入基础示例"""
    
    def __init__(self):
        # 模拟多模态嵌入模型初始化
        print("初始化多模态嵌入示例")
        print("支持的数据类型: 文本、图像、音频(模拟)")
        
    def encode_text(self, text: str) -> np.ndarray:
        """编码文本为向量"""
        print(f"编码文本: {text[:50]}...")
        
        # 模拟文本编码过程
        # 实际中这里会调用如CLIP的文本编码器
        vector = np.random.randn(512)  # 512维向量
        vector = vector / np.linalg.norm(vector)  # 归一化
        
        print(f"  文本向量维度: {len(vector)}")
        return vector
    
    def encode_image(self, image_input: Union[str, Image.Image, bytes]) -> np.ndarray:
        """编码图像为向量"""
        print("编码图像...")
        
        # 处理不同类型的图像输入
        if isinstance(image_input, str):
            # 假设是文件路径或URL
            print(f"  从路径加载图像: {image_input}")
            # 实际中会加载图像并进行预处理
        elif isinstance(image_input, Image.Image):
            print("  使用PIL图像对象")
        elif isinstance(image_input, bytes):
            print("  使用字节数据")
        
        # 模拟图像编码过程
        # 实际中这里会调用如CLIP的图像编码器
        vector = np.random.randn(512)  # 512维向量
        vector = vector / np.linalg.norm(vector)  # 归一化
        
        print(f"  图像向量维度: {len(vector)}")
        return vector
    
    def encode_audio(self, audio_input: Union[str, bytes]) -> np.ndarray:
        """编码音频为向量"""
        print("编码音频...")
        
        # 模拟音频编码过程
        vector = np.random.randn(512)  # 512维向量
        vector = vector / np.linalg.norm(vector)  # 归一化
        
        print(f"  音频向量维度: {len(vector)}")
        return vector
    
    def cross_modal_similarity(self, 
                              modality_a: str, 
                              data_a: Any, 
                              modality_b: str, 
                              data_b: Any) -> float:
        """计算跨模态相似度"""
        print(f"计算{modality_a}与{modality_b}的跨模态相似度...")
        
        # 根据模态类型选择编码方法
        encoders = {
            'text': self.encode_text,
            'image': self.encode_image,
            'audio': self.encode_audio
        }
        
        if modality_a not in encoders or modality_b not in encoders:
            raise ValueError(f"不支持的模态类型: {modality_a} 或 {modality_b}")
        
        # 编码两种模态的数据
        vector_a = encoders[modality_a](data_a)
        vector_b = encoders[modality_b](data_b)
        
        # 计算余弦相似度
        similarity = np.dot(vector_a, vector_b)
        
        print(f"  {modality_a}-{modality_b}相似度: {similarity:.4f}")
        return similarity
    
    def multimodal_search(self, 
                         query: Union[str, Any], 
                         query_modality: str,
                         database: List[Dict], 
                         top_k: int = 3) -> List[Dict]:
        """多模态检索示例"""
        print(f"执行{query_modality}模态检索，返回top-{top_k}结果...")
        
        # 编码查询
        if query_modality == 'text':
            query_vector = self.encode_text(query)
        elif query_modality == 'image':
            query_vector = self.encode_image(query)
        elif query_modality == 'audio':
            query_vector = self.encode_audio(query)
        else:
            raise ValueError(f"不支持的查询模态: {query_modality}")
        
        # 计算与数据库中每个项目的相似度
        results = []
        for item in database:
            item_modality = item.get('modality', 'unknown')
            item_data = item.get('data', '')
            item_id = item.get('id', 'unknown')
            
            try:
                # 编码数据库项目
                if item_modality == 'text':
                    item_vector = self.encode_text(item_data)
                elif item_modality == 'image':
                    item_vector = self.encode_image(item_data)
                elif item_modality == 'audio':
                    item_vector = self.encode_audio(item_data)
                else:
                    continue
                
                # 计算相似度
                similarity = np.dot(query_vector, item_vector)
                
                results.append({
                    'id': item_id,
                    'modality': item_modality,
                    'data_preview': str(item_data)[:30] + "...",
                    'similarity': similarity,
                    'original_data': item_data
                })
            except Exception as e:
                print(f"处理项目 {item_id} 时出错: {e}")
                continue
        
        # 按相似度排序
        results.sort(key=lambda x: x['similarity'], reverse=True)
        
        return results[:top_k]
```

### 基于CLIP的跨模态检索

```python
import torch
import clip
from PIL import Image

class CLIPMultimodalEmbeddings:
    """基于CLIP的多模态嵌入"""
    
    def __init__(self, model_name: str = "ViT-B/32", device: str = None):
        """
        初始化CLIP模型
        
        Args:
            model_name: CLIP模型名称，如'ViT-B/32', 'ViT-L/14', 'RN50'等
            device: 设备类型，'cuda'或'cpu'，默认为自动检测
        """
        print(f"加载CLIP模型: {model_name}")
        
        # 自动检测设备
        if device is None:
            self.device = "cuda" if torch.cuda.is_available() else "cpu"
        else:
            self.device = device
        
        print(f"使用设备: {self.device}")
        
        # 加载CLIP模型和处理器
        try:
            self.model, self.preprocess = clip.load(model_name, device=self.device)
            print(f"模型加载成功，文本编码器维度: {self.model.text_projection.shape[1]}")
            print(f"图像编码器输出维度: {self.model.visual.output_dim}")
        except Exception as e:
            print(f"模型加载失败: {e}")
            print("使用模拟模式")
            self.model = None
            self.preprocess = None
    
    def encode_texts(self, texts: List[str]) -> torch.Tensor:
        """编码文本列表为向量"""
        if self.model is None:
            print("警告: 使用模拟文本编码")
            return self._simulate_encoding(texts, modality='text')
        
        print(f"编码 {len(texts)} 个文本...")
        
        # 文本预处理和编码
        with torch.no_grad():
            text_tokens = clip.tokenize(texts).to(self.device)
            text_features = self.model.encode_text(text_tokens)
            
            # 归一化
            text_features = text_features / text_features.norm(dim=-1, keepdim=True)
        
        return text_features.cpu().numpy()
    
    def encode_images(self, images: List[Union[str, Image.Image]]) -> torch.Tensor:
        """编码图像列表为向量"""
        if self.model is None:
            print("警告: 使用模拟图像编码")
            return self._simulate_encoding(images, modality='image')
        
        print(f"编码 {len(images)} 个图像...")
        
        processed_images = []
        for img in images:
            if isinstance(img, str):
                # 从文件加载
                try:
                    image = Image.open(img)
                    processed_img = self.preprocess(image).unsqueeze(0)
                    processed_images.append(processed_img)
                except Exception as e:
                    print(f"加载图像失败 {img}: {e}")
                    continue
            elif isinstance(img, Image.Image):
                # 直接使用PIL图像
                processed_img = self.preprocess(img).unsqueeze(0)
                processed_images.append(processed_img)
            else:
                print(f"不支持的图像类型: {type(img)}")
                continue
        
        if not processed_images:
            raise ValueError("没有有效的图像可以编码")
        
        # 批量处理
        image_batch = torch.cat(processed_images).to(self.device)
        
        with torch.no_grad():
            image_features = self.model.encode_image(image_batch)
            image_features = image_features / image_features.norm(dim=-1, keepdim=True)
        
        return image_features.cpu().numpy()
    
    def _simulate_encoding(self, items: List, modality: str) -> np.ndarray:
        """模拟编码（当真实模型不可用时）"""
        dim = 512  # 模拟维度
        batch_size = len(items)
        
        # 生成随机向量
        vectors = np.random.randn(batch_size, dim)
        
        # 归一化
        norms = np.linalg.norm(vectors, axis=1, keepdims=True)
        vectors = vectors / norms
        
        print(f"  模拟{modality}编码，维度: {dim}, 批量: {batch_size}")
        return vectors
    
    def cross_modal_retrieval(self, 
                             query: Union[str, Image.Image], 
                             query_type: str,
                             candidates: List[Dict],
                             candidate_type: str,
                             top_k: int = 5) -> List[Dict]:
        """跨模态检索"""
        print(f"执行{query_type}->{candidate_type}跨模态检索...")
        
        # 编码查询
        if query_type == 'text':
            query_vector = self.encode_texts([query])[0]
        elif query_type == 'image':
            query_vector = self.encode_images([query])[0]
        else:
            raise ValueError(f"不支持的查询类型: {query_type}")
        
        # 准备候选数据
        candidate_data = []
        candidate_info = []
        
        for candidate in candidates:
            c_type = candidate.get('type', candidate_type)
            c_data = candidate.get('data')
            c_id = candidate.get('id', f"item_{len(candidate_info)}")
            c_metadata = candidate.get('metadata', {})
            
            if c_type != candidate_type:
                print(f"跳过类型不匹配的候选: {c_id} ({c_type} != {candidate_type})")
                continue
            
            candidate_data.append(c_data)
            candidate_info.append({
                'id': c_id,
                'type': c_type,
                'metadata': c_metadata,
                'original_data': c_data
            })
        
        if not candidate_data:
            print("没有有效的候选数据")
            return []
        
        # 编码候选数据
        if candidate_type == 'text':
            candidate_vectors = self.encode_texts(candidate_data)
        elif candidate_type == 'image':
            candidate_vectors = self.encode_images(candidate_data)
        else:
            raise ValueError(f"不支持的候选类型: {candidate_type}")
        
        # 计算相似度
        similarities = []
        for i, cand_vector in enumerate(candidate_vectors):
            similarity = np.dot(query_vector, cand_vector)
            similarities.append(similarity)
        
        # 组合结果并排序
        results = []
        for i, similarity in enumerate(similarities):
            result = {
                **candidate_info[i],
                'similarity': float(similarity),
                'rank': 0
            }
            results.append(result)
        
        results.sort(key=lambda x: x['similarity'], reverse=True)
        
        # 分配排名
        for i, result in enumerate(results[:top_k]):
            result['rank'] = i + 1
        
        return results[:top_k]
    
    def multimodal_fusion_search(self,
                                text_query: str = None,
                                image_query: Image.Image = None,
                                candidates: List[Dict] = None,
                                fusion_weight: float = 0.5) -> List[Dict]:
        """多模态融合检索"""
        print("执行多模态融合检索...")
        
        if text_query is None and image_query is None:
            raise ValueError("至少需要提供文本查询或图像查询")
        
        if candidates is None:
            candidates = []
        
        # 分别计算文本和图像相似度
        text_similarities = []
        image_similarities = []
        
        if text_query is not None:
            text_query_vector = self.encode_texts([text_query])[0]
            
            for candidate in candidates:
                if candidate.get('type') == 'text':
                    cand_text = candidate.get('data', '')
                    cand_vector = self.encode_texts([cand_text])[0]
                    similarity = np.dot(text_query_vector, cand_vector)
                else:
                    similarity = 0.0  # 非文本候选的文本相似度为0
                
                text_similarities.append(similarity)
        
        if image_query is not None:
            image_query_vector = self.encode_images([image_query])[0]
            
            for candidate in candidates:
                if candidate.get('type') == 'image':
                    cand_image = candidate.get('data')
                    cand_vector = self.encode_images([cand_image])[0]
                    similarity = np.dot(image_query_vector, cand_vector)
                else:
                    similarity = 0.0  # 非图像候选的图像相似度为0
                
                image_similarities.append(similarity)
        
        # 融合相似度
        fused_similarities = []
        for i in range(len(candidates)):
            text_sim = text_similarities[i] if text_query is not None else 0
            image_sim = image_similarities[i] if image_query is not None else 0
            
            # 加权融合
            if text_query is not None and image_query is not None:
                fused = text_sim * fusion_weight + image_sim * (1 - fusion_weight)
            elif text_query is not None:
                fused = text_sim
            else:
                fused = image_sim
            
            fused_similarities.append(fused)
        
        # 组合结果
        results = []
        for i, similarity in enumerate(fused_similarities):
            result = {
                'id': candidates[i].get('id', f"item_{i}"),
                'type': candidates[i].get('type', 'unknown'),

                'text_similarity': float(text_similarities[i]) if text_query else None,
                'image_similarity': float(image_similarities[i]) if image_query else None,
                'fused_similarity': float(similarity),
                'original_data': candidates[i].get('data')
            }
            results.append(result)
        
        # 按融合相似度排序
        results.sort(key=lambda x: x['fused_similarity'], reverse=True)
        
        return results[:10]
```

```python
class MultimodalVectorStore:
    """多模态向量存储"""
    
    def __init__(self, embedding_model=None):
        self.embedding_model = embedding_model
        self.vectors = {}  # id -> 向量
        self.metadata = {}  # id -> 元数据
        self.modality_index = {
            'text': set(),
            'image': set(),
            'audio': set(),
            'video': set()
        }
        
        print("初始化多模态向量存储")
        print(f"支持的模态: {list(self.modality_index.keys())}")
    
    def add_item(self, 
                item_id: str, 
                data: Any, 
                modality: str,
                metadata: Dict = None,
                vector: np.ndarray = None):
        """添加多模态项目"""
        if modality not in self.modality_index:
            raise ValueError(f"不支持的模态: {modality}")
        
        print(f"添加项目: {item_id} [{modality}]")
        
        # 如果没有提供向量，使用嵌入模型生成
        if vector is None and self.embedding_model is not None:
            print(f"  生成{modality}向量...")
            if modality == 'text':
                vector = self.embedding_model.encode_texts([data])[0]
            elif modality == 'image':
                vector = self.embedding_model.encode_images([data])[0]
            else:
                print(f"  警告: 模态{modality}的自动向量生成未实现")
                vector = np.random.randn(512)  # 模拟向量
                vector = vector / np.linalg.norm(vector)
        elif vector is None:
            print(f"  警告: 没有嵌入模型，生成模拟向量")
            vector = np.random.randn(512)  # 模拟向量
            vector = vector / np.linalg.norm(vector)
        
        # 存储向量和元数据
        self.vectors[item_id] = vector
        self.modality_index[modality].add(item_id)
        
        self.metadata[item_id] = {
            'data': data,
            'modality': modality,
            'added_time': datetime.now(),
            **(metadata or {})
        }
        
        print(f"  向量维度: {len(vector)}")
        return item_id
    
    def search(self, 
              query: Any,
              query_modality: str,
              target_modality: str = None,
              top_k: int = 10,
              filters: Dict = None) -> List[Dict]:
        """多模态搜索"""
        print(f"搜索: {query_modality}->{target_modality or 'all'}")
        
        # 编码查询
        if self.embedding_model is not None:
            if query_modality == 'text':
                query_vector = self.embedding_model.encode_texts([query])[0]
            elif query_modality == 'image':
                query_vector = self.embedding_model.encode_images([query])[0]
            else:
                raise ValueError(f"不支持的查询模态: {query_modality}")
        else:
            print("  警告: 使用模拟查询向量")
            query_vector = np.random.randn(512)
            query_vector = query_vector / np.linalg.norm(query_vector)
        
        # 确定搜索范围
        if target_modality:
            if target_modality not in self.modality_index:
                raise ValueError(f"不支持的目标模态: {target_modality}")
            candidate_ids = list(self.modality_index[target_modality])
        else:
            # 搜索所有模态
            candidate_ids = list(self.vectors.keys())
        
        # 应用过滤器
        if filters:
            candidate_ids = self._apply_filters(candidate_ids, filters)
        
        if not candidate_ids:
            print("  没有满足条件的候选项目")
            return []
        
        # 计算相似度
        similarities = []
        for item_id in candidate_ids:
            item_vector = self.vectors[item_id]
            similarity = np.dot(query_vector, item_vector)
            similarities.append((item_id, similarity))
        
        # 排序并返回结果
        similarities.sort(key=lambda x: x[1], reverse=True)
        
        results = []
        for i, (item_id, similarity) in enumerate(similarities[:top_k]):
            metadata = self.metadata[item_id]
            
            result = {
                'id': item_id,
                'modality': metadata['modality'],
                'similarity': float(similarity),
                'rank': i + 1,
                'data_preview': self._preview_data(metadata['data'], metadata['modality']),
                'metadata': {k: v for k, v in metadata.items() if k != 'data'}
            }
            results.append(result)
        
        print(f"  找到 {len(results)} 个结果")
        return results
    
    def _apply_filters(self, candidate_ids: List[str], filters: Dict) -> List[str]:
        """应用过滤器"""
        filtered_ids = []
        
        for item_id in candidate_ids:
            metadata = self.metadata[item_id]
            passes = True
            
            for key, value in filters.items():
                if key in metadata:
                    if isinstance(value, (list, tuple)):
                        if metadata[key] not in value:
                            passes = False
                            break
                    else:
                        if metadata[key] != value:
                            passes = False
                            break
            
            if passes:
                filtered_ids.append(item_id)
        
        return filtered_ids
    
    def _preview_data(self, data: Any, modality: str) -> str:
        """生成数据预览"""
        if modality == 'text':
            return str(data)[:50] + "..."
        elif modality == 'image':
            if isinstance(data, str):
                return f"图像: {data}"
            else:
                return "图像对象"
        elif modality == 'audio':
            return f"音频: {data}"
        else:
            return str(data)[:30]
    
    def cross_modal_recommendation(self,
                                  source_item_id: str,
                                  target_modality: str,
                                  top_k: int = 5) -> List[Dict]:
        """跨模态推荐"""
        print(f"跨模态推荐: {source_item_id} -> {target_modality}")
        
        if source_item_id not in self.vectors:
            raise ValueError(f"源项目不存在: {source_item_id}")
        
        source_modality = self.metadata[source_item_id]['modality']
        
        if source_modality == target_modality:
            print("  警告: 源模态和目标模态相同，执行同模态推荐")
        
        # 使用源项目的向量进行搜索
        source_vector = self.vectors[source_item_id]
        
        # 临时设置查询向量
        temp_results = self.search(
            query=None,  # 不使用实际查询数据
            query_modality=source_modality,
            target_modality=target_modality,
            top_k=top_k
        )
        
        # 使用源向量重新计算相似度
        for result in temp_results:
            target_vector = self.vectors[result['id']]
            similarity = np.dot(source_vector, target_vector)
            result['similarity'] = float(similarity)
        
        # 重新排序
        temp_results.sort(key=lambda x: x['similarity'], reverse=True)
        
        # 添加源项目信息
        for result in temp_results:
            result['source_item'] = source_item_id
            result['source_modality'] = source_modality
            result['recommendation_type'] = 'cross_modal'
        
        return temp_results[:top_k]
    
    def get_statistics(self) -> Dict:
        """获取存储统计信息"""
        stats = {
            'total_items': len(self.vectors),
            'modality_distribution': {},
            'vector_dimension': None,
            'storage_size_estimate': 0
        }
        
        # 统计模态分布
        for modality, item_set in self.modality_index.items():
            stats['modality_distribution'][modality] = len(item_set)
        
        # 获取向量维度
        if self.vectors:
            first_vector = next(iter(self.vectors.values()))
            stats['vector_dimension'] = len(first_vector)
            
            # 估算存储大小
            vector_size = len(first_vector) * 4  # float32 = 4字节
            metadata_size = 100  # 估算元数据大小（字节）
            stats['storage_size_estimate'] = len(self.vectors) * (vector_size + metadata_size)
        
        return stats
```

```python
class MultimodalFusionEmbeddingSystem:
    """多模态融合嵌入系统"""
    
    def __init__(self, modality_encoders: Dict[str, Any]):
        """
        初始化多模态融合嵌入系统
        
        Args:
            modality_encoders: 模态编码器字典，格式为 {模态: 编码器}
        """
        self.modality_encoders = modality_encoders
        self.fusion_strategies = {
            'concat': self._concat_fusion,
            'weighted_sum': self._weighted_sum_fusion,
            'attention': self._attention_fusion,
            'transformer': self._transformer_fusion
        }
        
        print(f"初始化多模态融合系统，支持模态: {list(modality_encoders.keys())}")
    
    def encode_multimodal(self,
                         data_dict: Dict[str, Any],
                         fusion_strategy: str = 'concat',
                         strategy_params: Dict = None) -> np.ndarray:
        """编码多模态数据"""
        print(f"编码多模态数据，融合策略: {fusion_strategy}")
        
        # 编码各模态数据
        modality_vectors = {}
        
        for modality, data in data_dict.items():
            if modality not in self.modality_encoders:
                print(f"  警告: 模态{modality}没有对应的编码器，跳过")
                continue
            
            encoder = self.modality_encoders[modality]
            
            try:
                if modality == 'text':
                    vector = encoder.encode_texts([data])[0]
                elif modality == 'image':
                    vector = encoder.encode_images([data])[0]
                elif modality == 'audio':
                    vector = encoder.encode_audio([data])[0]
                else:
                    print(f"  警告: 模态{modality}的编码方法未定义")
                    continue
                
                modality_vectors[modality] = vector
                print(f"  已编码{modality}，维度: {len(vector)}")
                
            except Exception as e:
                print(f"  编码{modality}失败: {e}")
                continue
        
        if not modality_vectors:
            raise ValueError("没有成功编码任何模态数据")
        
        # 应用融合策略
        if fusion_strategy in self.fusion_strategies:
            fused_vector = self.fusion_strategies[fusion_strategy](
                modality_vectors, 
                strategy_params or {}
            )
        else:
            print(f"  警告: 策略{fusion_strategy}未找到，使用默认concat策略")
            fused_vector = self._concat_fusion(modality_vectors)
        
        print(f"  融合后维度: {len(fused_vector)}")
        return fused_vector
    
    def _concat_fusion(self, modality_vectors: Dict[str, np.ndarray], params: Dict) -> np.ndarray:
        """拼接融合"""
        # 简单拼接所有模态向量
        vectors = list(modality_vectors.values())
        fused = np.concatenate(vectors)
        return fused
    
    def _weighted_sum_fusion(self, modality_vectors: Dict[str, np.ndarray], params: Dict) -> np.ndarray:
        """加权和融合"""
        weights = params.get('weights', {})
        
        # 为每个模态分配权重
        weighted_vectors = []
        total_weight = 0
        
        for modality, vector in modality_vectors.items():
            weight = weights.get(modality, 1.0)  # 默认权重1.0
            weighted_vector = vector * weight
            weighted_vectors.append(weighted_vector)
            total_weight += weight
        
        # 计算加权平均
        if total_weight > 0:
            fused = sum(weighted_vectors) / total_weight
        else:
            fused = np.mean(list(modality_vectors.values()), axis=0)
        
        return fused
    
    def _attention_fusion(self, modality_vectors: Dict[str, np.ndarray], params: Dict) -> np.ndarray:
        """注意力融合"""
        # 简化版注意力融合
        vectors = list(modality_vectors.values())
        modalities = list(modality_vectors.keys())
        
        # 计算注意力权重（基于向量范数）
        norms = [np.linalg.norm(v) for v in vectors]
        total_norm = sum(norms)
        
        if total_norm > 0:
            attention_weights = [n / total_norm for n in norms]
        else:
            attention_weights = [1.0 / len(vectors)] * len(vectors)
        
        # 加权融合
        fused = np.zeros_like(vectors[0])
        for i, (vector, weight) in enumerate(zip(vectors, attention_weights)):
            fused += vector * weight
        
        print(f"  注意力权重: {dict(zip(modalities, attention_weights))}")
        return fused
    
    def _transformer_fusion(self, modality_vectors: Dict[str, np.ndarray], params: Dict) -> np.ndarray:
        """Transformer融合（简化实现）"""
        # 在实际应用中，这里会使用真正的Transformer进行融合
        # 这里使用简化的自注意力实现
        
        vectors = list(modality_vectors.values())
        num_modalities = len(vectors)
        
        if num_modalities == 1:
            return vectors[0]
        
        # 将所有向量堆叠成矩阵 (num_modalities, dim)
        matrix = np.vstack(vectors)
        
        # 计算自注意力
        # QK^T / sqrt(dim)
        dim = matrix.shape[1]
        attention_scores = np.dot(matrix, matrix.T) / np.sqrt(dim)
        
        # Softmax
        exp_scores = np.exp(attention_scores - np.max(attention_scores, axis=-1, keepdims=True))
        attention_weights = exp_scores / np.sum(exp_scores, axis=-1, keepdims=True)
        
        # 加权平均
        fused = np.dot(attention_weights, matrix)
        
        # 取平均作为最终表示
        fused = np.mean(fused, axis=0)
        
        return fused
    
    def multimodal_similarity(self,
                            data_a: Dict[str, Any],
                            data_b: Dict[str, Any],
                            fusion_strategy: str = 'concat') -> Dict[str, Any]:
        """计算多模态数据相似度"""
        print("计算多模态相似度...")
        
        # 编码两个多模态数据
        vector_a = self.encode_multimodal(data_a, fusion_strategy)
        vector_b = self.encode_multimodal(data_b, fusion_strategy)
        
        # 计算相似度
        similarity = np.dot(vector_a, vector_b)
        
        # 计算各模态单独相似度
        modality_similarities = {}
        common_modalities = set(data_a.keys()) & set(data_b.keys())
        
        for modality in common_modalities:
            if modality in self.modality_encoders:
                encoder = self.modality_encoders[modality]
                
                try:
                    if modality == 'text':
                        vec_a = encoder.encode_texts([data_a[modality]])[0]
                        vec_b = encoder.encode_texts([data_b[modality]])[0]
                    elif modality == 'image':
                        vec_a = encoder.encode_images([data_a[modality]])[0]
                        vec_b = encoder.encode_images([data_b[modality]])[0]
                    else:
                        continue
                    
                    mod_similarity = np.dot(vec_a, vec_b)
                    modality_similarities[modality] = float(mod_similarity)
                    
                except Exception as e:
                    print(f"  计算{modality}相似度失败: {e}")
        
        return {
            'overall_similarity': float(similarity),
            'modality_similarities': modality_similarities,
            'fusion_strategy': fusion_strategy,
            'vector_dimension': len(vector_a)
        }
```

### Q1: 如何处理不同模态数据的维度不一致问题？
**A:** 维度统一化策略：

1. **投影对齐策略**：
   - 使用线性投影将不同维度的向量映射到统一维度
   - 通过训练学习投影矩阵，最小化语义信息损失
   - 支持动态维度调整，适应不同计算资源

2. **层次化对齐策略**：
   - 在多个粒度级别进行对齐：特征级、语义级、任务级
   - 使用注意力机制动态调整对齐权重
   - 结合领域知识优化对齐效果

3. **自适应融合策略**：
   - 根据数据特性和任务需求自动选择融合策略
   - 使用元学习优化融合参数
   - 支持在线学习和自适应调整

   - 内存使用效率

```python
class FusionStrategySelector:
    """融合策略选择器"""
    
    def select_strategy(self, 
                       task_type: str,
                       data_characteristics: Dict,
                       resource_constraints: Dict) -> Dict:
        """选择融合策略"""
        
        # 根据任务类型推荐
        task_recommendations = {
            'retrieval': {
                'primary': 'concat',
                'alternatives': ['weighted_sum', 'attention'],
                'reason': '检索任务需要保留各模态完整信息'
            },
            'classification': {
                'primary': 'attention',
                'alternatives': ['weighted_sum', 'transformer'],
                'reason': '分类任务需要关注关键模态特征'
            },
            'generation': {
                'primary': 'transformer',
                'alternatives': ['attention', 'weighted_sum'],
                'reason': '生成任务需要深度模态交互'
            },
            'similarity': {
                'primary': 'weighted_sum',
                'alternatives': ['concat', 'attention'],
                'reason': '相似度计算需要平衡各模态贡献'
            }
        }
        
        task_rec = task_recommendations.get(
            task_type, 
            task_recommendations['retrieval']
        )
        
        # 根据数据特征调整
        num_modalities = data_characteristics.get('num_modalities', 2)
        modality_balance = data_characteristics.get('modality_balance', 'balanced')
        
        if num_modalities > 3 and modality_balance == 'imbalanced':
            # 模态多且不平衡时，推荐注意力机制
            task_rec['primary'] = 'attention'
            task_rec['reason'] += '，注意力机制能更好处理多模态不平衡'
        
        # 根据资源约束调整
        if resource_constraints.get('compute_intensive', False):
            # 计算资源有限时，推荐简单策略
            if task_rec['primary'] in ['transformer', 'attention']:
                task_rec['primary'] = 'weighted_sum'
                task_rec['reason'] += '，加权和策略计算更高效'
        
        return task_rec
```

### Q4: 如何优化多模态嵌入的实时性能？
**A:** 实时性能优化策略：

1. **编码加速**：
   - 使用量化技术减少模型大小和计算量
   - 实现批处理编码，利用硬件并行性
   - 采用缓存机制，避免重复编码

2. **融合优化**：
   - 预计算融合模板，减少运行时计算
   - 使用近似融合算法，平衡精度和速度
   - 实现增量融合，支持流式处理

3. **存储优化**：
   - 使用高效的向量存储格式
   - 实现向量压缩，减少存储和传输开销
   - 采用分层存储，热点数据常驻内存

4. **检索优化**：
   - 使用近似最近邻搜索（ANN）算法
   - 实现多级索引，加速检索过程
   - 支持增量索引，避免全量重建

### Q5: 如何处理缺失模态的情况？
**A:** 缺失模态处理策略：

1. **模型推断策略**：
   - 使用生成模型推断缺失模态内容
   - 基于已有模态信息预测缺失模态特征
   - 实现模态间的条件生成

2. **降级处理策略**：
   - 当关键模态缺失时，自动降级到可用模态
   - 提供部分功能支持，而不是完全失败
   - 实现优雅降级，保证基本功能可用

3. **数据增强策略**：
   - 使用数据增强技术生成合成模态数据
   - 通过跨模态转换补充缺失信息
   - 利用外部知识库丰富缺失内容

4. **用户交互策略**：
   - 提示用户补充缺失模态信息
   - 提供替代输入方式
   - 记录用户偏好，优化未来交互
