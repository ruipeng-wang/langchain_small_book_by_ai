# 4.3 网络与数据库加载器

## 概念讲解

除了本地文件，实际项目中还需要从网络和数据库加载数据。LangChain提供了丰富的加载器来处理这些场景。

### 网络类加载器

| 加载器 | 说明 | 适用场景 |
|--------|------|---------|
| `WebBaseLoader` | 基于BeautifulSoup的网页加载器 | 通用网页内容提取 |
| `AsyncHtmlLoader` | 异步HTML加载器 | 批量网页并发加载 |
| `FireCrawlLoader` | 基于FireCrawl的爬虫加载器 | 专业网页抓取 |
| `SitemapLoader` | 站点地图加载器 | 批量抓取整个网站 |

### 数据库类加载器

| 加载器 | 说明 | 适用场景 |
|--------|------|---------|
| `SQLDatabaseLoader` | SQL数据库加载器 | 关系型数据库查询 |
| `MongoDBLoader` | MongoDB加载器 | 文档数据库查询 |
| `AirbyteLoader` | Airbyte同步加载器 | 多种数据源同步 |

## 核心要点

### 🔑 网页内容加载

`WebBaseLoader`是最常用的网页加载器，自动提取正文并清理HTML标签：

```python
from langchain_community.document_loaders import WebBaseLoader

# 加载单个网页
loader = WebBaseLoader("https://example.com/article")
documents = loader.load()

# 加载多个网页
urls = [
    "https://example.com/page1",
    "https://example.com/page2",
]
loader = WebBaseLoader(urls)
documents = loader.load()

# 自定义解析器
loader = WebBaseLoader(
    "https://example.com/article",
    bs_kwargs={"features": "html.parser"}  # 指定解析器
)
```

### 🔑 异步批量加载

对于大量网页，使用异步加载器提高效率：

```python
from langchain_community.document_loaders import AsyncHtmlLoader

# 异步并发加载
loader = AsyncHtmlLoader([
    "https://example.com/page1",
    "https://example.com/page2",
    "https://example.com/page3",
])

# 返回原始HTML文档
documents = loader.load()

# 通常配合HTML转换使用
from langchain_community.document_transformers import BeautifulSoupTransformer
transformer = BeautifulSoupTransformer()
docs_transformed = transformer.transform_documents(documents)
```

### 🔑 SQL数据库加载

从关系型数据库查询数据并转换为Document：

```python
from langchain_community.document_loaders import SQLDatabaseLoader

loader = SQLDatabaseLoader(
    db_uri="sqlite:///mydb.db",
    query="SELECT id, title, content, author FROM articles WHERE status='published'",
    source_columns=["id", "title"]  # 用于生成source元数据
)

documents = loader.load()
```

### 🔑 MongoDB加载器

从MongoDB文档数据库加载数据：

```python
from langchain_community.document_loaders import MongoDBLoader

loader = MongoDBLoader(
    connection_string="mongodb://localhost:27017",
    db_name="mydb",
    collection_name="articles",
    filter_query={"status": "published"},  # 可选过滤条件
    projection={"title": 1, "content": 1, "author": 1}  # 字段选择
)

documents = loader.load()
```

## 进阶应用

### 网页内容智能提取

使用`UnstructuredURLLoader`进行更智能的内容提取：

```python
from langchain_community.document_loaders import UnstructuredURLLoader

urls = [
    "https://example.com/blog/post1",
    "https://example.com/blog/post2",
]

loader = UnstructuredURLLoader(urls=urls)
documents = loader.load()

# 自动清理和格式化内容
for doc in documents:
    print(f"URL: {doc.metadata.get('source')}")
    print(f"标题: {doc.metadata.get('title')}")
    print(f"内容长度: {len(doc.page_content)}")
```

### 动态查询数据库

结合SQL查询构建动态加载器：

```python
from sqlalchemy import create_engine
import pandas as pd

class DynamicSQLLoader:
    """动态SQL加载器"""
    
    def __init__(self, connection_string: str):
        self.engine = create_engine(connection_string)
    
    def load_query(self, query: str, **params) -> list:
        """执行查询并返回Document列表"""
        df = pd.read_sql(query, self.engine, params=params)
        
        from langchain_core.documents import Document
        
        documents = []
        for _, row in df.iterrows():
            content = "\n".join(
                f"{col}: {row[col]}" 
                for col in df.columns 
                if col not in ["id", "created_at"]
            )
            
            doc = Document(
                page_content=content,
                metadata={
                    "source": f"db:row_{row.get('id', 'unknown')}",
                    **{col: row[col] for col in ["id"] if col in df.columns}
                }
            )
            documents.append(doc)
        
        return documents

# 使用示例

# docs = loader.load_query(
#     "SELECT * FROM articles WHERE category = :category",
#     category="tech"
# )
```

## 常见问题

### Q1: 网页加载时遇到反爬限制怎么办？

**A:** 几种解决方案：
1. 添加请求头模拟浏览器：`loader = WebBaseLoader(url, header_template={"User-Agent": "Mozilla/5.0..."})`
2. 使用代理IP
3. 使用专业的爬虫服务如FireCrawl

### Q2: 数据库加载器支持哪些数据库？

**A:** 通过SQLAlchemy支持，几乎支持所有主流数据库：
- SQLite（内置）
- MySQL（需要`pymysql`）
- PostgreSQL（需要`psycopg2`）
- Oracle（需要`cx_Oracle`）
- SQL Server（需要`pyodbc`）

### Q3: 如何处理大网站的批量加载？

**A:** 使用`SitemapLoader`配合站点地图：

```python
from langchain_community.document_loaders import SitemapLoader

loader = SitemapLoader(
    "https://example.com/sitemap.xml",
    continue_on_failure=True,  # 失败时继续
    blocksize=10,  # 每批处理数量
)
documents = loader.load()
```

## 本节总结

网络与数据库加载器扩展了LangChain的数据源范围：
1. `WebBaseLoader`适用于通用网页内容提取
2. `AsyncHtmlLoader`适合批量并发加载
3. SQL和MongoDB加载器支持数据库查询
4. 配合动态加载器实现灵活的数据获取
