# 4.3.1 文本清洗与规范化

## 概念讲解

文本清洗与规范化是文档预处理流程中的第一步，也是最为关键的环节。原始文档数据往往包含大量噪声、格式问题和不一致性，这些因素会直接影响后续的语义理解、向量化效果和检索质量。LangChain提供了系统化的文本清洗与规范化解决方案，帮助开发者高效处理各种文档质量问题。

### 文本噪声的主要来源

在真实业务场景中，文本数据通常面临以下噪声挑战：

1. **结构化噪声**：HTML标签、XML标记、JSON格式字符、Markdown语法等
2. **格式噪声**：多余的空格、换行符、制表符、不可见字符等
3. **编码噪声**：不同编码格式（UTF-8、GBK、ISO-8859-1）的混用和转换问题
4. **语言噪声**：拼写错误、大小写不一致、缩写、方言表达等
5. **内容噪声**：广告内容、版权信息、导航菜单、页眉页脚等无关信息

### LangChain清洗架构设计

LangChain采用分层清洗架构，支持从简单规则清洗到复杂语义清洗的多级处理：

- **字符层清洗**：处理编码、特殊字符、空白符等基础问题
- **语法层清洗**：处理HTML/XML标记、格式规范化、语言标准化
- **语义层清洗**：基于上下文的内容过滤、冗余信息识别、质量评估
- **管道层整合**：支持多个清洗步骤的串联执行和条件分支

## 核心要点

### 核心清洗功能
1. **HTML/XML标记去除**：完整去除结构化标记，保留有意义的文本内容
2. **空白符规范化**：合并多余空格、统一换行符、清理不可见字符
3. **编码统一化**：自动检测和转换不同编码格式为UTF-8标准
4. **语言规范化**：大小写统一、拼写纠正、缩写扩展等
5. **内容过滤**：基于规则的广告、版权、导航等内容识别与移除

### 关键技术特性
1. **增量处理支持**：支持流式输入和批量处理，适用于各种数据规模
2. **自定义规则扩展**：允许开发者定义领域特定的清洗规则
3. **处理质量监控**：提供清洗前后的对比分析和质量指标
4. **多语言支持**：针对不同语言特点提供专门的清洗策略
5. **性能优化**：并行处理、缓存机制、懒加载等优化技术

### 清洗策略分类
- **保守清洗**：仅去除明显噪声，最大限度保留原始内容
- **平衡清洗**：在保留内容和提升质量间取得平衡
- **激进清洗**：深度清理，适用于对质量要求极高的场景
- **自适应清洗**：根据文档类型和内容自动选择合适的清洗策略

```python
import re
from bs4 import BeautifulSoup
from langchain_community.document_loaders import RecursiveUrlLoader

def custom_html_extractor(html: str) -> str:
    """
    自定义HTML提取器，提供更精细的控制
    
    参数：
        html: 原始HTML字符串
        
    返回：
        清洗后的文本内容
    """
    # 创建BeautifulSoup解析器
    soup = BeautifulSoup(html, "lxml")
    
    # 移除不需要的元素
    for element in soup.find_all(["script", "style", "nav", "footer", "header"]):
        element.decompose()
    
    # 提取文本内容
    text = soup.get_text()
    
    # 规范化空白符
    # 1. 替换多个换行符为单个换行符
    text = re.sub(r'\n\s*\n', '\n\n', text)
    # 2. 替换多个空格为单个空格
    text = re.sub(r'\s+', ' ', text)
    # 3. 去除首尾空白
    text = text.strip()
    
    return text
```

### 空白符规范化处理

```python
from langchain.text_splitter import CharacterTextSplitter
from langchain_core.documents import Document

def normalize_whitespace(text: str) -> str:
    """
    规范化文本中的空白符
    
    参数：
        text: 原始文本
        
    返回：
        规范化后的文本
    """
    import re
    
    # 替换所有类型的空白符为单个空格
    text = re.sub(r'\s+', ' ', text)
    
    # 规范化换行符（统一为\n）
    text = text.replace('\r\n', '\n').replace('\r', '\n')
    
    # 清理行首行尾空白
    lines = text.split('\n')
    cleaned_lines = [line.strip() for line in lines]
    
    # 重新组合，过滤空行
    cleaned_lines = [line for line in cleaned_lines if line]
    
    return '\n'.join(cleaned_lines)
```

## 进阶应用

### 构建可配置的清洗管道

```python
from typing import List, Callable, Optional
from dataclasses import dataclass
from langchain_core.documents import BaseDocumentTransformer

@dataclass
class CleaningConfig:
    """清洗配置类"""
    remove_html: bool = True
    normalize_whitespace: bool = True
    fix_encoding: bool = True
    remove_ads: bool = False
    language_specific: Optional[str] = None  # 'zh-CN', 'en', 'ja', etc.
    custom_rules: List[Callable] = None

class ConfigurableCleaner(BaseDocumentTransformer):
    """可配置的文档清洗器"""
    
    def __init__(self, config: CleaningConfig):
        self.config = config
        self.custom_rules = config.custom_rules or []
        
    def transform_documents(self, documents, **kwargs):
        """转换文档"""
        cleaned_docs = []
        
        for doc in documents:
            content = doc.page_content
            metadata = doc.metadata.copy()
            
            # 记录清洗前的状态
            metadata['cleaning_original_length'] = len(content)
            
            # 应用HTML清洗
            if self.config.remove_html:
                content = self._remove_html_tags(content)
                
            # 应用空白符规范化
            if self.config.normalize_whitespace:
                content = self._normalize_whitespace(content)
                
            # 应用编码修复
            if self.config.fix_encoding:
                content = self._fix_encoding(content)
                
            # 应用语言特定处理
            if self.config.language_specific:
                content = self._apply_language_specific(content, self.config.language_specific)
                
            # 应用自定义规则
            for rule in self.custom_rules:
                content = rule(content)
                
            # 记录清洗后的状态
            metadata['cleaning_final_length'] = len(content)
            metadata['cleaning_reduction'] = (
                metadata['cleaning_original_length'] - metadata['cleaning_final_length']
            ) / metadata['cleaning_original_length'] * 100
            
            cleaned_docs.append(
                Document(page_content=content, metadata=metadata)
            )
            
        return cleaned_docs
    
    def _remove_html_tags(self, text: str) -> str:
        """移除HTML标签"""
        import re
        # 简单但有效的HTML标签移除
        clean = re.compile('<.*?>')
        return re.sub(clean, '', text)
    
    def _normalize_whitespace(self, text: str) -> str:
        """规范化空白符"""
        import re
        # 合并多个空白符
        text = re.sub(r'\s+', ' ', text)
        # 规范化换行符
        text = text.replace('\r\n', '\n').replace('\r', '\n')
        return text.strip()
    
    def _fix_encoding(self, text: str) -> str:
        """修复编码问题"""
        try:
            # 尝试解码为UTF-8
            return text.encode('utf-8').decode('utf-8')
        except (UnicodeDecodeError, UnicodeEncodeError):
            # 如果失败，尝试常见编码
            for encoding in ['gbk', 'gb2312', 'iso-8859-1', 'cp1252']:
                try:
                    return text.encode('utf-8').decode(encoding)
                except:
                    continue
            # 如果所有编码都失败，返回原始文本
            return text
    
    def _apply_language_specific(self, text: str, language: str) -> str:
        """应用语言特定处理"""
        if language == 'zh-CN':
            # 中文特定处理：全角字符转换、标点规范化等
            import re
            # 将英文标点转换为中文标点（示例）
            text = text.replace(',', '，').replace('.', '。').replace('!', '！').replace('?', '？')
            # 合并多个中文标点
            text = re.sub(r'[，。！？]{2,}', lambda m: m.group()[0], text)
        elif language == 'en':
            # 英文特定处理：大小写规范化、缩写处理等
            text = text.lower()  # 简单示例，实际可能需要更复杂的处理
        return text

# 使用示例
config = CleaningConfig(
    remove_html=True,
    normalize_whitespace=True,
    fix_encoding=True,
    language_specific='zh-CN',
    custom_rules=[
        lambda x: x.replace('广告', '').replace('推广', '')  # 简单广告过滤
    ]
)

cleaner = ConfigurableCleaner(config)
cleaned_documents = cleaner.transform_documents(documents)
```

### 多语言混合文档处理

```python
from langid import classify
import re

class MultilingualCleaner:
    """多语言文档清洗器"""
    
    def __init__(self):
        self.language_handlers = {
            'zh': self._clean_chinese,
            'en': self._clean_english,
            'ja': self._clean_japanese,
            'ko': self._clean_korean,
            'default': self._clean_generic
        }
        
    def clean_document(self, document):
        """清洗多语言文档"""
        content = document.page_content
        metadata = document.metadata.copy()
        
        # 检测语言
        lang, confidence = classify(content)
        metadata['detected_language'] = lang
        metadata['language_confidence'] = confidence
        
        # 应用语言特定清洗
        handler = self.language_handlers.get(lang, self.language_handlers['default'])
        cleaned_content = handler(content)
        
        # 应用通用清洗（所有语言）
        cleaned_content = self._apply_common_cleaning(cleaned_content)
        
        return Document(
            page_content=cleaned_content,
            metadata=metadata
        )
    
    def _clean_chinese(self, text):
        """中文文档清洗"""
        # 全角字符转换
        text = text.replace('，', ', ').replace('。', '. ')
        # 去除中文空格
        text = re.sub(r'[\u3000]+', ' ', text)
        # 规范化标点
        text = re.sub(r'[！!]{2,}', '!', text)
        text = re.sub(r'[？?]{2,}', '?', text)
        return text
    
    def _clean_english(self, text):
        """英文文档清洗"""
        # 大小写规范化（保留专有名词可能需要的更复杂处理）
        # 简单示例：仅规范化句子开头
        sentences = re.split(r'(?<=[.!?])\s+', text)
        cleaned_sentences = []
        for sentence in sentences:
            if sentence:
                cleaned_sentences.append(sentence[0].upper() + sentence[1:].lower())
        return ' '.join(cleaned_sentences)
    
    def _clean_japanese(self, text):
        """日文文档清洗"""
        # 日文特定处理
        # 全角字符处理、标点规范化等
        return text
    
    def _clean_korean(self, text):
        """韩文文档清洗"""
        # 韩文特定处理
        return text
    
    def _clean_generic(self, text):
        """通用清洗"""
        return text
    
    def _apply_common_cleaning(self, text):
        """应用所有语言的通用清洗"""
        # 移除控制字符
        text = re.sub(r'[\x00-\x08\x0B\x0C\x0E-\x1F\x7F]', '', text)
        # 规范化换行符
        text = text.replace('\r\n', '\n').replace('\r', '\n')
        # 合并多余空白
        text = re.sub(r'\s+', ' ', text)
        return text.strip()
```

### Q1: 如何处理HTML文档中的JavaScript和CSS内容？
**A:** LangChain的Html2TextTransformer会自动过滤script和style标签，但某些情况下可能需要自定义处理：

```python
from bs4 import BeautifulSoup

def advanced_html_cleaner(html_content):
    """高级HTML清洗器，彻底移除脚本和样式"""
    soup = BeautifulSoup(html_content, 'html.parser')
    
    # 移除所有脚本标签
    for script in soup.find_all('script'):
        script.decompose()
        
    # 移除所有样式标签
    for style in soup.find_all('style'):
        style.decompose()
        
    # 移除内联样式
    for tag in soup.find_all(True):
        if 'style' in tag.attrs:
            del tag.attrs['style']
    
    # 提取文本
    text = soup.get_text(separator=' ', strip=True)
    
    # 进一步规范化
    import re
    text = re.sub(r'\s+', ' ', text)
    
    return text
```

### Q2: 清洗过程中如何保留重要格式信息？
**A:** 采用选择性清洗策略，区分内容格式和噪声格式：

```python
class SelectiveCleaner:
    """选择性清洗器，保留重要格式"""
    
    def clean_with_preservation(self, text, preserve_patterns=None):
        """
        清洗文本，但保留指定模式
        
        参数：
            text: 原始文本
            preserve_patterns: 要保留的正则表达式模式列表
        """
        if preserve_patterns is None:
            preserve_patterns = [
                r'\*\*.+?\*\*',  # 保留加粗文本
                r'_.+?_',        # 保留斜体文本
                r'`.+?`',        # 保留代码片段
                r'#+ .+',        # 保留标题
            ]
        
        # 提取和临时存储要保留的内容
        preserved_content = {}
        for i, pattern in enumerate(preserve_patterns):
            matches = re.findall(pattern, text)
            for j, match in enumerate(matches):
                placeholder = f"__PRESERVE_{i}_{j}__"
                text = text.replace(match, placeholder)
                preserved_content[placeholder] = match
        
        # 应用基础清洗
        text = self._basic_cleaning(text)
        
        # 恢复保留的内容
        for placeholder, original in preserved_content.items():
            text = text.replace(placeholder, original)
        
        return text
    
    def _basic_cleaning(self, text):
        """基础清洗"""
        import re
        # 移除多余空白，但保留单个换行
        text = re.sub(r'[ \t]+', ' ', text)
        text = re.sub(r'\n\s*\n', '\n\n', text)
        return text.strip()
```

### Q3: 大规模文档清洗如何优化性能？
**A:** 采用并行处理和增量处理策略：

```python
from concurrent.futures import ThreadPoolExecutor, as_completed
from tqdm import tqdm

class ParallelCleaner:
    """并行文档清洗器"""
    
    def __init__(self, max_workers=4):
        self.max_workers = max_workers
        self.cleaner = ConfigurableCleaner(CleaningConfig())
    
    def clean_batch(self, documents, batch_size=100):
        """批量并行清洗"""
        cleaned_docs = []
        
        # 分批处理避免内存溢出
        for i in range(0, len(documents), batch_size):
            batch = documents[i:i+batch_size]
            
            with ThreadPoolExecutor(max_workers=self.max_workers) as executor:
                # 提交清洗任务
                future_to_doc = {
                    executor.submit(self.cleaner.transform_documents, [doc]): doc 
                    for doc in batch
                }
                
                # 收集结果
                for future in tqdm(as_completed(future_to_doc), total=len(batch), desc=f"清洗批次 {i//batch_size + 1}"):
                    try:
                        result = future.result()
                        cleaned_docs.extend(result)
                    except Exception as e:
                        doc = future_to_doc[future]
                        print(f"清洗文档时出错 {doc.metadata.get('source', 'unknown')}: {e}")
        
        return cleaned_docs
```

### Q4: 如何评估清洗效果？
**A:** 建立量化评估指标：

```python
class CleaningEvaluator:
    """清洗效果评估器"""
    
    def evaluate(self, original_docs, cleaned_docs):
        """评估清洗效果"""
        metrics = {
            'total_documents': len(original_docs),
            'total_characters_removed': 0,
            'avg_reduction_rate': 0,
            'quality_scores': []
        }
        
        for orig, cleaned in zip(original_docs, cleaned_docs):
            # 计算长度变化
            orig_len = len(orig.page_content)
            cleaned_len = len(cleaned.page_content)
            reduction = orig_len - cleaned_len
            
            metrics['total_characters_removed'] += reduction
            
            # 计算质量分数（简化版）
            quality_score = self._calculate_quality_score(cleaned.page_content)
            metrics['quality_scores'].append(quality_score)
        
        # 计算平均指标
        if metrics['total_documents'] > 0:
            total_orig_len = sum(len(d.page_content) for d in original_docs)
            metrics['avg_reduction_rate'] = (
                metrics['total_characters_removed'] / total_orig_len * 100
            )
            metrics['avg_quality_score'] = sum(metrics['quality_scores']) / len(metrics['quality_scores'])
        
        return metrics
    
    def _calculate_quality_score(self, text):
        """计算文本质量分数（0-100）"""
        # 简化实现：基于可读性指标
        if not text:
            return 0
        
        # 计算平均句子长度
        sentences = re.split(r'[.!?]+', text)
        if sentences:
            avg_sentence_len = sum(len(s.split()) for s in sentences if s.strip()) / len(sentences)
        else:
            avg_sentence_len = 0
        
        # 计算词汇多样性（简化）
        words = text.lower().split()
        unique_words = set(words)
        diversity = len(unique_words) / len(words) if words else 0
        
        # 综合评分
        score = min(100, avg_sentence_len * 5 + diversity * 50)
        return score
```

### Q5: 清洗过程中如何避免信息丢失？
**A:** 实施保守清洗策略和内容备份机制：

```python
class ConservativeCleaner:
    """保守清洗器，最大限度保留信息"""
    
    def __init__(self, backup_original=True):
        self.backup_original = backup_original
    
    def clean_conservatively(self, document):
        """保守清洗，记录所有变更"""
        original_content = document.page_content
        metadata = document.metadata.copy()
        
        if self.backup_original:
            metadata['original_content_backup'] = original_content
        
        # 只进行最必要的清洗
        cleaned_content = self._minimal_cleaning(original_content)
        
        # 记录变更日志
        changes = self._analyze_changes(original_content, cleaned_content)
        metadata['cleaning_changes'] = changes
        
        return Document(
            page_content=cleaned_content,
            metadata=metadata
        )
    
    def _minimal_cleaning(self, text):
        """最小化清洗：只处理明显噪声"""
        import re
        # 仅移除控制字符和合并连续空白
        text = re.sub(r'[\x00-\x08\x0B\x0C\x0E-\x1F\x7F]', '', text)
        text = re.sub(r'\s+', ' ', text)
        return text.strip()
    
    def _analyze_changes(self, original, cleaned):
        """分析清洗变更"""
        changes = {
            'characters_removed': len(original) - len(cleaned),
            'whitespace_changes': self._count_whitespace_changes(original, cleaned),
            'has_html_removed': '<' in original and '<' not in cleaned,
            'significant_changes': len(original) - len(cleaned) > len(original) * 0.1  # 超过10%变化
        }
        return changes
```
