# 4.2.2 网页爬取与API数据加载

## 概念讲解（文字+图示）

在当今数据驱动的AI应用开发中，静态文件系统只是数据来源的一部分。大量的有价值数据存在于**动态网络环境**中：实时更新的新闻网站、社交媒体内容、API服务提供的结构化数据、动态生成的网页内容等。网页爬取与API数据加载技术使得LangChain能够从这些动态源中获取数据，为AI应用提供**实时、多样、丰富**的数据输入。

### 网络数据加载的独特性

与文件系统加载相比，网络数据加载面临独特的挑战：

```
┌─────────────────────────────────────────────────────────────┐
│               网络数据 vs 文件系统数据                      │
├─────────────────────────────────────────────────────────────┤
│   动态性      实时更新，内容变化频繁                        │
│               vs 静态文件，相对稳定                         │
├─────────────────────────────────────────────────────────────┤
│   结构复杂度  非结构化HTML、半结构化JSON、完全结构化API      │
│               vs 相对固定的文件格式                         │
├─────────────────────────────────────────────────────────────┤
│   访问限制    速率限制、身份验证、反爬虫机制                 │
│               vs 文件系统权限控制                           │
├─────────────────────────────────────────────────────────────┤
│   网络因素    延迟、超时、连接中断、代理需求                 │
│               vs 本地IO延迟                                 │
└─────────────────────────────────────────────────────────────┘
```

### LangChain的网络数据加载架构

LangChain的网络数据加载采用**分层设计**，针对不同类型的网络源提供专门解决方案：

```
┌─────────────────────────────────────────────────────────────┐
│                  网络数据加载架构                           │
├─────────────────────────────────────────────────────────────┤
│  客户端层：HTTP客户端、浏览器模拟、API客户端                 │
│  ├─ aiohttp（异步HTTP）                                     │
│  ├─ requests（同步HTTP）                                    │
│  ├─ Playwright/Chromium（浏览器渲染）                        │
│  └─ 自定义客户端                                            │
├─────────────────────────────────────────────────────────────┤
│  加载器层：针对特定网络源的专用加载器                       │
│  ├─ AsyncHtmlLoader（异步网页加载）                         │
│  ├─ WebBaseLoader（基础网页加载）                           │
│  ├─ AsyncChromiumLoader（浏览器渲染加载）                    │
│  ├─ APILoader（通用API加载）                                │
│  └─ 第三方服务加载器（GitHub、Twitter等）                    │
├─────────────────────────────────────────────────────────────┤
│  解析器层：内容提取与结构化转换                            │
│  ├─ BeautifulSoupTransformer（HTML解析）                     │
│  ├─ MozillaReadabilityTransformer（可读性提取）              │
│  ├─ JSON解析器                                              │
│  └─ XML解析器                                               │
└─────────────────────────────────────────────────────────────┘
```

这种架构帮助开发者屏蔽了复杂的网络编程和解析细节，提供了统一的数据加载接口。

## 核心要点（重点标记）

### 🔑 **网页爬取的核心加载器**

根据LangChain v1.2.22官方文档，主要的网页爬取加载器包括：

1. **AsyncHtmlLoader**：**异步HTML加载器**，基于aiohttp实现，支持并发请求，适合大规模网页抓取
2. **WebBaseLoader**：**基础网页加载器**，使用requests库，适合简单、同步的网页加载场景
3. **AsyncChromiumLoader**：**浏览器渲染加载器**，使用Playwright驱动真实浏览器，适合JavaScript渲染的现代网页
4. **HTMLWebBaseLoader**：**HTML特定加载器**，专注于HTML文档的标准化加载

### 🔑 **API数据加载的核心组件**

1. **通用API加载器**：通过`requests`或`aiohttp`直接调用REST API
2. **结构化API加载器**：针对特定API格式（如GitHub API、Twitter API）的专用加载器
3. **GraphQL加载器**：支持GraphQL查询的特殊API加载器
4. **WebSocket加载器**：支持实时数据流的加载器

### 🔑 **内容解析与转换**

1. **BeautifulSoupTransformer**：基于BeautifulSoup4的HTML解析器，支持CSS选择器和标签提取
2. **MozillaReadabilityTransformer**：使用Mozilla的Readability算法，提取网页主要内容
3. **HTML2TextTransformer**：将HTML转换为纯文本，保留基本格式
4. **自定义解析器**：支持XPath、正则表达式等自定义解析规则

### 🔑 **高级特性与优化**

1. **并发控制**：支持请求速率限制、连接池管理、并发数控制
2. **错误处理**：智能重试机制、超时控制、错误分类处理
3. **缓存策略**：支持HTTP缓存、本地缓存、分布式缓存
4. **身份认证**：支持OAuth、API密钥、Cookie、Session等多种认证方式
5. **代理支持**：支持HTTP/HTTPS/SOCKS代理，适应复杂网络环境

### 示例1：使用AsyncHtmlLoader异步抓取网页

```python
# LangChain v1.2.22 异步网页抓取示例
import asyncio
from langchain_community.document_loaders import AsyncHtmlLoader
from langchain_community.document_transformers import BeautifulSoupTransformer

async def async_web_scraping_example():
    """异步网页抓取与解析示例"""
    
    # 1. 准备要抓取的URL列表
    urls = [
        "https://news.ycombinator.com",
        "https://github.com/langchain-ai/langchain",
        "https://www.reddit.com/r/MachineLearning/",
        "https://arxiv.org/list/cs.AI/recent",
        "https://huggingface.co/papers"
    ]
    
    print(f"开始异步抓取 {len(urls)} 个网页...")
    
    # 2. 创建AsyncHtmlLoader实例（支持代理配置）
    loader = AsyncHtmlLoader(
        urls, 
        requests_per_second=2,  # 限制请求频率，避免被封IP
        raise_for_status=True,   # 对HTTP错误抛出异常
        trust_env=True          # 使用系统代理配置
    )
    
    # 3. 异步抓取网页
    try:
        raw_docs = await loader.load()
        print(f"抓取完成，获得 {len(raw_docs)} 个原始文档")
        
        # 4. 使用BeautifulSoupTransformer提取主要内容
        transformer = BeautifulSoupTransformer()
        
        # 配置要提取的HTML标签
        transformed_docs = transformer.transform_documents(
            raw_docs,
            tags_to_extract=["p", "h1", "h2", "h3", "article", "section"],
            remove_comments=True,
            unwrap_tags=["span", "div"]  # 解开这些标签，保留内容
        )
        
        print(f"解析完成，获得 {len(transformed_docs)} 个清洗后的文档")
        
        # 5. 显示抓取结果
        for i, doc in enumerate(transformed_docs[:3]):  # 只显示前3个
            url = urls[i] if i < len(urls) else "未知URL"
            content_preview = doc.page_content[:200].replace("\n", " ")
            
            print(f"\n📄 文档 {i+1}: {url}")
            print(f"   内容预览: {content_preview}...")
            print(f"   元数据键: {list(doc.metadata.keys())}")
            
            # 提取统计信息
            word_count = len(doc.page_content.split())
            print(f"   单词数: {word_count}")
        
        return transformed_docs
        
    except Exception as e:
        print(f"抓取过程中发生错误: {e}")
        import traceback
        traceback.print_exc()
        return []

# 运行异步函数
if __name__ == "__main__":
    # Python 3.10+异步执行
    docs = asyncio.run(async_web_scraping_example())
    
    if docs:
        total_words = sum(len(doc.page_content.split()) for doc in docs)
        print(f"\n✅ 总计抓取 {len(docs)} 个文档，{total_words} 个单词")
```

### 示例2：使用WebBaseLoader加载单个网页

```python
# 使用WebBaseLoader加载单个网页
from langchain_community.document_loaders import WebBaseLoader
from langchain_community.document_transformers import Html2TextTransformer

def load_single_webpage(url: str):
    """加载单个网页并转换为纯文本"""
    
    print(f"开始加载: {url}")
    
    # 1. 创建WebBaseLoader实例
    loader = WebBaseLoader(
        url,
        continue_on_failure=False,  # 加载失败时抛出异常
        header_template={
            "User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36",
            "Accept": "text/html,application/xhtml+xml,application/xml;q=0.9,*/*;q=0.8",
            "Accept-Language": "zh-CN,zh;q=0.9,en;q=0.8",
            "Accept-Encoding": "gzip, deflate, br",
            "Connection": "keep-alive",
            "Upgrade-Insecure-Requests": "1"
        }
    )
    
    # 2. 加载网页
    try:
        documents = loader.load()
        
        if not documents:
            print(f"⚠️ 未获取到内容: {url}")
            return []
        
        print(f"✓ 原始HTML加载完成，长度: {len(documents[0].page_content)} 字符")
        
        # 3. 使用Html2TextTransformer转换为纯文本
        transformer = Html2TextTransformer(
            ignore_links=False,    # 保留链接
            ignore_images=True,    # 忽略图片
            bodywidth=0           # 不限制行宽
        )
        
        text_docs = transformer.transform_documents(documents)
        
        if text_docs:
            text_content = text_docs[0].page_content
            print(f"✓ 转换为纯文本，长度: {len(text_content)} 字符")
            print(f"   预览: {text_content[:300]}...")
            
            # 提取元数据
            metadata = text_docs[0].metadata
            print(f"   元数据: { {k: v for k, v in metadata.items() if k != 'source'} }")
        
        return text_docs
        
    except Exception as e:
        print(f"❌ 加载失败: {url} - {e}")
        return []

# 使用示例
if __name__ == "__main__":
    # 测试多个网站
    test_urls = [
        "https://docs.langchain.com",
        "https://python.langchain.com/docs/get_started/introduction",
        "https://github.com/langchain-ai/langchain"
    ]
    
    all_docs = []
    for url in test_urls:
        docs = load_single_webpage(url)
        all_docs.extend(docs)
        print()  # 空行分隔
    
    print(f"📊 总计加载 {len(all_docs)} 个网页文档")
```

```python
import asyncio
from langchain_community.document_loaders import AsyncChromiumLoader
from langchain_community.document_transformers import BeautifulSoupTransformer

async def load_javascript_rendered_page():
    """加载需要JavaScript渲染的现代网页"""
    
    # 需要先安装Playwright: pip install playwright && playwright install
    
    print("开始加载JavaScript渲染的网页...")
    
    # 1. 创建AsyncChromiumLoader实例

        wait_until="networkidle", # 等待网络空闲
        viewport={"width": 1920, "height": 1080}
    )
    
    # 2. 异步加载
    try:
        raw_docs = await loader.load()
        print(f"浏览器渲染完成，获得 {len(raw_docs)} 个文档")

        }
        
        transformed_docs = []
        for i, doc in enumerate(raw_docs):
            url = doc.metadata.get("source", "unknown")
            
            # 根据域名选择合适的选择器
            selectors = ["p", "h1", "h2", "h3", "div", "span"]
            for domain, domain_selectors in site_specific_selectors.items():
                if domain in url:
                    selectors = domain_selectors
                    break
            
            # 转换文档

                [doc],

        return transformed_docs
```

### 示例4：API数据加载与处理

```python
# API数据加载示例
import requests
import json
from typing import List, Dict, Any
from langchain_core.documents import Document

class GitHubAPILoader:
    """GitHub API数据加载器"""
    
    def __init__(self, access_token: str = None, per_page: int = 100):
        self.access_token = access_token
        self.per_page = per_page
        self.base_url = "https://api.github.com"
        self.headers = {
            "Accept": "application/vnd.github.v3+json"
        }
        
        if access_token:
            self.headers["Authorization"] = f"token {access_token}"
    
    def load_repository_issues(self, owner: str, repo: str, state: str = "open"):
        """加载仓库的Issues"""
        url = f"{self.base_url}/repos/{owner}/{repo}/issues"
        params = {
            "state": state,
            "per_page": self.per_page,
            "page": 1
        }
        
        print(f"开始加载 {owner}/{repo} 的 {state} issues...")
        
        all_issues = []
        documents = []
        
        while True:
            try:
                response = requests.get(url, headers=self.headers, params=params)
                response.raise_for_status()
                
                issues = response.json()
                if not issues:
                    break
                
                all_issues.extend(issues)
                
                # 转换为Document对象
                for issue in issues:
                    doc = self._issue_to_document(issue, owner, repo)
                    documents.append(doc)
                
                print(f"  已加载第 {params['page']} 页，共 {len(issues)} 个issues")
                
                # 检查是否有下一页
                if "next" in response.links:
                    params["page"] += 1
                else:
                    break
                    
            except requests.exceptions.RequestException as e:
                print(f"❌ API请求失败: {e}")
                break
        
        print(f"✅ 加载完成，共 {len(documents)} 个issue文档")
        return documents
    
    def _issue_to_document(self, issue: Dict[str, Any], owner: str, repo: str) -> Document:
        """将GitHub Issue转换为Document对象"""
        
        # 构建主要内容
        content_parts = [
            f"Title: {issue.get('title', 'No Title')}",
            f"State: {issue.get('state', 'unknown')}",
            f"Created: {issue.get('created_at', 'unknown')}",
            f"Updated: {issue.get('updated_at', 'unknown')}",
            "",
            f"Body: {issue.get('body', 'No description')}",
        ]
        
        # 添加评论（如果有）
        if issue.get('comments', 0) > 0:
            content_parts.append(f"\nComments: {issue.get('comments')}")
        
        # 添加标签
        labels = issue.get('labels', [])
        if labels:
            label_names = [label['name'] for label in labels]
            content_parts.append(f"\nLabels: {', '.join(label_names)}")
        
        content = "\n".join(content_parts)
        
        # 构建元数据
        metadata = {
            "source": f"github:{owner}/{repo}/issues/{issue.get('number')}",
            "type": "github_issue",
            "repository": f"{owner}/{repo}",
            "issue_number": issue.get('number'),
            "issue_id": issue.get('id'),
            "author": issue.get('user', {}).get('login', 'unknown'),
            "state": issue.get('state', 'unknown'),
            "created_at": issue.get('created_at'),
            "updated_at": issue.get('updated_at'),
            "comments_count": issue.get('comments', 0),
            "labels_count": len(labels),
            "url": issue.get('html_url', '')
        }
        
        return Document(page_content=content, metadata=metadata)

# 使用GitHub API加载器
def load_github_data_example():
    """GitHub数据加载示例"""
    
    # 注意：需要GitHub Personal Access Token来避免速率限制
    # 可以从 https://github.com/settings/tokens 获取
    
    # 如果没有token，可以使用匿名访问（但有限制）

        # 按作者统计
        authors = {}

        for doc in docs:
            author = doc.metadata.get('author', 'unknown')
            state = doc.metadata.get('state', 'unknown')
            
            authors[author] = authors.get(author, 0) + 1
            states[state] = states.get(state, 0) + 1
        
        print(f"- 作者数: {len(authors)}")

        # 显示示例文档

    return docs
```

### 1. 构建分布式网页爬取系统

```python
import asyncio
from typing import List, Set
from urllib.parse import urlparse
import aiohttp
import hashlib
from datetime import datetime
from concurrent.futures import ProcessPoolExecutor

class DistributedWebCrawler:
    """分布式网页爬取系统"""
    
    def __init__(self, 
                 max_concurrent: int = 10,
                 max_depth: int = 3,
                 respect_robots_txt: bool = True,
                 user_agent: str = None):
        
        self.max_concurrent = max_concurrent
        self.max_depth = max_depth
        self.respect_robots_txt = respect_robots_txt
        self.user_agent = user_agent or "LangChainWebCrawler/1.0"
        
        # 状态跟踪
        self.visited_urls: Set[str] = set()
        self.url_queue = asyncio.Queue()
        self.documents = []
        self.robots_cache = {}
        
    async def crawl(self, start_urls: List[str], 
                   domain_restriction: str = None):
        """开始爬取"""
        
        print(f"🚀 开始分布式爬取，起始URL: {len(start_urls)} 个")
        
        # 初始化队列
        for url in start_urls:
            await self.url_queue.put((url, 0))  # (url, depth)
        
        # 创建工作者任务
        tasks = []
        for i in range(self.max_concurrent):
            task = asyncio.create_task(
                self._crawl_worker(f"Worker-{i+1}", domain_restriction)
            )
            tasks.append(task)
        
        # 等待所有工作者完成
        await self.url_queue.join()
        
        # 取消工作者任务
        for task in tasks:
            task.cancel()
        
        # 等待任务完成
        await asyncio.gather(*tasks, return_exceptions=True)
        
        print(f"✅ 爬取完成，共访问 {len(self.visited_urls)} 个URL，获取 {len(self.documents)} 个文档")
        return self.documents
    
    async def _crawl_worker(self, worker_name: str, domain_restriction: str):
        """爬取工作者"""
        
        async with aiohttp.ClientSession() as session:
            while True:
                try:
                    # 从队列获取URL
                    url, depth = await asyncio.wait_for(
                        self.url_queue.get(), 
                        timeout=5.0
                    )
                    
                    # 检查深度限制
                    if depth > self.max_depth:
                        self.url_queue.task_done()
                        continue
                    
                    # 检查是否已访问
                    url_hash = hashlib.md5(url.encode()).hexdigest()
                    if url_hash in self.visited_urls:
                        self.url_queue.task_done()
                        continue
                    
                    # 检查robots.txt（如果启用）
                    if self.respect_robots_txt:
                        if not await self._check_robots_txt(url, session):
                            self.url_queue.task_done()
                            continue
                    
                    print(f"{worker_name}: 爬取深度{depth} - {url}")
                    
                    # 获取页面内容
                    document = await self._fetch_url(url, session, depth)
                    if document:
                        self.documents.append(document)
                    
                    # 标记为已访问
                    self.visited_urls.add(url_hash)
                    
                    # 提取新链接（如果未达到深度限制）
                    if depth < self.max_depth and document:
                        new_urls = await self._extract_links(
                            document.page_content, 
                            url,
                            domain_restriction
                        )
                        
                        for new_url in new_urls:
                            new_url_hash = hashlib.md5(new_url.encode()).hexdigest()
                            if new_url_hash not in self.visited_urls:
                                await self.url_queue.put((new_url, depth + 1))
                    
                    self.url_queue.task_done()
                    
                except asyncio.TimeoutError:
                    # 队列为空，工作者可以退出
                    break
                except Exception as e:
                    print(f"{worker_name}: 错误 - {e}")
                    self.url_queue.task_done()
    
    async def _fetch_url(self, url: str, session: aiohttp.ClientSession, depth: int):
        """获取URL内容"""
```

```python
import time

from datetime import datetime, timedelta
import asyncio

@dataclass
class APIMonitorConfig:
    """API监控配置"""
    endpoint: str
    interval_seconds: int = 60
    method: str = "GET"
    headers: Dict[str, str] = field(default_factory=dict)
    params: Dict[str, Any] = field(default_factory=dict)
    data: Any = None
    timeout: int = 30
    expected_status_codes: List[int] = field(default_factory=lambda: [200])
    
@dataclass
class APIChange:
    """API变化记录"""
    timestamp: datetime
    endpoint: str
    change_type: str  # "added", "modified", "removed"
    data_before: Any = None
    data_after: Any = None
    diff_summary: str = ""

class APIMonitor:
    """API数据监控器"""
    
    def __init__(self, configs: List[APIMonitorConfig]):
        self.configs = configs
        self.state_history: Dict[str, List[Dict]] = {}
        self.change_history: List[APIChange] = []
        self.running = False
        
    async def start_monitoring(self, callback: Callable = None):
        """开始监控"""
        self.running = True
        print(f"🚀 开始监控 {len(self.configs)} 个API端点")
        
        # 初始获取
        for config in self.configs:
            await self._check_endpoint(config)
        
        # 定期监控
        while self.running:
            await asyncio.sleep(1)  # 检查间隔
            
            current_time = datetime.now()
            
            for config in self.configs:
                # 检查是否到了监控时间
                last_check = self._get_last_check_time(config.endpoint)
                if last_check and (current_time - last_check).seconds < config.interval_seconds:
                    continue
                
                # 执行检查
                await self._check_endpoint(config, callback)
    
    async def _check_endpoint(self, config: APIMonitorConfig, callback: Callable = None):
        """检查API端点"""
        import aiohttp
        import difflib
        
        try:
            async with aiohttp.ClientSession() as session:
                # 准备请求参数
                kwargs = {
                    "headers": config.headers,
                    "timeout": config.timeout
                }
                
                if config.method == "GET":
                    kwargs["params"] = config.params
                else:
                    kwargs["data"] = config.data
                
                # 发送请求
                async with session.request(
                    config.method, 
                    config.endpoint, 
                    **kwargs
                ) as response:
                    
                    # 检查状态码
                    if response.status not in config.expected_status_codes:
                        print(f"⚠️ {config.endpoint}: 非预期状态码 {response.status}")
                        return
                    
                    # 获取响应数据
                    content_type = response.headers.get("Content-Type", "")
                    if "application/json" in content_type:
                        current_data = await response.json()
                    else:
                        current_data = await response.text()
                    
                    # 获取历史数据
                    endpoint_key = config.endpoint
                    if endpoint_key not in self.state_history:
                        self.state_history[endpoint_key] = []
                    
                    history = self.state_history[endpoint_key]
                    
                    # 检查是否有变化
                    if history:
                        last_data = history[-1]["data"]
                        
                        # 比较数据
                        if self._data_changed(last_data, current_data):
                            # 记录变化
                            change = APIChange(
                                timestamp=datetime.now(),
                                endpoint=config.endpoint,
                                change_type="modified",
                                data_before=last_data,
                                data_after=current_data,
                                diff_summary=self._generate_diff_summary(last_data, current_data)
                            )
                            
                            self.change_history.append(change)
                            
                            print(f"🔔 {config.endpoint}: 检测到变化")
                            print(f"   变化摘要: {change.diff_summary[:100]}...")
                            
                            # 调用回调函数
                            if callback:
                                await callback(change)
                    
                    # 记录当前状态
                    history.append({
                        "timestamp": datetime.now(),
                        "data": current_data,
                        "status_code": response.status
                    })
                    
                    # 保持最近10次记录
                    if len(history) > 10:
                        self.state_history[endpoint_key] = history[-10:]
        
        except Exception as e:
            print(f"❌ {config.endpoint}: 监控失败 - {e}")
    
    def _data_changed(self, old_data: Any, new_data: Any) -> bool:
        """检查数据是否发生变化"""
        if isinstance(old_data, dict) and isinstance(new_data, dict):
            return json.dumps(old_data, sort_keys=True) != json.dumps(new_data, sort_keys=True)
        elif isinstance(old_data, list) and isinstance(new_data, list):
            return json.dumps(old_data, sort_keys=True) != json.dumps(new_data, sort_keys=True)
        else:
            return str(old_data) != str(new_data)
    
    def _generate_diff_summary(self, old_data: Any, new_data: Any) -> str:
        """生成变化摘要"""
        old_str = json.dumps(old_data, indent=2) if isinstance(old_data, (dict, list)) else str(old_data)
        new_str = json.dumps(new_data, indent=2) if isinstance(new_data, (dict, list)) else str(new_data)
        
        # 使用difflib生成差异
        diff = difflib.unified_diff(
            old_str.splitlines(),
            new_str.splitlines(),
            lineterm=""
        )
        
        diff_lines = list(diff)
        if len(diff_lines) > 10:
            return "\n".join(diff_lines[:10]) + "\n..."
        else:
            return "\n".join(diff_lines)
    
    def _get_last_check_time(self, endpoint: str) -> datetime:
        """获取上次检查时间"""
        if endpoint in self.state_history and self.state_history[endpoint]:
            return self.state_history[endpoint][-1]["timestamp"]
        return None
    
    def stop_monitoring(self):
        """停止监控"""
        self.running = False
        print("🛑 停止API监控")
```

```python
        if avg_sentence_length < 10:

        elif avg_sentence_length < 20:
            return 0.7  # 易读
```

**A:** 处理反爬虫机制需要综合策略：

3. **IP轮换**：使用代理池，定期更换IP地址

6. **验证码处理**：集成验证码识别服务或人工干预机制

### Q2: 异步加载和同步加载在网页爬取中如何选择？

**A:** 选择建议：

1. **使用AsyncHtmlLoader的情况**：
   - 需要抓取大量网页（数百上千个）
   - 应用本身是异步架构（如FastAPI、Sanic）
   - 需要高并发性能
   - 能够处理异步编程的复杂性

2. **使用WebBaseLoader的情况**：
   - 抓取少量网页（几十个以内）
   - 同步代码架构，不想引入异步复杂性
   - 简单的脚本或批处理任务
   - 对性能要求不高

**性能对比**：
- AsyncHtmlLoader：通常比同步快3-10倍，取决于并发数和网络延迟
- WebBaseLoader：简单稳定，适合初学者和简单场景

### Q3: API数据加载时如何处理认证和速率限制？

**A:** API认证和速率限制处理策略：

1. **认证方式**：
   - **API密钥**：在请求头中添加`Authorization: Bearer <token>`
   - **OAuth 2.0**：使用专门的OAuth客户端库
   - **基本认证**：使用`auth=(username, password)`参数
   - **Cookie/Session**：维护会话状态

2. **速率限制处理**：
   - **请求队列**：实现请求队列，控制发送频率
   - **指数退避**：遇到限制时，指数级增加重试延迟
   - **分布式计数**：在多进程/多机器环境下共享请求计数
   - **监控响应头**：检查`X-RateLimit-*`头部信息

示例：
```python
import time
from tenacity import retry, stop_after_attempt, wait_exponential

@retry(
    stop=stop_after_attempt(5),
    wait=wait_exponential(multiplier=1, min=4, max=60)
)
def make_api_request_with_backoff(url, headers):
    """带指数退避的API请求"""
    response = requests.get(url, headers=headers)
    
    if response.status_code == 429:  # 速率限制
        retry_after = int(response.headers.get('Retry-After', 60))
        raise Exception(f"Rate limited, retry after {retry_after}s")
    
    response.raise_for_status()
    return response.json()
```
