# 4.2 文件类文档加载器

## 概念讲解

文档加载器（Document Loaders）是LangChain数据处理的**入口组件**，负责从各种数据源读取内容并转换为统一的Document对象。LangChain提供了100+种加载器，覆盖几乎所有常见的数据源类型。

### 加载器的统一接口

所有加载器都遵循相同的接口设计：

```
数据源 → BaseLoader.load() → List[Document]
```

这种设计让开发者只需学习一套API，就能处理PDF、文本文件、CSV、Word等多种数据源，实现**接口统一、实现多样**的设计目标。

### 文件类数据源分类

| 格式类型 | 典型扩展名 | 推荐加载器 | 特殊说明 |
|---------|-----------|-----------|---------|
| **纯文本** | .txt, .log | `TextLoader` | 最基础，支持编码指定 |
| **PDF** | .pdf | `PyPDFLoader` | 逐页加载，保留页码 |
| **CSV** | .csv | `CSVLoader` | 每行转为一个Document |
| **Word** | .docx | `Docx2txtLoader` | 保留基本格式 |
| **Excel** | .xlsx | `UnstructuredExcelLoader` | 表格数据处理 |
| **HTML** | .html | `BSHTMLLoader` | 自动清理HTML标签 |

## 核心要点

### 🔑 文本文件加载

`TextLoader`是最基础的加载器，适用于纯文本文件。核心特性包括：
- 支持指定编码（处理中文时很重要）
- 支持静默错误模式
- 支持自动编码检测

```python
from langchain_community.document_loaders import TextLoader

# 基础用法
loader = TextLoader("data/notes.txt")
documents = loader.load()

# 推荐用法（处理中文文件）
loader = TextLoader(
    "data/笔记.txt",
    encoding="utf-8",
    autodetect_encoding=False  # 禁用自动编码检测，提高性能
)
documents = loader.load()
```

### 🔑 PDF文件加载

PDF是最常见的企业文档格式。`PyPDFLoader`逐页加载，自动提取页码元数据：

```python
from langchain_community.document_loaders import PyPDFLoader

loader = PyPDFLoader("data/report.pdf")
documents = loader.load()

print(f"加载了 {len(documents)} 页")
print(f"第一页内容: {documents[0].page_content[:100]}...")
print(f"页码: {documents[0].metadata.get('page')}")

# 处理加密PDF
loader_encrypted = PyPDFLoader(
    "data/encrypted.pdf",
    password="your_password"
)
```

### 🔑 CSV文件加载

`CSVLoader`将表格数据转换为Document列表，每行数据对应一个Document：

```python
from langchain_community.document_loaders import CSVLoader

loader = CSVLoader(
    file_path="data/products.csv",
    source_column="product_name",    # 指定来源列
    metadata_columns=["category", "price"]  # 指定元数据列
)

documents = loader.load()

# 每行数据转换为一个Document
for doc in documents[:2]:
    print(f"产品: {doc.metadata.get('product_name')}")
    print(f"内容: {doc.page_content[:100]}")
```

### 🔑 批量文件加载

`DirectoryLoader`支持使用glob模式匹配批量加载文件：

```python
from langchain_community.document_loaders import DirectoryLoader
from langchain_community.document_loaders import PyPDFLoader

# 加载目录下所有PDF
loader = DirectoryLoader(
    "data/documents/",
    glob="**/*.pdf",           # 支持通配符：* 匹配文件名，** 匹配目录
    loader_cls=PyPDFLoader,    # 指定加载器类型
    show_progress=True,        # 显示进度条
    use_multithreading=True    # 启用多线程
)

documents = loader.load()
print(f"加载了 {len(documents)} 个文档")
```

## 进阶应用

### 惰性加载处理大文件

对于大型文件，使用惰性加载避免内存溢出：

```python
from langchain_community.document_loaders import PyPDFLoader

loader = PyPDFLoader("data/large_book.pdf")

# 惰性加载，逐页处理
for doc in loader.lazy_load():
    # 处理每一页
    print(f"处理第 {doc.metadata.get('page')} 页")
    # 可以在这里进行过滤、转换等操作
```

### 多格式文件统一加载

在实际项目中，经常需要处理多种文件格式。可以通过扩展名映射实现统一加载：

```python
import os
from langchain_community.document_loaders import (
    TextLoader, PyPDFLoader, CSVLoader, 
    Docx2txtLoader, BSHTMLLoader
)

# 扩展名到加载器的映射
LOADER_MAP = {
    ".txt": TextLoader,
    ".pdf": PyPDFLoader,
    ".csv": CSVLoader,
    ".docx": Docx2txtLoader,
    ".html": BSHTMLLoader,
}

def load_file(file_path: str):
    """根据文件扩展名自动选择加载器"""
    ext = os.path.splitext(file_path)[1].lower()
    loader_cls = LOADER_MAP.get(ext)
    
    if not loader_cls:
        raise ValueError(f"不支持的文件格式: {ext}")
    
    loader = loader_cls(file_path)
    return loader.load()

# 使用示例
# docs = load_file("data/report.pdf")
```

## 常见问题

### Q1: 如何处理中文编码问题？

**A:** 对于文本文件，显式指定编码：

```python
loader = TextLoader("data/file.txt", encoding="utf-8")
```

如果遇到编码问题，可以尝试使用`chardet`库自动检测编码。

### Q2: 加载器报模块缺失错误怎么办？

**A:** 许多加载器需要额外安装依赖：

```bash
# PDF处理
pip install pypdf

# Word处理
pip install docx2txt

# Excel处理
pip install unstructured openpyxl

# HTML处理
pip install beautifulsoup4
```

### Q3: DirectoryLoader的glob模式如何使用？

**A:** glob模式支持通配符匹配：
- `*.pdf`：当前目录下所有PDF
- `**/*.pdf`：递归所有子目录的PDF
- `report_*.pdf`：匹配report_开头的PDF
- `2024-??-*.pdf`：匹配2024年的PDF

## 本节总结

文件类加载器是处理本地文档的基础组件。通过统一的`load()`和`lazy_load()`接口，将各种文件格式转换为标准的Document对象。

核心要点：
1. 根据文件格式选择合适的加载器
2. 处理中文文件时注意编码问题
3. 大文件使用`lazy_load()`避免内存溢出
4. 批量文件使用`DirectoryLoader`提高效率
5. 加载时自动提取元数据（页码、来源等）
