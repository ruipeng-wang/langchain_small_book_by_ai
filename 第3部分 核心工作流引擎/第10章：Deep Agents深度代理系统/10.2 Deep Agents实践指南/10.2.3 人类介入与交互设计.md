# 10.2.3 人类介入与交互设计

## 概念讲解

人类介入与交互设计（Human-in-the-Loop, HITL）是LangChain Deep Agents架构中的关键特性，它在自动化AI工作流中引入人工监督、审批和决策点，确保系统的安全性、合规性和可控性。在复杂的业务场景中，某些操作可能涉及敏感数据、关键业务决策或高风险行为，人类介入机制为这些场景提供了必要的安全护栏。

LangChain v1.2.22通过`HumanInTheLoopMiddleware`中间件实现了强大的人类介入功能，该中间件可以：

1. **智能中断**：在代理执行敏感工具调用时自动暂停，等待人工审批
2. **决策收集**：提供标准化的决策接口，支持批准、拒绝、编辑等操作
3. **上下文保持**：在中断期间保持完整的会话状态，确保恢复时上下文不丢失
4. **异步协作**：支持多人审批、逐级审批等复杂的协作模式
5. **审计追踪**：记录所有人类介入决策，满足合规性要求

人类介入不仅是一种安全机制，更是提升AI系统可信度和实用性的重要手段。通过合理的人类介入设计，我们可以构建既自动化又可控的智能系统。

## 核心要点

### 人类介入的核心价值

- **安全控制**：防止AI执行危险或不合规的操作，如删除数据、发送敏感邮件等
- **质量保证**：通过人工审核确保输出质量，特别是在创意、法律、医疗等专业领域
- **合规要求**：满足法律法规对某些操作必须有人工审批的要求
- **用户信任**：提高用户对AI系统的信任度，知道关键决策有人工监督
- **持续学习**：通过人类反馈优化AI行为，实现持续的模型改进

### 人类介入的设计原则

1. **最小干预原则**：只在必要的时候请求人类介入，避免过度打扰
2. **上下文透明**：向人类决策者提供充分的上下文信息，支持明智决策
3. **决策标准化**：提供清晰、一致的决策选项和流程
4. **响应时效性**：考虑决策的时效要求，设计合理的超时机制
5. **可追溯性**：完整记录所有人类介入决策，支持审计和复盘

### 人类介入的触发条件

在LangChain中，人类介入主要通过以下条件触发：
- **特定工具调用**：如发送邮件、删除数据、审批流程等
- **置信度阈值**：当AI的置信度低于设定阈值时请求人工确认
- **业务规则**：基于自定义业务逻辑判断是否需要人工介入
- **异常检测**：检测到异常模式或潜在风险时自动暂停

### 基本使用：创建带有人类介入的Deep Agent

```python
from langchain_openai import ChatOpenAI
from langchain.agents import Tool
from langgraph.checkpoint.memory import InMemorySaver

# 定义工具函数
def send_email(to: str, subject: str, body: str) -> str:
    """发送邮件工具（模拟）"""

    """删除数据库工具（模拟）"""
    return f"数据库 {db_name} 已删除"

    """搜索工具（模拟）"""
    return f"搜索 '{query}' 的结果：相关文档..."

# 创建工具实例
send_email_tool = Tool(
    name="send_email",
    func=send_email,
    description="发送邮件给指定收件人"
)

delete_database_tool = Tool(
    name="delete_database", 
    func=delete_database,

    name="search",
    func=search_web,

    model=ChatOpenAI(model="gpt-4"),

            }

)

# 配置执行环境
```

### 处理人类介入中断

```python
from langgraph.types import Command
from typing import Dict, List

def get_user_decision(action_request: Dict) -> Dict:
    """模拟获取用户决策的流程"""
    print(f"需要审批的操作: {action_request['tool_name']}")
    print(f"操作参数: {action_request['tool_input']}")
    print(f"描述: {action_request.get('description', '无描述')}")
    
    # 模拟用户决策逻辑
    decision_type = input("请输入决策 (approve/reject/edit): ").strip().lower()
    
    if decision_type == "approve":
        return {"type": "approve"}
    elif decision_type == "reject":
        return {"type": "reject", "reason": "用户拒绝了该操作"}
    elif decision_type == "edit":
        # 允许用户编辑参数
        edited_params = input("请输入编辑后的参数 (JSON格式): ").strip()
        return {"type": "edit", "edited_input": edited_params}
    else:
        return {"type": "reject", "reason": "无效的决策类型"}

# 检查代理是否被中断
def handle_interrupt(result, agent, config):
    """处理人类介入中断"""
    if result.get("__interrupt__"):
        interrupts = result["__interrupt__"][0].value
        action_requests = interrupts["action_requests"]
        
        print(f"发现 {len(action_requests)} 个需要审批的操作")
        
        # 为每个操作获取用户决策
        decisions = []
        for action in action_requests:
            decision = get_user_decision(action)
            decisions.append(decision)
        
        # 使用决策恢复代理执行
        resume_result = agent.invoke(
            Command(resume={"decisions": decisions}),
            config=config
        )
        
        return resume_result
    return result

# 在主循环中使用中断处理
result = agent.invoke(
    {"messages": [{"role": "user", "content": "删除测试数据库"}]},
    config=config
)

result = handle_interrupt(result, agent, config)
print(f"最终结果: {result}")
```

### 自定义人类介入中间件

```python
from typing import Any, Dict, List, Optional
from langchain.agents.middleware.base import AgentMiddleware
from langchain.schema import AgentAction

class CustomHumanInTheLoopMiddleware(AgentMiddleware):
    """自定义人类介入中间件"""
    
    def __init__(self, 
                 approval_threshold: float = 0.8,
                 require_approval_for: List[str] = None,
                 admin_users: List[str] = None):
        super().__init__()
        self.approval_threshold = approval_threshold
        self.require_approval_for = require_approval_for or []
        self.admin_users = admin_users or []
        
    def on_agent_action(self, action: AgentAction, **kwargs) -> Optional[Dict[str, Any]]:
        """在代理执行动作时触发"""
        # 检查是否需要人工审批
        if action.tool in self.require_approval_for:
            return {
                "__interrupt__": [{
                    "type": "human_approval",
                    "tool_name": action.tool,
                    "tool_input": action.tool_input,
                    "approval_threshold": self.approval_threshold,
                    "timestamp": kwargs.get("timestamp"),
                    "session_id": kwargs.get("session_id")
                }]
            }
        return None
    
    def after_agent_action(self, action: AgentAction, result: Any, **kwargs) -> Optional[Dict[str, Any]]:
        """在代理执行动作后触发"""
        # 可以在这里记录审计日志
        audit_log = {
            "tool": action.tool,
            "input": action.tool_input,
            "result": result,
            "timestamp": kwargs.get("timestamp"),
            "approved_by": kwargs.get("approved_by", "auto")
        }
        print(f"审计日志: {audit_log}")
        return None

# 使用自定义中间件
custom_middleware = CustomHumanInTheLoopMiddleware(
    approval_threshold=0.7,
    require_approval_for=["send_email", "delete_database", "update_user_role"],
    admin_users=["admin@example.com", "supervisor@example.com"]
)
```

## 进阶应用

### 多级审批工作流

```python
from enum import Enum
from typing import Dict, List, Optional
from datetime import datetime, timedelta

class ApprovalLevel(Enum):
    """审批级别枚举"""
    TEAM_LEAD = "team_lead"
    DEPARTMENT_MANAGER = "department_manager"
    EXECUTIVE = "executive"

class MultiLevelApprovalMiddleware(AgentMiddleware):
    """多级审批中间件"""
    
    def __init__(self, approval_config: Dict[str, List[ApprovalLevel]]):
        super().__init__()
        self.approval_config = approval_config
        self.pending_approvals = {}  # 存储待审批请求
        
    def on_agent_action(self, action: AgentAction, **kwargs) -> Optional[Dict[str, Any]]:
        """检查是否需要多级审批"""
        if action.tool in self.approval_config:
            required_levels = self.approval_config[action.tool]
            
            # 创建审批请求
            approval_request = {
                "request_id": f"apr_{datetime.now().timestamp()}",
                "tool": action.tool,
                "input": action.tool_input,
                "required_levels": required_levels,
                "current_level": 0,
                "approvals": [],
                "created_at": datetime.now(),
                "timeout": datetime.now() + timedelta(hours=24)  # 24小时超时
            }
            
            # 保存审批请求
            self.pending_approvals[approval_request["request_id"]] = approval_request
            
            # 触发第一级审批
            return self._create_interrupt(approval_request, required_levels[0])
        return None
    
    def _create_interrupt(self, request: Dict, level: ApprovalLevel) -> Dict:
        """创建中断请求"""
        return {
            "__interrupt__": [{
                "type": "multi_level_approval",
                "request_id": request["request_id"],
                "tool_name": request["tool"],
                "tool_input": request["input"],
                "approval_level": level.value,
                "description": f"需要 {level.value} 审批",
                "allowed_decisions": ["approve", "reject", "escalate"]
            }]
        }
    
    def process_approval_decision(self, request_id: str, decision: str, approver: str) -> Optional[Dict]:
        """处理审批决策"""
        if request_id not in self.pending_approvals:
            return None
            
        request = self.pending_approvals[request_id]
        request["approvals"].append({
            "decision": decision,
            "approver": approver,
            "timestamp": datetime.now()
        })
        
        current_level_idx = request["current_level"]
        required_levels = request["required_levels"]
        
        if decision == "approve":
            # 当前级别批准，检查是否需要下一级
            if current_level_idx + 1 < len(required_levels):
                request["current_level"] += 1
                return self._create_interrupt(request, required_levels[current_level_idx + 1])
            else:
                # 所有级别都批准了
                del self.pending_approvals[request_id]
                return {"type": "all_approved", "request_id": request_id}
        elif decision == "reject":
            # 任何级别拒绝都终止
            del self.pending_approvals[request_id]
            return {"type": "rejected", "request_id": request_id}
        elif decision == "escalate":
            # 升级到更高权限
            # 这里可以触发通知或升级流程
            return {"type": "escalated", "request_id": request_id}
```

```python
import requests
from typing import Dict, Any, Optional
import json

class ExternalApprovalSystemMiddleware(AgentMiddleware):
    """集成外部审批系统的中间件"""
    
    def __init__(self, 
                 api_endpoint: str,
                 api_key: str,
                 approval_workflow_id: str):
        super().__init__()
        self.api_endpoint = api_endpoint
        self.api_key = api_key
        self.approval_workflow_id = approval_workflow_id
        self.headers = {
            "Authorization": f"Bearer {api_key}",
            "Content-Type": "application/json"
        }
    
    def on_agent_action(self, action: AgentAction, **kwargs) -> Optional[Dict[str, Any]]:
        """创建外部审批请求"""
        # 构建审批请求
        approval_request = {
            "workflow_id": self.approval_workflow_id,
            "tool_name": action.tool,
            "tool_input": action.tool_input,
            "context": {
                "session_id": kwargs.get("session_id"),
                "user_id": kwargs.get("user_id"),
                "timestamp": kwargs.isoformat() if hasattr(kwargs.get("timestamp"), "isoformat") else str(kwargs.get("timestamp"))
            },
            "callback_url": "https://your-app.com/approval/callback"  # 审批完成后的回调地址
        }
        
        try:
            # 发送审批请求到外部系统
            response = requests.post(
                f"{self.api_endpoint}/approval-requests",
                headers=self.headers,
                json=approval_request,
                timeout=10
            )
            
            if response.status_code == 201:
                approval_data = response.json()
                return {
                    "__interrupt__": [{
                        "type": "external_approval",
                        "approval_id": approval_data["id"],
                        "status": approval_data["status"],
                        "estimated_time": approval_data.get("estimated_completion_time"),
                        "message": "已提交外部审批系统，请等待审批结果"
                    }]
                }
        except Exception as e:
            print(f"外部审批系统调用失败: {e}")
            
        return None
    
    def check_approval_status(self, approval_id: str) -> Optional[Dict]:
        """检查外部审批状态"""
        try:
            response = requests.get(
                f"{self.api_endpoint}/approval-requests/{approval_id}",
                headers=self.headers,
                timeout=5
            )
            
            if response.status_code == 200:
                status_data = response.json()
                return {
                    "approval_id": approval_id,
                    "status": status_data["status"],
                    "decision": status_data.get("decision"),
                    "decision_reason": status_data.get("decision_reason"),
                    "decided_at": status_data.get("decided_at"),
                    "decided_by": status_data.get("decided_by")
                }
        except Exception as e:
            print(f"检查审批状态失败: {e}")
            
        return None
```

### 人类反馈学习系统

```python
from typing import Dict, List, Any
import pandas as pd
from dataclasses import dataclass
from datetime import datetime

@dataclass
class HumanFeedback:
    """人类反馈数据结构"""
    session_id: str
    tool_name: str
    tool_input: Dict[str, Any]
    ai_decision: Dict[str, Any]
    human_decision: Dict[str, Any]
    feedback_score: float  # 0-1，表示人类对AI决策的满意度
    feedback_comment: str
    timestamp: datetime

class HumanFeedbackLearningMiddleware(AgentMiddleware):
    """人类反馈学习中间件"""
    
    def __init__(self, feedback_storage_path: str = "human_feedback.csv"):
        super().__init__()
        self.feedback_storage_path = feedback_storage_path
        self.feedback_history: List[HumanFeedback] = []
        
    def after_agent_action(self, action: AgentAction, result: Any, **kwargs) -> Optional[Dict[str, Any]]:
        """收集人类反馈"""
        # 检查是否有相关的人类介入决策
        human_decision = kwargs.get("human_decision")
        if human_decision:
            # 创建反馈记录
            feedback = HumanFeedback(
                session_id=kwargs.get("session_id", "unknown"),
                tool_name=action.tool,
                tool_input=action.tool_input,
                ai_decision={
                    "tool": action.tool,
                    "input": action.tool_input,
                    "confidence": kwargs.get("confidence", 1.0)
                },
                human_decision=human_decision,
                feedback_score=self._calculate_feedback_score(human_decision),
                feedback_comment=human_decision.get("comment", ""),
                timestamp=datetime.now()
            )
            
            self.feedback_history.append(feedback)
            self._save_feedback(feedback)
            
            # 可以在这里触发模型微调或行为调整
            if len(self.feedback_history) % 100 == 0:  # 每100条反馈触发一次学习
                self._trigger_learning()
        
        return None
    
    def _calculate_feedback_score(self, human_decision: Dict) -> float:
        """计算反馈分数"""
        decision_type = human_decision.get("type", "")
        
        if decision_type == "approve":
            return 1.0  # 完全批准表示AI决策正确
        elif decision_type == "edit":
            return 0.5  # 需要编辑表示部分正确
        elif decision_type == "reject":
            return 0.0  # 拒绝表示AI决策错误
        else:
            return 0.5  # 默认中性分数
    
    def _save_feedback(self, feedback: HumanFeedback):
        """保存反馈到文件"""
        feedback_dict = {
            "session_id": feedback.session_id,
            "tool_name": feedback.tool_name,
            "tool_input": str(feedback.tool_input),
            "ai_decision": str(feedback.ai_decision),
            "human_decision": str(feedback.human_decision),
            "feedback_score": feedback.feedback_score,
            "feedback_comment": feedback.feedback_comment,
            "timestamp": feedback.timestamp.isoformat()
        }
        
        # 追加到CSV文件
        df = pd.DataFrame([feedback_dict])
        try:
            existing_df = pd.read_csv(self.feedback_storage_path)
            updated_df = pd.concat([existing_df, df], ignore_index=True)
        except FileNotFoundError:
            updated_df = df
            
        updated_df.to_csv(self.feedback_storage_path, index=False)
    
    def _trigger_learning(self):
        """触发基于反馈的学习"""
        print(f"触发学习：已收集 {len(self.feedback_history)} 条人类反馈")
        
        # 这里可以实现：
        # 1. 分析反馈模式，识别常见问题
        # 2. 调整代理的决策阈值
        # 3. 更新系统提示或工具描述
        # 4. 触发模型微调
        
        # 示例：调整工具使用频率
        tool_feedback_scores = {}
        for feedback in self.feedback_history:
            tool = feedback.tool_name
            if tool not in tool_feedback_scores:
                tool_feedback_scores[tool] = []
            tool_feedback_scores[tool].append(feedback.feedback_score)
        
        for tool, scores in tool_feedback_scores.items():
            avg_score = sum(scores) / len(scores)
            print(f"工具 {tool} 的平均反馈分数: {avg_score:.2f}")
            
            # 根据反馈调整工具使用策略
            if avg_score < 0.3:
                print(f"警告：工具 {tool} 的反馈分数较低，建议减少使用或改进")
```

## 常见问题

### Q1: 人类介入会影响系统的响应时间吗？
**A**: 是的，人类介入会引入延迟，但这是必要的权衡。设计时应考虑：
1. **异步处理**：允许代理在等待审批时处理其他任务
2. **超时机制**：设置合理的审批超时时间，超时后自动执行默认操作
3. **优先级队列**：为紧急任务设计快速审批通道
4. **批量审批**：支持一次性审批多个相关操作

### Q2: 如何平衡自动化程度和人工控制？
**A**: 推荐采用分级控制策略：
1. **全自动**：对于低风险、高频率的操作
2. **建议+确认**：AI提供建议，人类做最终决定
3. **人工审批**：高风险操作必须经过人工审批
4. **完全手动**：特别敏感或复杂的操作

可以通过置信度阈值、操作类型、用户角色等因素动态调整控制级别。

### Q3: 人类介入决策如何记录和审计？
**A**: LangChain提供了完整的审计追踪功能：
1. **决策日志**：记录所有人类介入决策，包括决策人、时间、原因
2. **上下文保存**：保存决策时的完整会话上下文
3. **版本控制**：支持决策的版本追踪和回滚
4. **导出功能**：可以将审计日志导出为标准格式（JSON、CSV等）

### Q4: 多个人类审批者如何协调？
**A**: 支持多种协作模式：
1. **顺序审批**：按设定顺序逐级审批
2. **并行审批**：多个审批者同时审批，达到阈值即通过
3. **投票机制**：多个审批者投票，多数决定
4. **专家路由**：根据问题类型自动路由给相应专家

### Q5: 如何处理人类决策不一致的情况？
**A**: 需要设计冲突解决机制：
1. **决策权重**：为不同角色的审批者设置不同权重
2. **冲突检测**：自动检测不一致的决策并触发协调流程
3. **升级机制**：当出现分歧时自动升级给更高权限者
