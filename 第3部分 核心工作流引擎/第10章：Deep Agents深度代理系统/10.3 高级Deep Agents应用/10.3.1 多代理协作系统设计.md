# 10.3.1 多代理协作系统设计

## 概念讲解

多代理协作系统是LangChain Deep Agents架构中的高级应用，它将多个独立的AI代理组织成协同工作的智能团队，共同完成复杂的业务目标。与传统单体代理相比，多代理协作系统通过分工、协调和集成，实现了"1+1>2"的系统效应，能够处理单个代理无法胜任的复杂任务。

在LangChain v1.2.22中，多代理协作系统基于以下核心设计理念构建：

1. **专业化分工**：每个代理专注于特定领域或任务类型，发挥各自的专业优势
2. **智能协调**：通过协调器或编排机制管理代理间的协作流程
3. **信息共享**：建立安全高效的信息交换机制，避免信息孤岛
4. **状态同步**：保持代理间状态的一致性，确保协作顺畅
5. **故障隔离**：单个代理的故障不会导致整个系统崩溃

多代理协作系统特别适用于以下场景：
- **复杂项目规划**：需要多个专业领域知识的项目
- **并行任务处理**：可以分解为独立子任务的复杂工作
- **决策支持系统**：需要多角度分析和验证的决策过程
- **业务流程自动化**：涉及多个步骤和角色的业务流程

## 核心要点

### 多代理协作的架构模式

LangChain v1.2.22支持多种多代理协作架构模式：

1. **中心化编排模式**：
   ```
   ┌─────────────────┐
   │     协调器       │
   │  (Orchestrator) │
   └────────┬────────┘
            │
    ┌───────┼───────┐
    ▼       ▼       ▼
  ┌─────┐ ┌─────┐ ┌─────┐
  │代理A│ │代理B│ │代理C│
  └─────┘ └─────┘ └─────┘
   ```

2. **去中心化对等模式**：
   ```
   ┌─────┐    ┌─────┐    ┌─────┐
   │代理A│◄──►│代理B│◄──►│代理C│
   └─────┘    └─────┘    └─────┘
      ▲          ▲          ▲
      │          │          │
      ▼          ▼          ▼
   ┌───────────────────────────┐
   │       共享状态存储         │
   └───────────────────────────┘
   ```

3. **分层协作模式**：
   ```
        ┌─────────────────┐
        │    管理代理      │
        └────────┬────────┘
                 │
        ┌────────▼────────┐
        │   协调层代理     │
        └─────┬─────┬─────┘
              │     │
        ┌─────▼┐ ┌─▼────┐
        │专业代理││专业代理│
        └──────┘ └──────┘
   ```

4. **流水线处理模式**：
   ```
   ┌─────┐    ┌─────┐    ┌─────┐    ┌─────┐
   │代理A│───►│代理B│───►│代理C│───►│代理D│
   └─────┘    └─────┘    └─────┘    └─────┘
    输入        处理        分析        输出
   ```

### 协作系统的关键组件

一个完整的多代理协作系统包含以下关键组件：

1. **代理注册表**：管理和跟踪所有参与协作的代理
2. **任务分配器**：根据代理能力和任务需求进行智能任务分配
3. **通信总线**：提供代理间的消息传递和数据交换能力
4. **状态管理器**：维护和同步代理间的共享状态
5. **协调引擎**：控制协作流程和处理异常情况
6. **结果聚合器**：整合各个代理的输出生成最终结果

### 协作协调策略

LangChain提供了多种协作协调策略：

1. **基于规则的协调**：使用预定义的业务规则决定协作流程
2. **基于目标的协调**：根据最终目标动态调整代理协作方式
3. **基于市场的协调**：代理通过"竞标"方式竞争任务
4. **基于协商的协调**：代理通过协商达成协作共识
5. **混合协调策略**：结合多种策略的优化方案

## 简单示例

### 基本使用：创建多代理协作系统

```python
from langchain.agents import create_agent
from langchain.multi_agent import MultiAgentOrchestrator
from langchain_openai import ChatOpenAI
from langchain.agents import Tool

# 定义专业代理工具
def market_research(topic: str) -> str:
    """市场调研工具"""
    return f"市场调研报告：{topic}领域的市场规模为...，主要竞争对手有..."

def financial_analysis(data: str) -> str:
    """财务分析工具"""
    return f"财务分析结果：{data}，预计投资回报率为..."

def technical_assessment(requirements: str) -> str:
    """技术评估工具"""
    return f"技术评估：{requirements}，推荐技术栈为..."

# 创建专业代理
market_agent = create_agent(
    model=ChatOpenAI(model="gpt-4"),
    tools=[Tool(name="research", func=market_research, description="市场调研")],
    system_prompt="你是市场分析专家，擅长市场调研和竞争分析。"
)

finance_agent = create_agent(
    model=ChatOpenAI(model="gpt-4"),
    tools=[Tool(name="analyze", func=financial_analysis, description="财务分析")],
    system_prompt="你是财务分析专家，擅长财务建模和投资回报分析。"
)

tech_agent = create_agent(
    model=ChatOpenAI(model="gpt-4"),
    tools=[Tool(name="assess", func=technical_assessment, description="技术评估")],

        "financial_analyst": finance_agent,
        "technical_architect": tech_agent

    coordination_strategy="goal_based",
    communication_mode="message_bus"

1. 支持多语言客服
2. 集成到现有网站和APP
3. 提供数据分析功能
4. 预算控制在10万元以内
5. 3个月内上线
"""

result = orchestrator.execute(
    task=project_requirements,
    max_iterations=10,
    collaboration_mode="parallel"

print(f"协作结果: {result['final_output'][:200]}...")
print(f"参与代理: {', '.join(result['participating_agents'])}")
print(f"执行时间: {result['execution_time']}秒")
```

### 基于LangGraph的协调器-工作器工作流

```python
from langgraph.graph import StateGraph, END
from typing import Dict, List, Any, TypedDict
from langchain.schema import SystemMessage, HumanMessage

# 定义状态结构
class ProjectState(TypedDict):
    """项目状态定义"""
    topic: str
    sections: List[Dict[str, str]]
    completed_sections: List[str]
    final_report: str
    current_agent: str

# 创建协调器代理
def create_planner_agent():
    """创建任务规划代理"""
    from langchain_openai import ChatOpenAI
    
    llm = ChatOpenAI(model="gpt-4", temperature=0.7)
    
    def planner(state: ProjectState) -> Dict[str, Any]:
        """任务规划函数"""
        # 生成报告章节规划
        system_msg = SystemMessage(
            content="你是一个项目规划专家，负责将复杂任务分解为可执行的子任务。"
        )
        human_msg = HumanMessage(
            content=f"请将以下主题分解为报告章节：{state['topic']}"
        )
        
        response = llm.invoke([system_msg, human_msg])
        
        # 解析响应，生成章节列表
        sections = [
            {"name": "引言", "description": "项目背景和目的"},
            {"name": "市场需求", "description": "目标市场和用户需求分析"},
            {"name": "技术方案", "description": "技术架构和实现方案"},
            {"name": "财务分析", "description": "成本估算和投资回报分析"},
            {"name": "实施计划", "description": "时间安排和资源分配"},
            {"name": "风险评估", "description": "可能的风险和应对措施"},
            {"name": "总结", "description": "项目总结和建议"}
        ]
        
        return {"sections": sections}
    
    return planner

# 创建工作器代理
def create_worker_agent(specialization: str):
    """创建工作器代理"""
    from langchain_openai import ChatOpenAI
    
    llm = ChatOpenAI(model="gpt-4", temperature=0.7)
    
    def worker(state: ProjectState) -> Dict[str, Any]:
        """工作器函数"""
        # 获取当前需要处理的章节
        current_section = None
        for section in state["sections"]:
            if section["name"] not in [s.get("name", "") for s in state.get("completed_sections", [])]:
                current_section = section
                break
        
        if not current_section:
            return {"completed_sections": state.get("completed_sections", [])}
        
        # 根据专业领域生成内容
        system_msg = SystemMessage(
            content=f"你是{specialization}专家，负责撰写报告的相关章节。"
        )
        human_msg = HumanMessage(
            content=f"请撰写关于'{current_section['name']}'的章节内容，主题是：{state['topic']}，章节描述：{current_section['description']}"
        )
        
        response = llm.invoke([system_msg, human_msg])
        
        # 更新完成章节
        completed = state.get("completed_sections", [])
        completed.append({
            "name": current_section["name"],
            "content": response.content,
            "worker": specialization
        })
        
        return {"completed_sections": completed}
    
    return worker

# 创建合成器代理
def create_synthesizer_agent():
    """创建报告合成代理"""
    from langchain_openai import ChatOpenAI
    
    llm = ChatOpenAI(model="gpt-4", temperature=0.7)
    
    def synthesizer(state: ProjectState) -> Dict[str, Any]:
        """合成器函数"""
        # 整合所有章节生成最终报告
        sections = state.get("completed_sections", [])
        
        if not sections:
            return {"final_report": "没有可合成的章节"}
        
        # 组织报告结构
        report_parts = [f"# {state['topic']}\n\n"]
        
        for section in sections:
            report_parts.append(f"## {section['name']}\n")
            report_parts.append(f"{section['content']}\n\n")
            report_parts.append(f"*撰写者：{section['worker']}*\n\n---\n\n")
        
        # 生成执行摘要
        system_msg = SystemMessage(
            content="你是报告编辑专家，负责生成专业的执行摘要。"
        )
        human_msg = HumanMessage(
            content=f"请为以下报告生成执行摘要：{''.join(report_parts)}"
        )
        
        summary_response = llm.invoke([system_msg, human_msg])
        
        # 插入执行摘要
        final_report = f"# 执行摘要\n\n{summary_response.content}\n\n" + "".join(report_parts)
        
        return {"final_report": final_report}
    
    return synthesizer

# 构建工作流图
def build_orchestrator_workflow():
    """构建协调器-工作器工作流"""
    
    # 创建状态图
    workflow = StateGraph(ProjectState)
    
    # 添加节点
    planner = create_planner_agent()
    market_worker = create_worker_agent("市场分析")
    tech_worker = create_worker_agent("技术架构")
    finance_worker = create_worker_agent("财务分析")
    risk_worker = create_worker_agent("风险管理")
    synthesizer = create_synthesizer_agent()
    
    workflow.add_node("planner", planner)
    workflow.add_node("market_worker", market_worker)
    workflow.add_node("tech_worker", tech_worker)
    workflow.add_node("finance_worker", finance_worker)
    workflow.add_node("risk_worker", risk_worker)
    workflow.add_node("synthesizer", synthesizer)
    
    # 添加边
    workflow.add_edge("planner", "market_worker")
    
    # 条件路由：根据章节类型选择工作器
    def route_to_worker(state: ProjectState) -> str:
        """路由到合适的工作器"""
        # 查找下一个需要处理的章节
        completed_names = [s.get("name", "") for s in state.get("completed_sections", [])]
        
        for section in state.get("sections", []):
            if section["name"] not in completed_names:
                # 根据章节名称路由到对应工作器
                if "市场" in section["name"] or "需求" in section["name"]:
                    return "market_worker"
                elif "技术" in section["name"] or "架构" in section["name"]:
                    return "tech_worker"
                elif "财务" in section["name"] or "成本" in section["name"]:
                    return "finance_worker"
                elif "风险" in section["name"]:
                    return "risk_worker"
                else:
                    return "market_worker"  # 默认路由
        
        # 所有章节完成，转到合成器
        return "synthesizer"
    
    workflow.add_conditional_edges(
        "market_worker",
        route_to_worker,
        {
            "market_worker": "market_worker",
            "tech_worker": "tech_worker",
            "finance_worker": "finance_worker",
            "risk_worker": "risk_worker",
            "synthesizer": "synthesizer"
        }
    )
    
    workflow.add_conditional_edges(
        "tech_worker",
        route_to_worker,
        {
            "market_worker": "market_worker",
            "tech_worker": "tech_worker",
            "finance_worker": "finance_worker",
            "risk_worker": "risk_worker",
            "synthesizer": "synthesizer"
        }
    )
    
    workflow.add_conditional_edges(
        "finance_worker",
        route_to_worker,
        {
            "market_worker": "market_worker",
            "tech_worker": "tech_worker",
            "finance_worker": "finance_worker",
            "risk_worker": "risk_worker",
            "synthesizer": "synthesizer"
        }
    )
    
    workflow.add_conditional_edges(
        "risk_worker",
        route_to_worker,
        {
            "market_worker": "market_worker",
            "tech_worker": "tech_worker",
            "finance_worker": "finance_worker",
            "risk_worker": "risk_worker",
            "synthesizer": "synthesizer"
        }
    )
    
    workflow.add_edge("synthesizer", END)
    
    # 设置入口点
    workflow.set_entry_point("planner")
    
    return workflow.compile()

# 执行工作流
workflow = build_orchestrator_workflow()

# 初始状态
initial_state = {
    "topic": "基于AI的中小企业数字化转型解决方案",
    "sections": [],
    "completed_sections": [],
    "final_report": "",
    "current_agent": ""
}

# 执行工作流
result = workflow.invoke(initial_state)

print(f"最终报告长度: {len(result['final_report'])} 字符")
print(f"生成的章节数: {len(result['completed_sections'])}")
print("\n报告预览:")
print(result['final_report'][:500] + "...")
```

### 代理间的通信与协调

```python
from typing import Dict, List, Any, Optional
from dataclasses import dataclass
from datetime import datetime
import asyncio

@dataclass
class AgentMessage:
    """代理消息定义"""
    sender: str
    recipient: str
    message_type: str  # request, response, notification, error
    content: Dict[str, Any]
    timestamp: datetime
    message_id: str
    requires_response: bool = False
    priority: str = "normal"  # low, normal, high, critical
    
class AgentCommunicationBus:
    """代理通信总线"""
    
    def __init__(self):
        self.message_queues: Dict[str, asyncio.Queue] = {}
        self.message_history: List[AgentMessage] = []
        self.registered_agents: Dict[str, Any] = {}
    
    def register_agent(self, agent_id: str, agent_instance: Any):
        """注册代理"""
        if agent_id not in self.message_queues:
            self.message_queues[agent_id] = asyncio.Queue()
        self.registered_agents[agent_id] = agent_instance
        print(f"代理 {agent_id} 已注册")
    
    async def send_message(self, message: AgentMessage) -> bool:
        """发送消息"""
        # 验证接收者
        if message.recipient not in self.message_queues:
            print(f"错误：接收者 {message.recipient} 未注册")
            return False
        
        # 记录消息历史
        self.message_history.append(message)
        
        # 放入接收者队列
        await self.message_queues[message.recipient].put(message)
        
        print(f"消息已发送：{message.sender} -> {message.recipient} [{message.message_type}]")
        return True
    
    async def broadcast_message(self, sender: str, message_type: str, content: Dict[str, Any], exclude: List[str] = None):
        """广播消息"""
        exclude = exclude or []
        
        for agent_id in self.registered_agents:
            if agent_id != sender and agent_id not in exclude:
                message = AgentMessage(
                    sender=sender,
                    recipient=agent_id,
                    message_type=message_type,
                    content=content,
                    timestamp=datetime.now(),
                    message_id=f"broadcast_{datetime.now().timestamp()}",
                    requires_response=False
                )
                await self.send_message(message)
    
    async def receive_messages(self, agent_id: str, timeout: float = 1.0) -> List[AgentMessage]:
        """接收消息"""
        messages = []
        
        try:
            # 非阻塞获取消息
            while True:
                message = await asyncio.wait_for(
                    self.message_queues[agent_id].get(),
                    timeout=timeout
                )
                messages.append(message)
                print(f"代理 {agent_id} 收到消息：{message.sender} -> {agent_id}")
        except asyncio.TimeoutError:
            # 超时，返回已收集的消息
            pass
        
        return messages
    
    def get_communication_stats(self) -> Dict[str, Any]:
        """获取通信统计"""
        return {
            "total_messages": len(self.message_history),
            "registered_agents": list(self.registered_agents.keys()),
            "message_types": self._count_message_types(),
            "busy_queues": self._identify_busy_queues()
        }
    
    def _count_message_types(self) -> Dict[str, int]:
        """统计消息类型"""
        counts = {}
        for msg in self.message_history:
            counts[msg.message_type] = counts.get(msg.message_type, 0) + 1
        return counts

# 使用通信总线的协作代理示例
class CollaborativeAgent:
    """协作代理基类"""
    
    def __init__(self, agent_id: str, communication_bus: AgentCommunicationBus):
        self.agent_id = agent_id
        self.communication_bus = communication_bus
        self.knowledge_base: Dict[str, Any] = {}
        self.task_queue: List[Dict[str, Any]] = []
        
    async def process_task(self, task: Dict[str, Any]):
        """处理任务"""
        print(f"代理 {self.agent_id} 开始处理任务：{task.get('description', '未知任务')}")
        
        # 处理任务...
        await asyncio.sleep(1)  # 模拟处理时间
        
        result = {
            "task_id": task.get("task_id"),
            "agent_id": self.agent_id,
            "result": f"任务完成：{task.get('description')}",
            "timestamp": datetime.now().isoformat()
        }
        
        # 发送完成通知
        if task.get("requires_coordination", False):
            await self._notify_coordinator(result)
        
        return result
    
    async def _notify_coordinator(self, result: Dict[str, Any]):
        """通知协调器"""
        message = AgentMessage(
            sender=self.agent_id,
            recipient="coordinator",
            message_type="task_completed",
            content=result,
            timestamp=datetime.now(),
            message_id=f"complete_{datetime.now().timestamp()}",
            requires_response=True
        )
        
        await self.communication_bus.send_message(message)

# 多代理协作系统示例
async def multi_agent_collaboration_example():
    """多代理协作示例"""
    
    # 创建通信总线
    comm_bus = AgentCommunicationBus()
    
    # 创建专业代理
    agents = {
        "data_collector": CollaborativeAgent("data_collector", comm_bus),
        "data_analyzer": CollaborativeAgent("data_analyzer", comm_bus),
        "report_writer": CollaborativeAgent("report_writer", comm_bus),
        "quality_checker": CollaborativeAgent("quality_checker", comm_bus),
        "coordinator": CollaborativeAgent("coordinator", comm_bus)
    }
    
    # 注册所有代理
    for agent_id, agent in agents.items():
        comm_bus.register_agent(agent_id, agent)
    
    # 定义协作任务
    collaboration_task = {
        "project_id": "project_001",
        "description": "生成季度市场分析报告",
        "subtasks": [
            {"task_id": "task_001", "description": "收集市场数据", "assigned_to": "data_collector"},

    }
    
    print(f"开始协作任务：{collaboration_task['description']}")
    
    # 协调器广播任务
    await comm_bus.broadcast_message(
        sender="coordinator",
        message_type="project_announcement",
        content=collaboration_task,
        exclude=["coordinator"]
    )
    
    # 模拟任务执行（简化版）
    tasks = []
    for agent_id, agent in agents.items():
        if agent_id != "coordinator":
            # 为每个代理分配任务
            task_for_agent = {
                "task_id": f"{agent_id}_task",
                "description": f"{agent_id}的专属任务",
                "requires_coordination": True
            }
            tasks.append(agent.process_task(task_for_agent))
    
    # 并行执行任务
    results = await asyncio.gather(*tasks)
    
    print(f"\n任务执行完成，共完成 {len(results)} 个子任务")
    
    # 显示通信统计
    stats = comm_bus.get_communication_stats()
    print(f"\n通信统计：")
    print(f"  总消息数：{stats['total_messages']}")
    print(f"  注册代理数：{len(stats['registered_agents'])}")
    print(f"  消息类型分布：{stats['message_types']}")

# 运行示例
if __name__ == "__main__":
    asyncio.run(multi_agent_collaboration_example())
```

```python
from enum import Enum
from typing import Dict, List, Any, Optional
from dataclasses import dataclass
from datetime import datetime

class AgentRole(Enum):
    """代理角色枚举"""
    STRATEGIST = "strategist"      # 战略规划
    RESEARCHER = "researcher"      # 研究分析
    ANALYST = "analyst"          # 数据分析
    CREATOR = "creator"          # 内容创作
    REVIEWER = "reviewer"        # 评审验证
    COORDINATOR = "coordinator"  # 协调管理

@dataclass
class AgentCapability:
    """代理能力定义"""
    role: AgentRole
    skills: List[str]
    expertise_level: int  # 1-10
    available: bool = True
    current_workload: int = 0

class RoleBasedMultiAgentSystem:
    """基于角色的多代理系统"""
    
    def __init__(self):
        self.agents: Dict[str, Dict[str, Any]] = {}
        self.role_capabilities: Dict[AgentRole, List[str]] = {
            AgentRole.STRATEGIST: ["planning", "decision_making", "goal_setting"],
            AgentRole.RESEARCHER: ["information_gathering", "literature_review", "data_collection"],
            AgentRole.ANALYST: ["data_analysis", "statistical_modeling", "insight_generation"],
            AgentRole.CREATOR: ["content_writing", "design", "prototyping"],
            AgentRole.REVIEWER: ["quality_assurance", "validation", "feedback_provision"],
            AgentRole.COORDINATOR: ["task_assignment", "progress_tracking", "conflict_resolution"]
        }
        
        self.workflow_templates = {
            "research_project": [
                {"phase": "planning", "roles": [AgentRole.STRATEGIST, AgentRole.COORDINATOR]},
                {"phase": "research", "roles": [AgentRole.RESEARCHER, AgentRole.ANALYST]},
                {"phase": "creation", "roles": [AgentRole.CREATOR]},
                {"phase": "review", "roles": [AgentRole.REVIEWER, AgentRole.ANALYST]},
                {"phase": "finalization", "roles": [AgentRole.COORDINATOR, AgentRole.STRATEGIST]}
            ],
            "development_project": [
                {"phase": "requirements", "roles": [AgentRole.STRATEGIST, AgentRole.RESEARCHER]},
                {"phase": "design", "roles": [AgentRole.CREATOR, AgentRole.ANALYST]},
                {"phase": "implementation", "roles": [AgentRole.CREATOR]},
                {"phase": "testing", "roles": [AgentRole.REVIEWER, AgentRole.ANALYST]},
                {"phase": "deployment", "roles": [AgentRole.COORDINATOR]}
            ]
        }
    
    def register_agent(self, agent_id: str, capabilities: List[AgentCapability]):
        """注册代理"""
        self.agents[agent_id] = {
            "capabilities": capabilities,
            "assigned_tasks": [],
            "performance_metrics": {},
            "availability": True
        }
        print(f"代理 {agent_id} 已注册，能力：{[c.role.value for c in capabilities]}")
    
    def assign_task(self, project_type: str, project_requirements: Dict[str, Any]) -> Dict[str, Any]:
        """分配任务"""
        if project_type not in self.workflow_templates:
            raise ValueError(f"未知的项目类型：{project_type}")
        
        workflow = self.workflow_templates[project_type]
        task_assignments = {}
        
        for phase in workflow:
            phase_name = phase["phase"]
            required_roles = phase["roles"]
            
            print(f"\n阶段：{phase_name}")
            print(f"需要的角色：{[r.value for r in required_roles]}")
            
            phase_assignments = []
            for role in required_roles:
                # 选择最合适的代理
                selected_agent = self._select_agent_for_role(role, project_requirements)
                
                if selected_agent:
                    task_description = self._generate_task_description(role, phase_name, project_requirements)
                    
                    assignment = {
                        "agent_id": selected_agent,
                        "role": role.value,
                        "task": task_description,
                        "phase": phase_name,
                        "estimated_duration": self._estimate_task_duration(role, task_description)
                    }
                    
                    phase_assignments.append(assignment)
                    
                    # 更新代理工作负载
                    self.agents[selected_agent]["assigned_tasks"].append(assignment)
                    
                    print(f"  分配：{selected_agent} -> {role.value}")
                else:
                    print(f"  警告：没有找到合适的 {role.value} 代理")
            
            task_assignments[phase_name] = phase_assignments
        
        return task_assignments
    
    def _select_agent_for_role(self, role: AgentRole, requirements: Dict[str, Any]) -> Optional[str]:
        """为角色选择代理"""
        suitable_agents = []
        
        for agent_id, agent_info in self.agents.items():
            if not agent_info["availability"]:
                continue
            
            # 检查代理是否有此角色能力
            for capability in agent_info["capabilities"]:
                if capability.role == role and capability.available:
                    # 计算匹配分数
                    match_score = self._calculate_role_match_score(capability, role, requirements)
                    
                    suitable_agents.append({
                        "agent_id": agent_id,
                        "capability": capability,
                        "match_score": match_score,
                        "current_workload": capability.current_workload
                    })
        
        if not suitable_agents:
            return None
        
        # 选择最佳代理（考虑匹配分数和工作负载）
        suitable_agents.sort(key=lambda x: (
            -x["match_score"],  # 匹配分数越高越好
            x["current_workload"]  # 工作负载越少越好
        ))
        
        return suitable_agents[0]["agent_id"]
    
    def _calculate_role_match_score(self, capability: AgentCapability, required_role: AgentRole, requirements: Dict[str, Any]) -> float:
        """计算角色匹配分数"""
        base_score = capability.expertise_level / 10.0
        
        # 检查技能匹配
        required_skills = self.role_capabilities.get(required_role, [])
        skill_overlap = len(set(capability.skills) & set(required_skills))
        skill_score = skill_overlap / max(len(required_skills), 1)
        
        # 项目特定要求
        project_complexity = requirements.get("complexity", "medium")
        if project_complexity == "high" and capability.expertise_level >= 8:
            complexity_bonus = 0.2
        elif project_complexity == "medium" and capability.expertise_level >= 5:
            complexity_bonus = 0.1
        else:
            complexity_bonus = 0.0
        
        total_score = base_score * 0.6 + skill_score * 0.4 + complexity_bonus
        
        return min(total_score, 1.0)
    
    def monitor_collaboration_progress(self, task_assignments: Dict[str, Any]) -> Dict[str, Any]:
        """监控协作进度"""
        progress_report = {
            "total_phases": len(task_assignments),
            "completed_phases": 0,
            "active_agents": 0,
            "blocked_tasks": [],
            "performance_metrics": {}
        }
        
        for phase_name, assignments in task_assignments.items():
            phase_complete = True
            
            for assignment in assignments:
                agent_id = assignment["agent_id"]
                
                # 检查代理状态
                if agent_id in self.agents:
                    agent_tasks = self.agents[agent_id]["assigned_tasks"]
                    
                    # 检查任务完成情况
                    task_complete = any(
                        t.get("phase") == phase_name and 
                        t.get("status") == "completed"
                        for t in agent_tasks
                    )
                    
                    if not task_complete:
                        phase_complete = False
                        
                        # 检查是否阻塞
                        if assignment.get("blocked_by"):
                            progress_report["blocked_tasks"].append({
                                "phase": phase_name,
                                "agent": agent_id,
                                "task": assignment["task"],
                                "blocked_by": assignment["blocked_by"]
                            })
            
            if phase_complete:
                progress_report["completed_phases"] += 1
        
        # 统计活跃代理
        progress_report["active_agents"] = sum(
            1 for agent_info in self.agents.values() 
            if agent_info["assigned_tasks"] and not all(
                t.get("status") == "completed" 
                for t in agent_info["assigned_tasks"]
            )
        )
        
        return progress_report
```

## 常见问题

### Q1: 如何确定需要多少个代理？
**A**: 代理数量的确定需要考虑多个因素：
1. **任务复杂度**：复杂任务需要更多专业代理
2. **并行性需求**：可并行处理的子任务越多，需要的代理越多
3. **资源限制**：计算资源、API调用限制等
4. **协调开销**：更多代理意味着更高的协调成本

**经验法则**：
- **简单任务**：1-2个代理
- **中等任务**：3-5个代理
- **复杂任务**：5-10个代理
- **超复杂任务**：10+个代理，需要分层管理

### Q2: 代理间通信的最佳实践是什么？
**A**: 代理间通信需要平衡效率和可靠性：
1. **消息格式标准化**：使用统一的消息格式和协议
2. **异步通信**：避免阻塞，提高系统响应性
3. **消息队列**：使用消息队列处理高并发通信
4. **超时和重试**：设置合理的超时和重试机制
5. **监控和日志**：完整的通信监控和日志记录

### Q3: 如何处理代理间的冲突？
**A**: 冲突处理策略包括：
1. **预防策略**：通过清晰的职责划分和规则避免冲突
2. **检测机制**：实时监控代理行为和状态，及时发现冲突
3. **解决策略**：
   - **协商解决**：代理通过协商达成共识
   - **权威决策**：由协调器或权威代理做最终决定
   - **投票机制**：多个代理投票决定
   - **优先级规则**：基于优先级解决冲突

### Q4: 多代理系统的性能瓶颈在哪里？
**A**: 常见的性能瓶颈包括：
1. **协调开销**：协调器可能成为瓶颈
2. **通信延迟**：代理间通信的网络延迟
3. **资源竞争**：代理竞争共享资源（如GPU、API配额）
4. **状态同步**：大量代理的状态同步开销
5. **任务分配**：复杂的任务分配算法可能成为瓶颈

**优化策略**：
- 使用分布式协调器
- 优化通信协议和压缩算法
- 实施资源配额和优先级调度
- 采用增量状态同步
- 简化任务分配算法

### Q5: 如何确保多代理系统的安全性？
**A**: 安全考虑包括：
1. **身份验证**：确保只有授权的代理可以参与协作
2. **权限控制**：每个代理只能访问授权的资源和工具
3. **通信加密**：代理间通信使用加密传输
4. **输入验证**：对所有输入进行验证和清洗
5. **审计日志**：完整记录所有操作便于审计
