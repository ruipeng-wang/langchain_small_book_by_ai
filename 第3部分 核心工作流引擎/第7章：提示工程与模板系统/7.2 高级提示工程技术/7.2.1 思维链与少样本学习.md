# 7.2.1 思维链与少样本学习

## 概念讲解

### 思维链（Chain-of-Thought）

思维链（CoT）是一种提示技术，通过引导模型"一步一步思考"，将复杂问题拆解为中间推理步骤。研究表明，CoT能显著提升模型在数学、逻辑推理等任务上的表现。

**核心原理**：让模型展示推理过程，而非直接给出答案。这相当于让模型"大声思考"，暴露出中间步骤，从而提高准确性。

### 少样本学习（Few-Shot Learning）

少样本学习指在提示中提供少量示例，让模型学习任务的模式和格式。相比零样本（Zero-Shot），少样本能让模型更好地理解预期输出的风格和结构。

## 核心要点

1. **CoT提示**：使用"Let's think step by step"或"请一步一步分析"引导推理
2. **Few-Shot示例**：提供2-5个输入输出对作为参考
3. **FewShotPromptTemplate**：LangChain提供的少样本模板类
4. **组合使用**：CoT + Few-Shot效果最佳

## 简单示例

### 思维链提示

```python
from langchain_openai import ChatOpenAI

model = ChatOpenAI(model="gpt-4o-mini")

# 零样本（直接回答）
response1 = model.invoke("鸡和兔共有10个头，32只脚，各有几只？")

# 思维链（引导推理）
response2 = model.invoke(
    "鸡和兔共有10个头，32只脚，各有几只？"
    "请一步一步分析。"
)
```

### 少样本学习

```python
from langchain_core.prompts import FewShotPromptTemplate, PromptTemplate

# 定义示例
examples = [
    {"input": "2+2=?", "answer": "2+2=4"},
    {"input": "10-3=?", "answer": "10-3=7"},
    {"input": "3*4=?", "answer": "3*4=12"},
]

# 创建示例模板
example_template = PromptTemplate.from_template(
    "问题：{input}\n答案：{answer}"
)

# 创建Few-Shot提示
few_shot_prompt = FewShotPromptTemplate(
    examples=examples,
    example_prompt=example_template,
    suffix="问题：{input}\n答案：",
    input_variables=["input"],
)

# 使用
prompt = few_shot_prompt.format(input="5*6=?")
```

### CoT + Few-Shot 组合

```python
cot_examples = [
    {"q": "小明有5个苹果，吃了2个，还剩几个？",
     "a": "小明原来有5个苹果。吃了2个后，剩下5-2=3个。所以还剩3个苹果。"},
    {"q": "一本书原价30元，打8折后多少钱？",
     "a": "原价30元，打8折就是30×0.8=24元。所以打折后是24元。"},
]
```

## 常见问题

**Q: 少样本需要多少个示例？**

A: 通常2-5个即可。过多会占用context window，且边际效益递减。

**Q: CoT对所有任务都有效吗？**

A: 对需要推理的任务（数学、逻辑）效果显著。对简单问答或创意写作效果不明显。

## 本节总结

思维链和少样本学习是提升模型推理能力的两大核心技术。通过引导模型逐步推理，并提供参考示例，可以显著提高复杂任务的准确率。
