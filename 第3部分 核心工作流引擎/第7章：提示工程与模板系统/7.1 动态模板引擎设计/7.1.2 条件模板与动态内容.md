# 7.1.2 条件模板与动态内容

## 概念讲解

### 什么是条件模板

条件模板是指根据运行时条件动态生成不同模板内容的技术。在实际应用中，同一个功能在不同场景下可能需要不同的提示词：例如针对初学者和专家的解释深度不同，或者根据用户语言偏好切换中英文输出。

### 动态内容的类型

| 类型 | 说明 | 示例 |
|------|------|------|
| **变量插值** | 根据输入变量动态替换内容 | `"{topic}"` |
| **条件渲染** | 根据条件选择不同模板分支 | 如果用户是新手，使用简单版提示 |
| **动态列表** | 根据运行时数据动态生成内容列表 | 对话历史、上下文片段 |
| **部分渲染** | 某些变量预先填充，运行时补充 | 系统提示固定，用户输入动态 |

## 核心要点

1. **条件选择**：根据运行时数据选择不同模板
2. **MessagesPlaceholder**：动态插入可变长度的消息列表
3. **模板组合**：多个模板按需组合使用
4. **RunnableWithMessageHistory**：自动管理对话历史

## 简单示例

### 条件模板：根据用户水平选择提示

```python
from langchain_core.prompts import PromptTemplate

# 不同水平的模板
beginner_template = PromptTemplate.from_template(
    "你是一位耐心的老师。请用简单易懂的语言解释{topic}，"
    "避免使用专业术语，多用比喻。"
)

expert_template = PromptTemplate.from_template(
    "你是一位资深专家。请从技术角度深入分析{topic}，"
    "可以引用相关理论和最佳实践。"
)

def get_prompt(user_level: str, topic: str) -> str:
    """根据用户水平选择模板"""
    template = beginner_template if user_level == "beginner" else expert_template
    return template.format(topic=topic)

# 使用
prompt1 = get_prompt("beginner", "机器学习")
prompt2 = get_prompt("expert", "机器学习")
```

### 动态插入上下文

```python
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder

# 支持上下文的模板
prompt = ChatPromptTemplate.from_messages([
    ("system", "你是一个知识助手，基于以下上下文回答问题。"),
    ("system", "相关上下文：\n{context}"),
    MessagesPlaceholder("history"),
    ("human", "{question}"),
])

# 使用时动态提供上下文和对话历史
messages = prompt.format_messages(
    context="机器学习是AI的一个分支，主要包括监督学习、无监督学习和强化学习。",
    history=[("human", "什么是AI？"), ("ai", "AI是人工智能...")],
    question="机器学习有哪些类型？"
)
```

## 进阶应用

### 模板工厂：动态生成模板

```python
from langchain_core.prompts import PromptTemplate

def create_translation_template(
    source_lang: str = "English",
    target_lang: str = "Chinese",
    style: str = "formal"
) -> PromptTemplate:
    """根据参数动态生成翻译模板"""
    
    templates = {
        ("English", "Chinese", "formal"): PromptTemplate.from_template(
            "请将以下英文正式翻译成中文：\n{text}"
        ),
        ("English", "Chinese", "casual"): PromptTemplate.from_template(
            "请将以下英文用通俗的中文表达：\n{text}"
        ),
        ("Chinese", "English", "formal"): PromptTemplate.from_template(
            "Please translate the following Chinese into formal English:\n{text}"
        ),
    }
    
    key = (source_lang, target_lang, style)
    return templates.get(key, PromptTemplate.from_template(
        "请将以下{text}翻译：\n{content}"
    ))

# 动态生成
template = create_translation_template(style="casual")
```

### 结合Runnable实现条件逻辑

```python
from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI

# 基础模板
base_prompt = ChatPromptTemplate.from_messages([
    ("system", "你是一个助手。"),
    ("human", "{input}"),
])

# 带上下文的模板
context_prompt = ChatPromptTemplate.from_messages([
    ("system", "你是一个知识助手，基于以下上下文回答：\n{context}"),
    ("human", "{input}"),
])

def create_chain(has_context: bool):
    """根据是否有上下文选择不同链"""
    prompt = context_prompt if has_context else base_prompt
    model = ChatOpenAI(model="gpt-4o-mini")
    return prompt | model

# 使用
simple_chain = create_chain(has_context=False)
context_chain = create_chain(has_context=True)
```

## 常见问题

**Q: 如何管理大量条件模板？**

A: 建议将模板存储在配置文件中（YAML/JSON），按场景分类，运行时按需加载。

**Q: MessagesPlaceholder和直接插入消息有什么区别？**

A: MessagesPlaceholder是一个占位符，在模板解析时才填充，支持链式调用中自动传递。直接插入则是一次性生成。

## 本节总结

条件模板和动态内容让提示词具备了适应不同场景的能力。通过条件选择、动态插入和模板组合，可以构建灵活、可扩展的提示系统。
