# 7.1.1 变量绑定与模板渲染

## 概念讲解

### 为什么需要提示模板

在AI应用开发中，提示词（Prompt）是连接用户意图与模型能力的桥梁。然而，直接在代码中硬编码提示词会带来严重的维护问题：

1. **修改困难**：每次调整提示词需要全局搜索替换
2. **重复编写**：相似提示词在多处重复出现
3. **无法适配动态场景**：静态内容无法根据输入变化

**PromptTemplate**通过将提示词的静态文本与动态变量分离，解决了这些问题。

### PromptTemplate的设计哲学

```
最终提示 = 模板文本 + 变量绑定 + 格式渲染
```

这种分离带来了三个关键优势：

- **模板即配置**：提示词变成可配置的参数
- **变量即接口**：输入变量定义了调用契约
- **格式即输出**：支持预填充和默认值

## 核心要点

1. **自动变量提取**：LangChain自动从`{变量名}`中识别输入变量
2. **partial_variables**：支持预填充默认值
3. **ChatPromptTemplate**：适用于对话模型的消息模板
4. **MessagesPlaceholder**：动态插入历史消息

## 简单示例

### 基础模板

```python
from langchain_core.prompts import PromptTemplate

# 创建模板（自动识别变量）
template = PromptTemplate.from_template(
    "请用简短的语言解释什么是{topic}。"
)

# 渲染模板
prompt = template.format(topic="机器学习")
print(prompt)  # 请用简短的语言解释什么是机器学习。
```

### 多变量模板

```python
template = PromptTemplate.from_template(
    "你是一位{role}。请为{audience}写一篇关于{topic}的{length}字介绍。"
)

prompt = template.format(
    role="技术专家",
    audience="初学者",
    topic="人工智能",
    length="200"
)
```

### 带默认值的模板

```python
# 使用partial预填充默认值
template = PromptTemplate.from_template(
    "请用{language}语言回答{question}。"
).partial(language="中文")

# 只需提供question即可
prompt = template.format(question="什么是深度学习？")
```

### ChatPromptTemplate：对话模板

```python
from langchain_core.prompts import ChatPromptTemplate

# 创建对话模板
prompt = ChatPromptTemplate.from_messages([
    ("system", "你是一个专业的{role}，回答要准确简洁。"),
    ("human", "{question}"),
])

messages = prompt.format_messages(
    role="编程助手",
    question="Python和Java有什么区别？"
)
# 返回: [SystemMessage, HumanMessage]
```

### MessagesPlaceholder：动态插入历史

```python
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder

prompt = ChatPromptTemplate.from_messages([
    ("system", "你是一个友好的助手。"),
    MessagesPlaceholder("history"),  # 动态插入对话历史
    ("human", "{input}"),
])

# 使用时传入history
result = prompt.format_messages(
    history=[("human", "你好"), ("ai", "你好！有什么可以帮助你的？")],
    input="今天天气怎么样？"
)
```

## 进阶应用

### 模板与模型链式组合

```python
from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI

prompt = ChatPromptTemplate.from_template("解释{topic}")
model = ChatOpenAI(model="gpt-4o-mini")

# 直接链式调用
chain = prompt | model
response = chain.invoke({"topic": "量子计算"})
```

### 从JSON加载模板

```python
import json

# 模板可以存储在配置文件中
template_config = {
    "template": "你是一个{role}。请回答：{question}",
    "input_variables": ["role", "question"]
}

template = PromptTemplate(**template_config)
```

## 常见问题

**Q: 变量名可以用中文吗？**

A: 可以，但推荐使用英文变量名，避免编码问题。

**Q: partial_variables和默认值有什么区别？**

A: `partial`返回的是一个部分绑定的新模板对象，原模板不变。它更适合链式调用场景。

**Q: ChatPromptTemplate和PromptTemplate如何选择？**

A: 使用LLM（纯文本模型）时用PromptTemplate；使用ChatModel（对话模型）时用ChatPromptTemplate。

## 本节总结

PromptTemplate是LangChain提示工程的基础。通过变量绑定和模板渲染，开发者可以将提示词从硬编码中解放出来，实现可维护、可复用、可测试的提示词管理。
