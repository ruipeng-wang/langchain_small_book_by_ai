### 解析器抽象化的历史演进：从硬编码模式到统一接口

在大语言模型应用开发的发展历程中，自定义解析器的开发经历了从**手工艺时代**到**工业化时代**的深刻转变。这种转变不仅体现在技术实现上，更体现在设计哲学、工程实践和组织协同的多个层面。

**解析器开发的三个阶段演进：**

1. **手工艺阶段（2010s-2018）**：每个解析器都是独立艺术品
   - 特征：完全硬编码，正则表达式驱动，零复用
   - 局限：维护成本高，难以扩展，缺乏标准化
   - 典型：财务报告解析器、日志文件解析器、HTML爬虫

2. **组件化阶段（2018-2021）**：模块化设计初见端倪  
   - 特征：基础组件库，有限复用，初步接口规范
   - 突破：代码复用率提升，维护成本下降
   - 局限：接口不一致，集成复杂度依然较高

3. **框架化阶段（2021-至今）**：LangChain带来的革命性变革
   - 特征：统一接口标准，类型安全体系，完整生态集成
   - 创新：抽象基类，泛型支持，标准化错误处理
   - 价值：开发效率提升10倍以上，维护成本降低80%

#### BaseOutputParser的设计哲学

5. **计费计量**：使用量和性能指标收集

**微服务架构集成**：
```python
# 解析器微服务架构示意
class ParserService:
    """解析器微服务"""
    
    def __init__(self):
        self.parser_registry = ParserRegistry()
        self.load_balancer = ParserLoadBalancer()
        self.monitor = ParserMonitor()
    
    async def parse(self, request: ParseRequest) -> ParseResponse:
        """处理解析请求"""
        
        # 1. 服务发现：选择最合适的解析器
        parser = self.load_balancer.select_parser(
            request.parser_type, 
            request.text_type
        )
        
        # 2. 健康检查：确保解析器可用
        if not parser.is_healthy():
            parser = self.load_balancer.fallback_parser()
        
        # 3. 执行解析：带超时和重试
        result = await self._execute_with_retry(
            parser.parse, 
            request.text,
            max_retries=3,
            timeout_seconds=30
        )
        
        # 4. 记录指标：用于计费和监控
        self.monitor.record_usage(
            parser_id=parser.id,
            text_length=len(request.text),
            duration_ms=result.duration_ms,
            success=True
        )
        
        return ParseResponse(
            success=True,
            result=result.data,
            parser_used=parser.id,
            confidence=result.confidence
        )
```

```python
class HybridParser(BaseOutputParser[T]):
    """混合解析器：结合规则和AI"""
    
    def __init__(self, llm: BaseLanguageModel = None):
        super().__init__()
        self.llm = llm
        self.rule_engine = RuleEngine()
        self.ai_engine = AIEngine(llm) if llm else None
    
    def parse(self, text: str) -> T:
        # 第一阶段：规则引擎解析
        rule_result = self.rule_engine.parse(text)
        
        if rule_result.confidence > 0.9:
            # 高置信度规则解析，直接返回
            return rule_result.data
        
        # 第二阶段：AI增强解析
        if self.ai_engine:
            ai_result = self.ai_engine.parse(text, rule_result)
            
            # 第三阶段：结果融合
            fused_result = self._fuse_results(rule_result, ai_result)
            return fused_result.data
        
        # 回退到规则解析
        return rule_result.data
    
    def _fuse_results(self, rule_result, ai_result) -> ParseResult:
        """融合规则和AI解析结果"""
        # 基于置信度的加权融合
        total_confidence = rule_result.confidence + ai_result.confidence
        
        if total_confidence == 0:
            return ParseResult(confidence=0, data=None)
        
        # 加权平均融合
        weight_rule = rule_result.confidence / total_confidence
        weight_ai = ai_result.confidence / total_confidence
        
        # 融合逻辑（根据具体类型实现）
        fused_data = self._fuse_data(rule_result.data, ai_result.data, weight_rule, weight_ai)
        fused_confidence = max(rule_result.confidence, ai_result.confidence)
        
        return ParseResult(confidence=fused_confidence, data=fused_data)
```

#### 创新三：流式解析与实时处理

传统解析器通常需要完整数据才能开始处理，而LangChain支持**流式解析**，实现了实时数据处理：

**流式解析的技术优势**：
1. **低延迟处理**：边接收边解析，无需等待完整数据
2. **内存效率**：处理大文件时内存占用恒定
3. **实时反馈**：逐步输出解析结果，提供即时反馈
4. **容错性**：部分数据损坏不影响整体处理
5. **可中断性**：支持处理过程中的优雅中断

**流式解析的实现模式**：
```python
class StreamingParser(BaseOutputParser[T]):
    """流式解析器基类"""
    
    def parse_stream(self, text_stream: Iterator[str]) -> Iterator[ParseChunk]:
        """流式解析"""
        
        buffer = ""
        for chunk in text_stream:
            buffer += chunk
            
            # 尝试从缓冲区解析完整记录
            while self._has_complete_record(buffer):
                record, remaining = self._extract_record(buffer)
                buffer = remaining
                
                # 解析记录
                parsed_chunk = self._parse_record(record)
                yield parsed_chunk
        
        # 处理缓冲区剩余内容
        if buffer.strip():
            final_chunk = self._parse_record(buffer)
            yield final_chunk
    
    async def aparse_stream(self, text_stream: AsyncIterator[str]) -> AsyncIterator[ParseChunk]:
        """异步流式解析"""
        
        buffer = ""
        async for chunk in text_stream:
            buffer += chunk
            
            # 异步处理缓冲区
            while self._has_complete_record(buffer):
                record, remaining = self._extract_record(buffer)
                buffer = remaining
                
                # 异步解析记录
                parsed_chunk = await self._aparse_record(record)
                yield parsed_chunk
        
        # 处理最终缓冲区
        if buffer.strip():
            final_chunk = await self._aparse_record(buffer)
            yield final_chunk
```

## 进阶应用：企业级解析器架构设计

### 企业级解析器系统的架构模式

#### 模式一：分层解析架构

**分层解析的设计理念**：
```
原始文本 → 预处理层 → 解析核心层 → 验证层 → 后处理层 → 结构化输出
    │         │           │          │          │           │
    ├── 多种格式 ├── 清洗转换 ├── 规则/AI ├── 业务规则 ├── 格式转换 ├── 标准接口
    ├── 混合内容 ├── 编码处理 ├── 混合策略 ├── 质量检查 ├── 富化增强 ├── 类型安全
    └── 实时流式 └── 分块处理 └── 增量解析 └── 置信评估 └── 缓存优化 └── 监控集成
```

**分层实现的技术要点**：
1. **预处理层的职责**：
   - 文本清洗：移除无关字符，标准化格式
   - 编码处理：统一字符编码，处理特殊字符
   - 分块策略：将大文本分割为可管理的块
   - 格式检测：识别输入数据的格式类型

2. **解析核心层的策略**：
   - 规则引擎：基于正则表达式和语法规则的解析
   - 机器学习：基于训练模型的智能解析
   - 大语言模型：基于提示工程的语义解析
   - 混合策略：多策略的加权融合和仲裁

3. **验证层的质量保障**：
   - 语法验证：检查解析结果的语法正确性
   - 语义验证：验证解析结果的语义合理性
   - 业务规则验证：应用领域特定的业务约束
   - 一致性验证：检查不同部分间的一致性

4. **后处理层的价值增强**：
   - 格式转换：转换为目标系统需要的格式
   - 数据富化：补充缺失信息，增强数据价值
   - 质量控制：评估和报告数据质量指标
   - 缓存管理：优化性能的缓存策略

```python
    """解析器微服务"""
    
    def __init__(self):
        self.service_registry = ServiceRegistry()
        self.load_balancer = LoadBalancer()
        self.circuit_breaker = CircuitBreaker()
        self.metrics = MetricsCollector()
    
    async def parse_request(self, request: ParseRequest) -> ParseResponse:
        """处理解析请求"""
        
        # 1. 服务发现：查找合适的解析器
        parser_endpoint = self.service_registry.discover(
            request.parser_type,
            request.quality_requirements
        )
        
        # 2. 熔断器检查：避免调用故障服务
        if self.circuit_breaker.is_open(parser_endpoint):
            # 快速失败或使用降级服务
            return await self._fallback_parse(request)
        
        # 3. 负载均衡：选择健康的实例
        instance = self.load_balancer.select_instance(parser_endpoint)
        
        # 4. 调用解析器服务
        try:
            start_time = time.time()
            
            # 使用熔断器包装调用
            result = await self.circuit_breaker.execute(
                lambda: self._call_parser_service(instance, request),
                parser_endpoint
            )
            
            duration = time.time() - start_time
            
            # 5. 记录成功指标
            self.metrics.record_success(
                parser_type=request.parser_type,
                duration_ms=duration * 1000,
                text_length=len(request.text)
            )
            
            return ParseResponse(
                success=True,
                result=result,
                parser_used=instance.id,
                confidence=result.confidence if hasattr(result, 'confidence') else 1.0
            )
            
        except Exception as e:
            # 6. 记录失败指标
            self.metrics.record_failure(
                parser_type=request.parser_type,
                error_type=type(e).__name__
            )
            
            # 7. 失败处理：重试或降级
            return await self._handle_failure(request, e)
```

**智能路由的设计原则**：

2. **性能感知路由**：根据解析器性能指标动态选择
3. **成本感知路由**：在性能和成本间取得平衡
4. **质量感知路由**：根据质量要求选择合适解析器
5. **故障感知路由**：自动避开故障或性能下降的解析器

**智能路由的实现算法**：

```python
class IntelligentParserRouter:
    """智能解析器路由器"""
    
    def __init__(self):
        self.parser_registry = ParserRegistry()
        self.performance_monitor = PerformanceMonitor()
        self.routing_policy = RoutingPolicy()
    
    def select_parser(self, text: str, requirements: ParseRequirements) -> BaseOutputParser:
        """智能选择解析器"""
        
        # 1. 内容特征分析
        content_features = self._analyze_content(text)
        
        # 2. 候选解析器筛选
        candidates = self._filter_candidates(content_features, requirements)
        
        if not candidates:
            raise NoSuitableParserError("没有找到合适的解析器")
        
        # 3. 评分和排序
        scored_candidates = []
        for parser in candidates:
            score = self._score_parser(parser, content_features, requirements)
            scored_candidates.append((parser, score))
        
        # 按分数降序排序
        scored_candidates.sort(key=lambda x: x[1], reverse=True)
        
        # 4. 选择最优解析器
        best_parser, best_score = scored_candidates[0]
        
        # 5. 记录路由决策
        self._log_routing_decision(
            parser_id=best_parser.id,
            score=best_score,
            alternatives=[(p.id, s) for p, s in scored_candidates[1:3]]
        )
        
        return best_parser
    
    def _analyze_content(self, text: str) -> ContentFeatures:
        """分析内容特征"""
        features = ContentFeatures()
        
        # 文本长度
        features.length = len(text)
        
        # 格式检测
        features.format = self._detect_format(text)
        
        # 语言检测
        features.language = self._detect_language(text)
        
        # 复杂度分析
        features.complexity = self._analyze_complexity(text)
        
        # 领域特征
        features.domain_features = self._extract_domain_features(text)
        
        return features
    
    def _score_parser(self, parser: BaseOutputParser, 
                     features: ContentFeatures,
                     requirements: ParseRequirements) -> float:
        """为解析器评分"""
        
        score = 0.0
        
        # 1. 性能评分（基于历史性能）
        perf_score = self.performance_monitor.get_performance_score(parser.id)
        score += perf_score * 0.3  # 30%权重
        
        # 2. 质量评分（基于历史质量）
        quality_score = self.performance_monitor.get_quality_score(parser.id)
        score += quality_score * 0.3  # 30%权重
        
        # 3. 成本评分（基于使用成本）
        cost_score = self._calculate_cost_score(parser, features.length)
        score += cost_score * 0.2  # 20%权重
        
        # 4. 匹配度评分（基于特征匹配）
        match_score = self._calculate_match_score(parser, features, requirements)
        score += match_score * 0.2  # 20%权重
        
        return score
```

```python
class ParallelParser(BaseOutputParser[List[T]]):
    """并行解析器"""
    
    def parse_parallel(self, texts: List[str], 
                      max_workers: int = None) -> List[T]:
        """并行解析多个文本"""
        
        from concurrent.futures import ThreadPoolExecutor, as_completed
        
        if max_workers is None:
            max_workers = min(32, len(texts) + 4)
        
        results = [None] * len(texts)
        
        with ThreadPoolExecutor(max_workers=max_workers) as executor:
            # 提交所有解析任务
            future_to_index = {
                executor.submit(self.parse, text): i 
                for i, text in enumerate(texts)
            }
            
            # 收集结果
            for future in as_completed(future_to_index):
                index = future_to_index[future]
                try:
                    results[index] = future.result()
                except Exception as e:
                    results[index] = self._handle_error(texts[index], e)
        
        return results
```

```python
class PluginBasedParserSystem:
    """插件化解析器系统"""
    
    def __init__(self, plugin_dir: str):
        self.plugin_dir = plugin_dir
        self.plugins = {}
        self._load_plugins()
    
    def _load_plugins(self):
        """加载插件"""
        import importlib
        import os
        
        for filename in os.listdir(self.plugin_dir):
            if filename.endswith('.py') and not filename.startswith('_'):
                plugin_name = filename[:-3]
                
                try:
                    # 动态导入插件
                    spec = importlib.util.spec_from_file_location(
                        plugin_name,
                        os.path.join(self.plugin_dir, filename)
                    )
                    module = importlib.util.module_from_spec(spec)
                    spec.loader.exec_module(module)
                    
                    # 注册插件
                    if hasattr(module, 'register_parsers'):
                        module.register_parsers(self)
                    
                    self.plugins[plugin_name] = module
                    
                except Exception as e:
                    print(f"加载插件 {plugin_name} 失败: {e}")
    
    def register_parser(self, name: str, parser_class: type):
        """注册解析器"""
        self.parsers[name] = parser_class
    
    def get_parser(self, name: str) -> BaseOutputParser:
        """获取解析器"""
        if name not in self.parsers:
            raise ValueError(f"解析器未找到: {name}")
        
        parser_class = self.parsers[name]
        return parser_class()
```

```python
class VersionedParser(BaseOutputParser[T]):
    """版本化解析器"""
    
    def __init__(self, version: str = "latest"):
        super().__init__()
        self.version = version
        self._load_implementation(version)
    
    def _load_implementation(self, version: str):
        """加载指定版本的实现"""
        
        if version == "latest":
            version = self._get_latest_version()
        
        # 根据版本加载不同的实现类
        impl_class = self._get_implementation_class(version)
        self.impl = impl_class()
    
    def parse(self, text: str) -> T:
        """解析文本（版本化实现）"""
        return self.impl.parse(text)
    
    def upgrade(self, new_version: str, data: List[T]) -> List[T]:
        """升级数据到新版本"""
        
        if new_version == self.version:
            return data  # 无需升级
        
        # 检查版本兼容性
        if not self._is_compatible(self.version, new_version):
            raise VersionCompatibilityError(
                f"版本 {self.version} 到 {new_version} 不兼容"
            )
        
        # 执行升级转换
        upgraded_data = []
        for item in data:
            upgraded_item = self._convert_item(item, self.version, new_version)
            upgraded_data.append(upgraded_item)
        
        return upgraded_data
```

```python
class FinancialDataParser(BaseOutputParser[Dict[str, Any]]):
    """财务数据解析器"""
    
    def __init__(self, 
                 default_currency: str = "CNY",
                 decimal_places: int = 2,
                 strict_mode: bool = False):
        super().__init__()
        self.default_currency = default_currency
        self.decimal_places = decimal_places
        self.strict_mode = strict_mode
        
        # 初始化解析模式
        self.patterns = self._initialize_patterns()
        
    def _initialize_patterns(self) -> Dict[str, re.Pattern]:
        """初始化解析模式"""
        
        return {
            # 金额模式：支持正负数和千位分隔符
            "amount": re.compile(r'([-+]?\d{1,3}(?:,\d{3})*(?:\.\d+)?|\d+(?:\.\d+)?)\s*([万千亿]?[元美元]?)'),
            
            # 百分比模式
            "percentage": re.compile(r'([-+]?\d+(?:\.\d+)?)%'),
            
            # 日期模式
            "date": re.compile(r'(\d{4})[-/年](\d{1,2})[-/月](\d{1,2})?日?'),
            
            # 会计科目模式
            "account_item": re.compile(r'([^\d\n:：]{2,30}?)[:：]\s*(.+)'),
            
            # 货币单位模式
            "currency_unit": re.compile(r'([A-Z]{3})?\s*(?:人民币|美元|欧元|日元)?\s*([万千亿]?[元美元欧元日元])?')
        }
    
    def _parse_amount(self, text: str) -> Optional[Decimal]:
        """解析金额"""
        
        match = self.patterns["amount"].search(text)
        if not match:
            return None
        
        amount_text = match.group(1).replace(',', '')
        unit_text = match.group(2) if match.group(2) else ""
        
        try:
            # 解析数值
            amount = Decimal(amount_text)
            
            # 单位转换
            if unit_text:
                amount = self._convert_unit(amount, unit_text)
            
            # 四舍五入
            amount = amount.quantize(
                Decimal('1.' + '0' * self.decimal_places),
                rounding=ROUND_HALF_UP
            )
            
            return amount
            
        except Exception as e:
            if self.strict_mode:
                raise ValueError(f"金额解析失败: {text}, 错误: {e}")
            return None
    
    def _convert_unit(self, amount: Decimal, unit: str) -> Decimal:
        """转换单位"""
        
        conversion_factors = {
            "元": Decimal('1'),
            "万元": Decimal('10000'),
            "亿元": Decimal('100000000'),
            "美元": Decimal('1'),  # 简化处理
            "万美元": Decimal('10000'),
            "百万美元": Decimal('1000000'),
            "欧元": Decimal('1'),  # 简化处理
            "万欧元": Decimal('10000'),
            "日元": Decimal('1'),  # 简化处理
            "万円": Decimal('10000')
        }
        
        factor = conversion_factors.get(unit, Decimal('1'))
        return amount * factor
    
    def _parse_percentage(self, text: str) -> Optional[Decimal]:
        """解析百分比"""
        
        match = self.patterns["percentage"].search(text)
        if not match:
            return None
        
        try:
            percentage_text = match.group(1)
            percentage = Decimal(percentage_text) / Decimal('100')
            return percentage.quantize(Decimal('0.0001'), rounding=ROUND_HALF_UP)
        except Exception:
            return None
    
    def _parse_date(self, text: str) -> Optional[date]:
        """解析日期"""
        
        match = self.patterns["date"].search(text)
        if not match:
            return None
        
        try:
            year = int(match.group(1))
            month = int(match.group(2))
            day = int(match.group(3)) if match.group(3) else 1
            
            return date(year, month, day)
        except Exception:
            return None
    
    def parse(self, text: str) -> Dict[str, Any]:
        """解析财务数据文本"""
        
        result = {
            "metadata": {},
            "financial_items": [],
            "parsing_errors": [],
            "warnings": []
        }
        
        lines = text.strip().split('\n')
        
        for i, line in enumerate(lines, 1):
            line = line.strip()
            if not line:
                continue
            
            # 尝试解析会计科目项目
            account_match = self.patterns["account_item"].search(line)
            if account_match:
                item_name = account_match.group(1).strip()
                value_text = account_match.group(2).strip()
                
                # 解析金额
                amount = self._parse_amount(value_text)
                percentage = self._parse_percentage(value_text)
                
                if amount is not None:
                    item = {
                        "line_number": i,
                        "item_name": item_name,
                        "value_type": "amount",
                        "value": float(amount),
                        "currency": self.default_currency,
                        "raw_text": line
                    }
                    result["financial_items"].append(item)
                    
                elif percentage is not None:
                    item = {
                        "line_number": i,
                        "item_name": item_name,
                        "value_type": "percentage",
                        "value": float(percentage),
                        "raw_text": line
                    }
                    result["financial_items"].append(item)
                    
                else:
                    error_msg = f"第{i}行无法解析: {line}"
                    result["parsing_errors"].append(error_msg)
                    
                    if self.strict_mode:
                        raise ValueError(error_msg)
            
            # 检查是否包含日期信息
            date_value = self._parse_date(line)
            if date_value:
                result["metadata"]["date_found"] = date_value.isoformat()
            
            # 检查是否包含货币单位信息
            currency_match = self.patterns["currency_unit"].search(line)
            if currency_match:
                currency_code = currency_match.group(1)
                currency_unit = currency_match.group(2)
                
                if currency_code:
                    result["metadata"]["currency_code"] = currency_code
                if currency_unit:
                    result["metadata"]["currency_unit"] = currency_unit
        
        # 统计信息
        result["statistics"] = {
            "total_lines": len(lines),
            "parsed_items": len(result["financial_items"]),
            "parsing_errors": len(result["parsing_errors"]),
            "success_rate": len(result["financial_items"]) / max(len(lines), 1)
        }
        
        return result

    @property

        self.primary_parser = FinancialDataParser()
        self.simple_parser = SimpleKeyValueParser()

        """解析文本，支持重试和降级"""

            try:
                # 尝试主解析器
                return self.primary_parser.parse(text)
            except Exception as e:
                last_error = e
                print(f"解析尝试 {attempt + 1} 失败: {e}")
                
                # 最后一次尝试
                if attempt == self.max_retries - 1:
                    break

        if self.fallback_to_simple:
            print("主解析器失败，尝试降级到简单解析器...")

                simple_result = self.simple_parser.parse(text)
                return {
                    "data": simple_result,
                    "metadata": {
                        "parser_used": "simple_fallback",
                        "original_error": str(last_error),
                        "warning": "使用降级解析器，数据可能不完整"
                    }
                }
            except Exception as fallback_error:
                raise ValueError(f"所有解析尝试失败: {last_error}, 降级也失败: {fallback_error}")
        else:
            raise last_error
```

```python
class DateRange(BaseModel):
    """日期范围"""
    start_date: date = Field(description="开始日期")
    end_date: date = Field(description="结束日期")
    range_type: str = Field(description="范围类型，如：本月、本季度、去年等")
    
    @validator('end_date')
    def validate_date_range(cls, end_date, values):
        """验证日期范围合理性"""
        if 'start_date' in values and end_date < values['start_date']:
            raise ValueError('结束日期不能早于开始日期')
        return end_date

class SmartDateRangeParser(BaseOutputParser[DateRange]):
    """智能日期范围解析器"""
    
    def __init__(self, reference_date: Optional[date] = None):
        super().__init__()
        self.reference_date = reference_date or date.today()
    
    def parse(self, text: str) -> DateRange:
        """解析文本为日期范围"""
        
        text = text.strip().lower()
        
        # 常见日期范围模式
        patterns = {
            # 相对日期范围
            r'今天|今日': lambda d: (d, d, "今日"),
            r'昨天|昨日': lambda d: (d.replace(day=d.day-1), d.replace(day=d.day-1), "昨日"),
            r'明天|明日': lambda d: (d.replace(day=d.day+1), d.replace(day=d.day+1), "明日"),
            r'本周|这周': lambda d: self._get_week_range(d, 0),
            r'上周|上一周': lambda d: self._get_week_range(d, -1),
            r'下周|下一周': lambda d: self._get_week_range(d, 1),
            r'本月|这个月': lambda d: self._get_month_range(d, 0),
            r'上月|上个月': lambda d: self._get_month_range(d, -1),
            r'下月|下个月': lambda d: self._get_month_range(d, 1),
            r'本季度|这个季度': lambda d: self._get_quarter_range(d, 0),
            r'上季度|上个季度': lambda d: self._get_quarter_range(d, -1),
            r'下季度|下个季度': lambda d: self._get_quarter_range(d, 1),
            r'今年|本年': lambda d: self._get_year_range(d, 0),
            r'去年|上年': lambda d: self._get_year_range(d, -1),
            r'明年|下年': lambda d: self._get_year_range(d, 1),
            
            # 具体日期范围
            r'(\d{4})年(\d{1,2})月(\d{1,2})日到(\d{4})年(\d{1,2})月(\d{1,2})日': 
                self._parse_explicit_date_range,
            r'(\d{4})年(\d{1,2})月到(\d{4})年(\d{1,2})月':
                self._parse_month_range,
            r'(\d{4})年到(\d{4})年':
                self._parse_year_range,
        }
        
        # 尝试匹配模式
        for pattern, handler in patterns.items():
            match = re.search(pattern, text)
            if match:
                try:
                    start_date, end_date, range_type = handler(self.reference_date, match)
                    return DateRange(
                        start_date=start_date,
                        end_date=end_date,
                        range_type=range_type
                    )
                except Exception as e:
                    continue
        
        # 默认返回今天
        return DateRange(
            start_date=self.reference_date,
            end_date=self.reference_date,
            range_type="今日（默认）"
        )
    
    def _get_week_range(self, ref_date: date, week_offset: int) -> tuple:
        """获取周范围"""
        from datetime import timedelta
        
        # 计算目标周的周一
        days_to_monday = ref_date.weekday()  # Monday is 0
        current_monday = ref_date - timedelta(days=days_to_monday)
        target_monday = current_monday + timedelta(weeks=week_offset)
        
        start_date = target_monday
        end_date = target_monday + timedelta(days=6)
        
        range_type = f"第{week_offset+1}周" if week_offset != 0 else "本周"
        
        return start_date, end_date, range_type
    
    def _get_month_range(self, ref_date: date, month_offset: int) -> tuple:
        """获取月范围"""
        from datetime import timedelta
        
        # 计算目标月的第一天
        year = ref_date.year
        month = ref_date.month + month_offset
        
        # 处理年跨月
        while month > 12:
            month -= 12
            year += 1
        while month < 1:
            month += 12
            year -= 1
        
        start_date = date(year, month, 1)
        
        # 计算下个月的第一天，然后减一天得到本月最后一天
        if month == 12:
            next_month = date(year + 1, 1, 1)
        else:
            next_month = date(year, month + 1, 1)
        
        end_date = next_month - timedelta(days=1)
        
        range_type = f"{month_offset+1}月" if month_offset != 0 else "本月"
        
        return start_date, end_date, range_type
    
    def _get_quarter_range(self, ref_date: date, quarter_offset: int) -> tuple:
        """获取季度范围"""
        # 计算当前季度
        current_quarter = (ref_date.month - 1) // 3 + 1
        
        # 计算目标季度
        target_quarter = current_quarter + quarter_offset
        target_year = ref_date.year
        
        # 处理年跨季度
        while target_quarter > 4:
            target_quarter -= 4
            target_year += 1
        while target_quarter < 1:
            target_quarter += 4
            target_year -= 1
        
        # 计算季度开始和结束月份
        start_month = (target_quarter - 1) * 3 + 1
        end_month = start_month + 2
        
        start_date = date(target_year, start_month, 1)
        
        # 计算季度最后一天
        if end_month == 12:
            next_month = date(target_year + 1, 1, 1)
        else:
            next_month = date(target_year, end_month + 1, 1)
        
        end_date = next_month - timedelta(days=1)
        
        range_type = f"第{target_quarter}季度"
        if quarter_offset == -1:
            range_type = "上季度"
        elif quarter_offset == 0:
            range_type = "本季度"
        elif quarter_offset == 1:
            range_type = "下季度"
        
        return start_date, end_date, range_type
    
    def _get_year_range(self, ref_date: date, year_offset: int) -> tuple:
        """获取年范围"""
        target_year = ref_date.year + year_offset
        
        start_date = date(target_year, 1, 1)
        end_date = date(target_year, 12, 31)
        
        range_type = f"{target_year}年"
        if year_offset == -1:
            range_type = "去年"
        elif year_offset == 0:
            range_type = "今年"
        elif year_offset == 1:
            range_type = "明年"
        
        return start_date, end_date, range_type
    
    def _parse_explicit_date_range(self, ref_date: date, match: re.Match) -> tuple:
        """解析显式日期范围"""
        start_year = int(match.group(1))
        start_month = int(match.group(2))
        start_day = int(match.group(3))
        end_year = int(match.group(4))
        end_month = int(match.group(5))
        end_day = int(match.group(6))
        
        start_date = date(start_year, start_month, start_day)
        end_date = date(end_year, end_month, end_day)
        
        return start_date, end_date, "自定义日期范围"
    
    def _parse_month_range(self, ref_date: date, match: re.Match) -> tuple:
        """解析月范围"""
        start_year = int(match.group(1))
        start_month = int(match.group(2))
        end_year = int(match.group(3))
        end_month = int(match.group(4))
        
        start_date = date(start_year, start_month, 1)
        
        # 计算结束月的最后一天
        if end_month == 12:
            next_month = date(end_year + 1, 1, 1)
        else:
            next_month = date(end_year, end_month + 1, 1)
        
        end_date = next_month - timedelta(days=1)
        
        return start_date, end_date, "月范围"
    
    def _parse_year_range(self, ref_date: date, match: re.Match) -> tuple:
        """解析年范围"""
        start_year = int(match.group(1))
        end_year = int(match.group(2))
        
        start_date = date(start_year, 1, 1)
        end_date = date(end_year, 12, 31)
        
        return start_date, end_date, "年范围"
    
    def get_format_instructions(self) -> str:
        """返回格式指令"""
        return """请指定日期范围，支持以下格式：
1. 相对日期：今天、昨天、明天、本周、上周、本月、上个月、本季度、去年、明年等
2. 具体范围：2024年1月1日到2024年12月31日
3. 月范围：2024年1月到2024年12月
4. 年范围：2023年到2024年

请明确指定日期范围。"""
    
    @property
    def _type(self) -> str:
        return "smart_date_range_parser"

class MultiLanguageProductReview(BaseModel):

class MultiLanguageReviewParser(BaseOutputParser[MultiLanguageProductReview]):
    """多语言产品评价解析器"""
    
    def __init__(self, 
                 target_language: str = "zh",
                 enable_translation: bool = True,
                 min_confidence: float = 0.7):
        super().__init__()
        self.target_language = target_language
        self.enable_translation = enable_translation
        self.min_confidence = min_confidence
        
        # 语言检测模式
        self.language_patterns = {
            "en": r'\b(the|and|is|in|of|to|a|for)\b',
            "zh": r'[\u4e00-\u9fff]',  # 中文字符
            "ja": r'[\u3040-\u309f\u30a0-\u30ff]',  # 日文假名
            "ko": r'[\uac00-\ud7af]',  # 韩文字符
            "fr": r'\b(le|la|les|un|une|des)\b',
            "de": r'\b(der|die|das|ein|eine)\b',
            "es": r'\b(el|la|los|las|un|una)\b'
        }
        
        # 情感关键词
        self.sentiment_keywords = {
            "positive": {
                "en": ["good", "excellent", "great", "amazing", "love", "perfect", "best"],
                "zh": ["好", "优秀", "很棒", "很好", "喜欢", "完美", "最好"],
                "ja": ["良い", "優秀", "素晴らしい", "大好き", "完璧", "最高"],
                "ko": ["좋다", "훌륭하다", "멋지다", "사랑한다", "완벽하다", "최고"]
            },
            "negative": {
                "en": ["bad", "poor", "terrible", "awful", "hate", "worst", "disappointing"],
                "zh": ["差", "糟糕", "很差", "讨厌", "最差", "失望"],
                "ja": ["悪い", "ひどい", "最悪", "嫌い", "失望"],
                "ko": ["나쁘다", "형편없다", "최악", "싫다", "실망"]
            }
        }
    
    def _detect_language(self, text: str) -> str:
        """检测文本语言"""
        
        scores = {}
        
        for lang_code, pattern in self.language_patterns.items():
            matches = re.findall(pattern, text, re.IGNORECASE)
            score = len(matches) / max(len(text.split()), 1)
            scores[lang_code] = score
        
        # 返回得分最高的语言
        if scores:
            detected_lang = max(scores.items(), key=lambda x: x[1])[0]
            confidence = scores[detected_lang]
            
            if confidence >= self.min_confidence:
                return detected_lang
        
        # 默认语言
        return "en"
    
    def _detect_sentiment(self, text: str, language: str) -> str:
        """检测情感"""
        
        text_lower = text.lower()
        
        # 检查正面关键词
        positive_keywords = self.sentiment_keywords["positive"].get(language, [])
        negative_keywords = self.sentiment_keywords["negative"].get(language, [])
        
        positive_score = sum(1 for keyword in positive_keywords if keyword.lower() in text_lower)
        negative_score = sum(1 for keyword in negative_keywords if keyword.lower() in text_lower)
        
        if positive_score > negative_score:
            return "positive"
        elif negative_score > positive_score:
            return "negative"
        else:
            return "neutral"
    
    def _extract_rating(self, text: str) -> float:
        """提取评分"""
        
        # 查找数字评分
        rating_patterns = [
            r'(\d+(?:\.\d+)?)\s*(?:out of|/|分|点)\s*(?:5|5\.0|五)',
            r'评分[：:]\s*(\d+(?:\.\d+)?)',
            r'rate[：:]\s*(\d+(?:\.\d+)?)',
            r'★{1,5}',  # 星号评分
            r'⭐{1,5}'   # 星星评分
        ]
        
        for pattern in rating_patterns:
            match = re.search(pattern, text)
            if match:
                if pattern in [r'★{1,5}', r'⭐{1,5}']:
                    # 星号评分：数星号数量
                    stars = len(match.group())
                    return float(stars)
                else:
                    # 数字评分
                    rating_text = match.group(1)
                    try:
                        rating = float(rating_text)
                        # 规范化到1-5分
                        if rating > 5:
                            rating = 5.0
                        elif rating < 1:
                            rating = 1.0
                        return rating
                    except ValueError:
                        continue
        
        # 默认评分
        return 3.0
    
    def _extract_product_name(self, text: str) -> str:
        """提取产品名称"""
        
        # 简单实现：查找可能的品牌和产品关键词
        product_keywords = [
            r'([A-Z][a-z]+(?:\s+[A-Z][a-z]+)*)\s+(?:Pro|Max|Plus|Lite|Air|Mini)',
            r'(?:购买|买了|试用|使用)\s+([^\s，。！？]{2,20}?)(?:\s+|的|产品)',
            r'product[：:]\s*([^\n]{2,50})',
            r'产品[：:]\s*([^\n]{2,50})'
        ]
        
        for pattern in product_keywords:
            match = re.search(pattern, text)
            if match:
                product_name = match.group(1).strip()
                if len(product_name) >= 2:  # 最小长度
                    return product_name
        
        # 默认产品名称
        return "未知产品"
    
    def _extract_key_points(self, text: str, language: str) -> List[str]:
        """提取关键点"""
        
        key_points = []
        
        # 根据语言使用不同的分割符
        if language in ["zh", "ja", "ko"]:
            # 中文、日文、韩文：使用逗号、句号、分号分割
            sentences = re.split(r'[，。；！？\n]', text)
        else:
            # 英文和其他语言：使用句号、逗号、分号分割
            sentences = re.split(r'[.,;!?\n]', text)
        
        # 提取有意义的句子作为关键点
        for sentence in sentences:
            sentence = sentence.strip()
            if len(sentence) >= 10:  # 最小长度
                # 清理句子
                sentence = re.sub(r'\s+', ' ', sentence)
                key_points.append(sentence)
        
        # 限制关键点数量
        return key_points[:5]  # 最多5个关键点
    
    def parse(self, text: str) -> MultiLanguageProductReview:
        """解析多语言产品评价"""
        
        # 检测语言
        detected_language = self._detect_language(text)
        
        # 提取信息
        product_name = self._extract_product_name(text)
        rating = self._extract_rating(text)
        sentiment = self._detect_sentiment(text, detected_language)
        key_points = self._extract_key_points(text, detected_language)
        
        # 生成翻译摘要（如果启用）
        translated_summary = None
        if self.enable_translation and detected_language != self.target_language:
            # 这里应该调用翻译API
            # 为了演示，生成简单摘要
            translated_summary = f"[{detected_language}->{self.target_language}] 产品评价摘要"
        
        return MultiLanguageProductReview(
            product_name=product_name,
            rating=rating,
            sentiment=sentiment,
            language=detected_language,
            key_points=key_points,
            translated_summary=translated_summary
        )
    
    def get_format_instructions(self) -> str:
        """返回格式指令"""

评价应包含：

    @property
    def _type(self) -> str:
        return "multi_language_review_parser"

from typing import Iterator, AsyncIterator

class StreamingListParser(BaseOutputParser[List[str]]):
    """流式列表解析器"""
    
    def __init__(self, chunk_size: int = 100):
        super().__init__()
        self.chunk_size = chunk_size
        self.buffer = ""
    
    def parse(self, text: str) -> List[str]:
        """批量解析"""
        # 简单实现：按行分割
        return [line.strip() for line in text.split('\n') if line.strip()]
    
    def parse_stream(self, text_stream: Iterator[str]) -> Iterator[List[str]]:
        """流式解析"""
        
        for chunk in text_stream:
            self.buffer += chunk
            
            # 检查缓冲区是否有完整行
            lines = self.buffer.split('\n')
            
            # 除了最后一行（可能不完整），其他都是完整行
            if len(lines) > 1:
                complete_lines = lines[:-1]
                self.buffer = lines[-1]  # 保留不完整的最后一行
                
                if complete_lines:
                    yield [line.strip() for line in complete_lines if line.strip()]
        
        # 处理缓冲区剩余内容
        if self.buffer.strip():
            yield [self.buffer.strip()]
    
    async def aparse_stream(self, text_stream: AsyncIterator[str]) -> AsyncIterator[List[str]]:
        """异步流式解析"""
        
        async for chunk in text_stream:
            self.buffer += chunk
            
            # 检查缓冲区是否有完整行
            lines = self.buffer.split('\n')
            
            # 除了最后一行（可能不完整），其他都是完整行
            if len(lines) > 1:
                complete_lines = lines[:-1]
                self.buffer = lines[-1]  # 保留不完整的最后一行
                
                if complete_lines:
                    yield [line.strip() for line in complete_lines if line.strip()]
        
        # 处理缓冲区剩余内容
        if self.buffer.strip():
            yield [self.buffer.strip()]
    
    def get_format_instructions(self) -> str:
        return "请将内容格式化为多行文本，每行一个项目。"
    
    @property
    def _type(self) -> str:
        return "streaming_list_parser"

class ParserFactory:
    """解析器工厂"""
    
    def __init__(self):
        self.parser_registry = {}
        self.parser_configs = {}
    
    def register_parser(self, parser_type: str, parser_class: type, config: Dict[str, Any] = None):
        """注册解析器"""
        self.parser_registry[parser_type] = parser_class
        self.parser_configs[parser_type] = config or {}
    
    def create_parser(self, parser_type: str, **kwargs) -> BaseOutputParser:
        """创建解析器实例"""
        
        if parser_type not in self.parser_registry:
            raise ValueError(f"未知的解析器类型: {parser_type}")
        
        parser_class = self.parser_registry[parser_type]
        config = self.parser_configs.get(parser_type, {})
        
        # 合并配置
        merged_config = {**config, **kwargs}
        
        # 创建解析器实例
        return parser_class(**merged_config)
    
    def get_available_parsers(self) -> List[str]:
        """获取可用的解析器列表"""
        return list(self.parser_registry.keys())

    def parse_with_metrics(self, parser_type: str, text: str, **kwargs) -> Any:
        """带指标收集的解析"""
        
        start_time = datetime.now()
        
        try:
            parser = self.get_or_create_parser(parser_type, **kwargs)
            result = parser.parse(text)
            
            # 记录成功
            end_time = datetime.now()
            duration = (end_time - start_time).total_seconds() * 1000
            
            self.metrics.record_success(parser_type, duration, len(text))
            
            return {
                "success": True,
                "result": result,
                "parser_type": parser_type,
                "metrics": {
                    "duration_ms": duration,
                    "text_length": len(text),
                    "timestamp": end_time.isoformat()
                }
            }
            
        except Exception as e:
            # 记录失败

            duration = (end_time - start_time).total_seconds() * 1000
            
            self.metrics.record_failure(parser_type, duration, str(e))
            
            return {
                "success": False,
                "error": str(e),
                "parser_type": parser_type,
                "metrics": {
                    "duration_ms": duration,
                    "text_length": len(text),
                    "timestamp": end_time.isoformat()
                }
            }

        """获取解析器统计信息"""
        return self.metrics.get_stats()

class ParserMetrics:
    """解析器指标收集器"""
    
    def __init__(self):
        self.metrics = {
            "total_calls": 0,
            "successful_calls": 0,
            "failed_calls": 0,
            "total_duration_ms": 0,
            "total_text_length": 0,
            "parser_usage": {},
            "error_distribution": {}
        }
    
    def record_success(self, parser_type: str, duration_ms: float, text_length: int):
        """记录成功解析"""
        
        self.metrics["total_calls"] += 1
        self.metrics["successful_calls"] += 1
        self.metrics["total_duration_ms"] += duration_ms
        self.metrics["total_text_length"] += text_length
        
        # 更新解析器使用统计
        if parser_type not in self.metrics["parser_usage"]:
            self.metrics["parser_usage"][parser_type] = {
                "calls": 0,
                "successful": 0,
                "total_duration_ms": 0,
                "total_text_length": 0
            }
        
        usage = self.metrics["parser_usage"][parser_type]
        usage["calls"] += 1
        usage["successful"] += 1
        usage["total_duration_ms"] += duration_ms
        usage["total_text_length"] += text_length
    
    def record_failure(self, parser_type: str, duration_ms: float, error: str):
        """记录失败解析"""
        
        self.metrics["total_calls"] += 1
        self.metrics["failed_calls"] += 1
        self.metrics["total_duration_ms"] += duration_ms
        
        # 更新解析器使用统计
        if parser_type not in self.metrics["parser_usage"]:
            self.metrics["parser_usage"][parser_type] = {
                "calls": 0,
                "successful": 0,
                "failed": 0,
                "total_duration_ms": 0
            }
        
        usage = self.metrics["parser_usage"][parser_type]
        usage["calls"] += 1
        usage["failed"] += 1
        usage["total_duration_ms"] += duration_ms
        
        # 更新错误分布
        error_type = type(error).__name__ if hasattr(error, '__name__') else "Unknown"
        if error_type not in self.metrics["error_distribution"]:
            self.metrics["error_distribution"][error_type] = 0
        self.metrics["error_distribution"][error_type] += 1
    
    def get_stats(self) -> Dict[str, Any]:
        """获取统计信息"""
        
        stats = self.metrics.copy()
        
        # 计算派生指标
        if stats["total_calls"] > 0:
            stats["success_rate"] = stats["successful_calls"] / stats["total_calls"]
            stats["failure_rate"] = stats["failed_calls"] / stats["total_calls"]
            
            if stats["successful_calls"] > 0:
                stats["average_duration_ms"] = stats["total_duration_ms"] / stats["successful_calls"]
                stats["average_text_length"] = stats["total_text_length"] / stats["successful_calls"]
            else:
                stats["average_duration_ms"] = 0
                stats["average_text_length"] = 0
        
        # 计算每个解析器的指标
        for parser_type, usage in stats["parser_usage"].items():
            if usage["calls"] > 0:
                usage["success_rate"] = usage["successful"] / usage["calls"] if usage["successful"] > 0 else 0
                usage["average_duration_ms"] = usage["total_duration_ms"] / usage["calls"] if usage["calls"] > 0 else 0
        
        return stats
```
