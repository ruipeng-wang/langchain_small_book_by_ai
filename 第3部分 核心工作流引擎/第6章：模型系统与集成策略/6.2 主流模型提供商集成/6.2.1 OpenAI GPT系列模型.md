# 6.2.1 OpenAI GPT系列模型

## 概念讲解

### OpenAI在AI生态中的地位

OpenAI是目前最主流的大型语言模型提供商，其GPT系列模型已成为AI应用开发的事实标准。LangChain通过`langchain-openai`包提供了对OpenAI模型的全面支持。

#### 支持的模型

| 模型 | 特点 | 适用场景 |
|------|------|---------|
| `gpt-4o` | 最新旗舰，速度快 | 通用任务 |
| `gpt-4o-mini` | 轻量快速，成本低 | 简单任务、批量处理 |
| `gpt-4` | 强推理能力 | 复杂分析、代码 |
| `o1` | 深度推理 | 数学、科学计算 |

### ChatOpenAI核心类

LangChain通过`ChatOpenAI`类封装了OpenAI的聊天模型：

```python
from langchain_openai import ChatOpenAI

# 最简用法（使用环境变量OPENAI_API_KEY）
model = ChatOpenAI(model="gpt-4o-mini")

# 完整配置
model = ChatOpenAI(
    model="gpt-4o",
    temperature=0.7,      # 创造性控制（0-1）
    max_tokens=1024,      # 最大输出长度
    timeout=30,           # 超时时间（秒）
    max_retries=2         # 自动重试次数
)
```

## 核心要点

1. **推荐模型**：新项目使用`gpt-4o-mini`（性价比最优）
2. **temperature**：控制输出随机性，0=确定性，1=创造性
3. **工具调用**：使用`bind_tools()`绑定自定义工具
4. **结构化输出**：使用`with_structured_output()`获取格式化的数据

## 简单示例

### 基本调用

```python
from langchain_openai import ChatOpenAI
from langchain_core.messages import HumanMessage

model = ChatOpenAI(model="gpt-4o-mini")

# 直接传字符串
response = model.invoke("解释什么是递归")
print(response.content)

# 传消息列表（支持多轮对话）
messages = [
    HumanMessage(content="你好"),
]
response = model.invoke(messages)
```

### 工具调用

```python
from langchain_openai import ChatOpenAI
from langchain_core.tools import tool

@tool
def get_weather(location: str) -> str:
    """获取指定城市的天气"""
    # 实际实现调用天气API
    return f"{location}: 晴, 25°C"

model = ChatOpenAI(model="gpt-4o-mini")
model_with_tools = model.bind_tools([get_weather])

response = model_with_tools.invoke("北京今天天气怎么样？")
```

### 结构化输出

```python
from pydantic import BaseModel, Field
from langchain_openai import ChatOpenAI

class Person(BaseModel):
    name: str = Field(description="姓名")
    age: int = Field(description="年龄")
    occupation: str = Field(description="职业")

model = ChatOpenAI(model="gpt-4o-mini")
structured_model = model.with_structured_output(Person)

result = structured_model.invoke("马云，1964年生，企业家，创立了阿里巴巴")
print(result.name)  # 马云
print(result.occupation)  # 企业家
```

## 进阶应用

### 流式输出

```python
model = ChatOpenAI(model="gpt-4o-mini", streaming=True)

for chunk in model.stream("写一个短篇故事"):
    print(chunk.content, end="", flush=True)
```

### 速率限制处理

```python
from langchain_openai import ChatOpenAI
from langchain_core.rate_limiters import InMemoryRateLimiter

# 配置速率限制
rate_limiter = InMemoryRateLimiter(
    requests_per_second=10,
    check_every_n_seconds=0.1
)

model = ChatOpenAI(
    model="gpt-4o-mini",
    rate_limiter=rate_limiter
)
```

## 常见问题

**Q: 如何选择OpenAI模型？**

A: 简单任务用`gpt-4o-mini`（成本低、速度快），复杂推理用`gpt-4o`，数学/科学问题用`o1`。

**Q: API密钥如何设置？**

A: 推荐通过环境变量设置：`export OPENAI_API_KEY="your-key"`，或使用`.env`文件配合`python-dotenv`。

**Q: `with_structured_output()`支持哪些格式？**

A: 支持Pydantic模型和JSON Schema。推荐使用Pydantic，它提供类型验证和IDE提示。

## 本节总结

`ChatOpenAI`是LangChain中最常用的模型类。通过统一的接口、工具调用和结构化输出等功能，开发者可以快速构建高质量的AI应用。选择合适模型、配置速率限制、利用结构化输出，是高效使用OpenAI模型的关键。
