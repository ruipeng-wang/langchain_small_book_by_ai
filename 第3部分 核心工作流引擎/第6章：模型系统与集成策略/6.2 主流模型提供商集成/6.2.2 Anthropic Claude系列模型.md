# 6.2.2 Anthropic Claude系列模型

## 概念讲解

### Anthropic与Claude

Anthropic是OpenAI前员工创立的AI安全研究公司，其Claude系列模型以安全性强、长上下文处理和良好的指令遵循能力著称。

#### Claude模型家族

| 模型 | 定位 | 上下文长度 | 适用场景 |
|------|------|-----------|---------|
| `claude-sonnet-4` | 高性能 | 200K | 复杂任务、代码 |
| `claude-sonnet-4-mini` | 性价比 | 200K | 日常任务 |
| `claude-opus-4` | 旗舰级 | 200K | 高难度推理 |

### ChatAnthropic核心类

```python
from langchain_anthropic import ChatAnthropic

# 最简用法（使用环境变量ANTHROPIC_API_KEY）
model = ChatAnthropic(model="claude-sonnet-4-20250514")

# 完整配置
model = ChatAnthropic(
    model="claude-sonnet-4-20250514",
    temperature=0.7,
    max_tokens=1024,
    timeout=30
)
```

## 核心要点

1. **系统提示**：Claude支持`system`参数直接设置系统提示，与OpenAI有所不同
2. **长上下文**：支持200K上下文，适合文档分析
3. **安全优先**：内置安全机制，减少有害输出
4. **工具调用**：与OpenAI类似的`bind_tools()`接口

## 简单示例

### 基本调用

```python
from langchain_anthropic import ChatAnthropic

model = ChatAnthropic(model="claude-sonnet-4-20250514")

# 直接调用
response = model.invoke("解释什么是函数式编程")
print(response.content)
```

### 系统提示的设置

Claude有独特的系统提示处理方式：

```python
from langchain_anthropic import ChatAnthropic
from langchain_core.messages import HumanMessage, SystemMessage

model = ChatAnthropic(model="claude-sonnet-4-20250514")

# 方式1：使用system参数（推荐）
response = model.invoke(
    [HumanMessage(content="帮我写一个快速排序")],
    system="你是一个Python编程专家，代码要简洁高效"
)

# 方式2：使用SystemMessage
messages = [
    SystemMessage(content="你是一个Python编程专家"),
    HumanMessage(content="帮我写一个快速排序")
]
response = model.invoke(messages)
```

### 工具调用

```python
from langchain_anthropic import ChatAnthropic
from langchain_core.tools import tool

@tool
def search_documentation(query: str) -> str:
    """搜索文档获取相关信息"""
    return f"关于'{query}'的搜索结果..."

model = ChatAnthropic(model="claude-sonnet-4-20250514")
model_with_tools = model.bind_tools([search_documentation])

response = model_with_tools.invoke("如何配置LangChain的缓存？")
```

## 进阶应用

### 长文档处理

```python
from langchain_anthropic import ChatAnthropic

model = ChatAnthropic(model="claude-sonnet-4-20250514")

# 读取长文档
with open("long_document.txt", "r") as f:
    document = f.read()

# Claude可以处理超长上下文
response = model.invoke([
    HumanMessage(content=f"请总结以下文档的关键点：\n\n{document}")
])
```

### 流式输出

```python
model = ChatAnthropic(model="claude-sonnet-4-20250514")

for chunk in model.stream("写一个关于AI未来的短篇故事"):
    print(chunk.content, end="", flush=True)
```

## 与OpenAI的对比

| 特性 | Claude | OpenAI |
|------|--------|--------|
| 上下文长度 | 200K | 128K |
| 系统提示 | 专用`system`参数 | 用`SystemMessage` |
| 工具调用 | 支持 | 支持 |
| 价格 | 中等 | 中等偏高 |
| 安全性 | 较高 | 中等 |

## 常见问题

**Q: Claude和GPT哪个更好？**

A: 各有所长。Claude在安全性和长文本处理上更优，GPT在工具生态和集成便利性上更好。建议根据具体任务测试选择。

**Q: 系统提示用哪种方式？**

A: 推荐使用`system`参数，这是Claude的原生方式，效果通常更好。

**Q: 如何处理API速率限制？**

A: 设置`max_retries`参数自动重试，或使用`InMemoryRateLimiter`控制请求频率。

## 本节总结

Claude系列模型以安全性和长上下文处理能力见长。通过`ChatAnthropic`类，开发者可以方便地集成Claude到应用中。理解其独特的系统提示处理方式和工具调用能力，是高效使用Claude的关键。
